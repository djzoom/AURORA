## 参考实现 2

`quiz_answers`——读谱图挑战（验证格按关键词匹配，答案包含下列关键词即可通过）：

```python
quiz_answers = {
    "图A": "白噪声——所有频率、所有时刻能量均匀分布",
    "图B": "和弦——C 大调三和弦 C4+E4+G4（261/330/392 Hz 三条水平亮线）",
    "图C": "扫频（glissando）——频率从 200 Hz 线性爬升到 3000 Hz，对应滑音演奏技法",
    "图D": "衰减的纯音——频率位置不变，能量（亮度）随时间逐渐减弱",
}
```

**要点**
- 谱图三轴：横轴时间、纵轴频率、颜色亮度 = 能量。
- 四种指纹：纯音 = 一条水平线；和弦 = 多条平行水平线；扫频 = 斜线；白噪声 = 整幅均匀亮。
- 亮度随时间变化对应振幅包络（envelope）——频率不动、亮度变暗即衰减纯音。
