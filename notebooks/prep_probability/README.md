# 前导课程 · 代码优先的概率统计

目标是 ML 常用的那部分：分布、标准化、softmax + 交叉熵。

```bash
jupyter lab    # 打开 notebooks/prep_probability/，从 s1 开始
```

| 笔记 | 学什么 | 连到 Aurora |
|---|---|---|
| `s1_basics` | 随机模拟、大数定律 | 采样、dropout、数据增强 |

| `s3_distributions` | 均匀/正态分布、高斯密度 | 权重初始化、生成模型采样 |
| `s4_softmax_crossentropy` | softmax、交叉熵损失 | 分类器输出层 + 训练损失 |

每课：讲解 → numpy 演示 → ✏️ 填空 → 自动判卷 → 🔗 Aurora 连接。
