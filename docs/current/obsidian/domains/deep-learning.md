## 自动微分与计算图

### 计算图 (Computation Graph)
- **英文全称**: computation graph
- **缩写**: 无
- **定义**: 记录算术运算及其依赖关系的有向图结构，用于自动微分
- **出现课程**: L54, L55, L56, L57, L60, L65
- **标注状态**: ❌ 缺失标注

### 自动微分 (Automatic Differentiation)
- **英文全称**: automatic differentiation
- **缩写**: autograd
- **定义**: 通过计算图自动计算函数对参数梯度的机制，避免手动推导偏导
- **出现课程**: L54, L60
- **标注状态**: ❌ 缺失标注

### 有向无环图 (DAG)
- **英文全称**: directed acyclic graph
- **缩写**: DAG
- **定义**: 由节点和有向边组成、不含环路的图结构，计算图是其特例
- **出现课程**: L54, L56, L60, L65
- **标注状态**: ✅ 已标注（首次标注于 L54）

### 拓扑排序 (Topological Sort)
- **英文全称**: topological sort
- **缩写**: 无
- **定义**: 对有向无环图节点进行线性排序，使每个节点都在其所有前驱之后出现
- **出现课程**: L54, L56
- **标注状态**: ❌ 缺失标注

### 算子 (Operator)
- **英文全称**: operator
- **缩写**: 无
- **定义**: 计算图中的运算节点，如加法、乘法、激活函数等，携带前向值与反向梯度闭包
- **出现课程**: L54, L55
- **标注状态**: ❌ 缺失标注

### 闭包 (Closure)
- **英文全称**: closure
- **缩写**: 无
- **定义**: 捕获外部作用域变量的函数对象，用于封装每个算子节点的梯度传播规则
- **出现课程**: L54, L55
- **标注状态**: ❌ 缺失标注

### 叶节点 (Leaf Node)
- **英文全称**: leaf node
- **缩写**: 无
- **定义**: 计算图中由用户直接创建（非运算派生）的节点，梯度存储于 .grad 属性
- **出现课程**: L60
- **标注状态**: ✅ 已标注（首次标注于 L60）

### 非叶节点 (Non-Leaf Node)
- **英文全称**: non-leaf node
- **缩写**: 无
- **定义**: 由运算派生的中间节点，is_leaf 为 False，默认不保留梯度
- **出现课程**: L60
- **标注状态**: ❌ 缺失标注

## 反向传播

### 反向传播 (Backpropagation)
- **英文全称**: backpropagation
- **缩写**: backprop
- **定义**: 通过链式法则从输出节点向输入节点反向传播梯度的算法
- **出现课程**: L54, L55, L56, L57, L58, L59, L60, L64, L65
- **标注状态**: ❌ 缺失标注

### 链式法则 (Chain Rule)
- **英文全称**: chain rule
- **缩写**: 无
- **定义**: 复合函数求导的基本法则：d(f∘g)/dx = (df/dg)·(dg/dx)
- **出现课程**: L54, L56
- **标注状态**: ❌ 缺失标注

### 梯度 (Gradient)
- **英文全称**: gradient
- **缩写**: grad
- **定义**: 函数对各参数偏导数组成的向量，指示损失增加最快的方向
- **出现课程**: L54–L65
- **标注状态**: ❌ 缺失标注

### 偏导 (Partial Derivative)
- **英文全称**: partial derivative
- **缩写**: 无
- **定义**: 多变量函数对某一变量的导数，其余变量视为常数
- **出现课程**: L54, L56, L57
- **标注状态**: ❌ 缺失标注

### 局部偏导 (Local Partial Derivative)
- **英文全称**: local partial derivative
- **缩写**: 无
- **定义**: 算子节点对其直接输入的偏导数，链式法则通过组合局部偏导得到全局梯度
- **出现课程**: L54, L56
- **标注状态**: ❌ 缺失标注

### 乘积法则 (Product Rule)
- **英文全称**: product rule
- **缩写**: 无
- **定义**: 微积分中两函数之积的求导法则：d(uv)/dx = u'v + uv'
- **出现课程**: L54
- **标注状态**: ❌ 缺失标注

### 梯度累积 (Gradient Accumulation)
- **英文全称**: gradient accumulation
- **缩写**: 无
- **定义**: 反向传播时将多条路径的梯度相加到同一参数节点（+=），而非覆盖
- **出现课程**: L56, L59, L60
- **标注状态**: ❌ 缺失标注

### 数值微分 (Numerical Differentiation)
- **英文全称**: numerical differentiation
- **缩写**: 无
- **定义**: 用有限差分公式（如中心差分）近似计算导数，用于验证解析梯度
- **出现课程**: L56
- **标注状态**: ❌ 缺失标注

## 网络结构

### 前向传播 (Forward Pass)
- **英文全称**: forward pass
- **缩写**: 无
- **定义**: 从输入到输出按计算图顺序依次执行各算子的过程
- **出现课程**: L55, L57, L59, L60
- **标注状态**: ❌ 缺失标注

### 激活函数 (Activation Function)
- **英文全称**: activation function
- **缩写**: 无
- **定义**: 神经网络中引入非线性的函数，如 tanh、ReLU，使网络可拟合非线性映射
- **出现课程**: L55, L57, L63
- **标注状态**: ❌ 缺失标注

### 感知机 (Perceptron)
- **英文全称**: perceptron
- **缩写**: 无
- **定义**: 最简单的神经网络单元，计算输入的加权求和后经激活函数输出
- **出现课程**: L55, L57
- **标注状态**: ❌ 缺失标注

### 次梯度 (Subgradient)
- **英文全称**: subgradient
- **缩写**: 无
- **定义**: 非处处可微函数（如 ReLU 在 x=0 处）的广义导数，用于次梯度下降
- **出现课程**: L55
- **标注状态**: ❌ 缺失标注

### 全连接层 (Fully Connected Layer)
- **英文全称**: fully connected layer
- **缩写**: FC
- **定义**: 每个神经元与上一层所有神经元相连的网络层，执行仿射变换加激活
- **出现课程**: L57, L61
- **标注状态**: ❌ 缺失标注

### 线性层 (Linear Layer)
- **英文全称**: linear layer
- **缩写**: 无
- **定义**: 执行仿射变换（y = Wx + b）的网络层，即无激活函数的全连接层
- **出现课程**: L57, L61
- **标注状态**: ❌ 缺失标注

### 权重 (Weight)
- **英文全称**: weight
- **缩写**: w / W
- **定义**: 神经网络中可训练的参数，表示输入信号的强度系数
- **出现课程**: L54, L57, L58, L62
- **标注状态**: ❌ 缺失标注

### 偏置 (Bias)
- **英文全称**: bias
- **缩写**: b
- **定义**: 神经元中除权重外的可训练偏移量，用于调整激活阈值
- **出现课程**: L57, L58
- **标注状态**: ❌ 缺失标注

### 残差连接 (Residual Connection)
- **英文全称**: residual connection
- **缩写**: 无
- **定义**: 将某层输入直接加到其输出上的跳跃连接，缓解深层网络的梯度消失问题
- **出现课程**: L61
- **标注状态**: ❌ 缺失标注

### 跳连 (Skip Connection)
- **英文全称**: skip connection
- **缩写**: 无
- **定义**: 跨越若干层直接连接的网络结构，残差连接是其典型实现方式
- **出现课程**: L61
- **标注状态**: ❌ 缺失标注

## 训练循环

### 损失函数 (Loss Function)
- **英文全称**: loss function
- **缩写**: 无
- **定义**: 衡量模型预测值与真实标签差距的函数，训练目标是最小化该值
- **出现课程**: L54, L57, L58
- **标注状态**: ❌ 缺失标注

### 梯度下降 (Gradient Descent)
- **英文全称**: gradient descent
- **缩写**: GD
- **定义**: 沿梯度反方向迭代更新参数以最小化损失的基础优化算法
- **出现课程**: L57, L58

### 训练循环 (Training Loop)
- **英文全称**: training loop
- **缩写**: 无
- **定义**: 前向计算→损失→反向传播→参数更新→梯度清零的迭代过程
- **出现课程**: L58, L64
- **标注状态**: ❌ 缺失标注

### 收敛曲线 (Convergence Curve)
- **英文全称**: convergence curve
- **缩写**: 无
- **定义**: 训练过程中损失值随 epoch 变化的曲线，用于判断训练是否正常收敛
- **出现课程**: L58
- **标注状态**: ❌ 缺失标注

## 评估与调试

### 混淆矩阵 (Confusion Matrix)
- **英文全称**: confusion matrix
- **缩写**: CM
- **定义**: C[i][j] 表示真实标签为 i 被预测为 j 的样本数，对角线为正确预测
- **出现课程**: L64, L65
- **标注状态**: ❌ 缺失标注

### 过拟合 (Overfitting)
- **英文全称**: overfitting
- **缩写**: 无
- **定义**: 模型在训练集上表现良好但在验证集/测试集上性能下降的现象
- **出现课程**: L64, L65
- **标注状态**: ❌ 缺失标注

### 欠拟合 (Underfitting)
- **英文全称**: underfitting
- **缩写**: 无
- **定义**: 模型容量不足导致在训练集上也无法拟合数据规律的现象
- **出现课程**: L65
- **标注状态**: ❌ 缺失标注

### 每类准确率 (Per-Class Accuracy)
- **英文全称**: per-class accuracy
- **缩写**: 无
- **定义**: 分别统计每个类别预测正确的比例，反映类别不均衡情况下的真实性能

- **标注状态**: ✅ 已标注（首次标注于 L64）

### 自适应学习率 (Adaptive Learning Rate)

### 泛化间隔 (Generalization Gap)

- **缩写**: 无
- **定义**: 梯度向量的 L2 范数，用于监测训练稳定性和梯度消失/爆炸问题
