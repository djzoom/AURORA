## Transformer 注意力

### 多头注意力 (Multi-Head Attention / MHA)
- **英文全称**: Multi-Head Attention
- **缩写**: MHA
- **定义**: 将注意力机制并行运行于多个独立子空间，各头独立学习不同相关性模式，最后拼接输出并线性投影
- **出现课程**: L83, L91
- **标注状态**: ⚠️ 部分标注（首次标注于 L83）

### 缩放点积注意力 (Scaled Dot-Product Attention / SDPA)
- **英文全称**: Scaled Dot-Product Attention
- **缩写**: SDPA
- **定义**: Q·Kᵀ / √d_k 后经 softmax 归一化再加权聚合 V 的注意力计算核心公式
- **出现课程**: L83, L85, L91
- **标注状态**: ⚠️ 部分标注（首次标注于 L83）

### 查询 (Query)
- **英文全称**: Query
- **缩写**: Q
- **定义**: 注意力机制中表示「当前位置想要查什么」的投影向量
- **出现课程**: L83
- **标注状态**: ✅ 已标注（首次标注于 L83）

### 键 (Key)
- **英文全称**: Key
- **缩写**: K
- **定义**: 注意力机制中每个位置提供的「索引标签」向量，与 Query 做点积计算相似度
- **出现课程**: L83
- **标注状态**: ✅ 已标注（首次标注于 L83）

### 值 (Value)
- **英文全称**: Value
- **缩写**: V
- **定义**: 注意力机制中每个位置实际携带的信息向量，按注意力权重加权求和后输出
- **出现课程**: L83
- **标注状态**: ✅ 已标注（首次标注于 L83）

### 位置编码 (Positional Encoding)
- **英文全称**: Positional Encoding
- **缩写**: PE
- **定义**: 向序列嵌入中注入位置信息的机制，弥补 Transformer 无法感知 token 顺序的缺陷
- **出现课程**: L83
- **标注状态**: ❌ 缺失标注

### 缩放因子 (Scaling Factor)
- **英文全称**: Scaling Factor
- **缩写**: 无
- **定义**: 注意力计算中的除数 1/√d_k，将点积方差归一化为 1，防止 softmax 进入饱和区
- **出现课程**: L83
- **标注状态**: ❌ 缺失标注

### 因果掩码 (Causal Mask)
- **英文全称**: Causal Mask
- **缩写**: 无
- **定义**: 上三角布尔矩阵，将未来位置的注意力分数置为 -inf，保证自回归生成时当前 token 只看历史
- **出现课程**: L83
- **标注状态**: ❌ 缺失标注

### 数值稳定 (Numerical Stability)
- **英文全称**: Numerical Stability
- **缩写**: 无
- **定义**: 通过减去最大值等操作防止浮点计算溢出或下溢的工程技巧，常见于 softmax 实现
- **出现课程**: L83, L85
- **标注状态**: ❌ 缺失标注

### 自注意力 (Self-Attention)
- **英文全称**: Self-Attention
- **缩写**: 无
- **定义**: 序列中每个 token 对序列内所有 token（包括自身）计算注意力权重的机制
- **出现课程**: L91
- **标注状态**: ❌ 缺失标注

### 注意力权重 (Attention Weights)
- **英文全称**: Attention Weights
- **缩写**: 无
- **定义**: softmax 归一化后的 (seq×seq) 矩阵，元素 A[i,j] 表示第 i 个 token 对第 j 个 token 的关注程度
- **出现课程**: L91
- **标注状态**: ❌ 缺失标注

### 热力图 (Heatmap)
- **英文全称**: Heatmap
- **缩写**: 无
- **定义**: 用颜色深浅可视化矩阵数值分布的图表，本模块用于展示注意力权重矩阵
- **出现课程**: L91
- **标注状态**: ❌ 缺失标注

## LoRA 参数高效微调

### 低秩适配 (Low-Rank Adaptation / LoRA)
- **英文全称**: Low-Rank Adaptation
- **缩写**: LoRA
- **定义**: 用两个低秩矩阵 B@A 近似权重更新量 ΔW 的参数高效微调方法，仅训练 A 和 B
- **出现课程**: L84
- **标注状态**: ✅ 已标注（首次标注于 L84）

### 低秩分解 (Low-Rank Decomposition)
- **英文全称**: Low-Rank Decomposition
- **缩写**: 无
- **定义**: 将矩阵近似表示为两个小矩阵乘积的分解方式，可大幅减少参数数量
- **出现课程**: L84, L91
- **标注状态**: ❌ 缺失标注

### 全参数微调 (Full Parameter Fine-Tuning)
- **英文全称**: Full Parameter Fine-Tuning
- **缩写**: 无
- **定义**: 更新预训练模型全部参数的微调方式，显存和计算需求高
- **出现课程**: L84
- **标注状态**: ❌ 缺失标注

### 预训练权重 (Pre-Trained Weights)
- **英文全称**: Pre-trained Weights
- **缩写**: 无
- **定义**: 在大规模数据上预先训练好的模型参数，LoRA 中这些参数被冻结
- **出现课程**: L84, L85
- **标注状态**: ❌ 缺失标注

### 冻结 (Freeze)
- **英文全称**: Freeze
- **缩写**: 无
- **定义**: 将模型参数设置为不参与梯度计算和更新，在 LoRA 中用于保留预训练知识
- **出现课程**: L84
- **标注状态**: ❌ 缺失标注

### 显存 (VRAM)
- **英文全称**: Video RAM
- **缩写**: VRAM
- **定义**: GPU 显卡的高速内存，用于存储模型参数、梯度和激活值
- **出现课程**: L84, L87
- **标注状态**: ❌ 缺失标注

### 可训练参数 (Trainable Parameters)
- **英文全称**: Trainable Parameters
- **缩写**: 无
- **定义**: 训练过程中参与梯度更新的模型参数，LoRA 中仅 A 和 B 矩阵为可训练参数
- **出现课程**: L84, L91
- **标注状态**: ❌ 缺失标注

## KV 缓存与推理

### 键值缓存 (Key-Value Cache / KV-Cache)
- **英文全称**: Key-Value Cache
- **缩写**: KV-Cache
- **定义**: 存储每层历史 token 的 K/V 投影向量，推理时逐步 append 以避免重复计算，将单步复杂度从 O(t²) 降至 O(t)
- **出现课程**: L85
- **标注状态**: ✅ 已标注（首次标注于 L85）

### 自回归生成 (Autoregressive Generation)
- **英文全称**: Autoregressive Generation
- **缩写**: AR
- **定义**: 逐 token 生成序列的方式，第 t 步将已生成的 0..t-1 个 token 全部作为上下文输入
- **出现课程**: L85
- **标注状态**: ❌ 缺失标注

## 采样与解码

### 贪婪解码 (Greedy Decoding)
- **英文全称**: Greedy Decoding
- **缩写**: 无
- **定义**: 每步选择概率最高 token 的确定性生成策略，易卡入重复局部最优
- **出现课程**: L86
- **标注状态**: ❌ 缺失标注

### 温度缩放 (Temperature Scaling)
- **英文全称**: Temperature Scaling
- **缩写**: 无
- **定义**: 将 logits 除以温度 T 以调节分布尖锐度：T<1 更尖锐趋向贪婪，T>1 更平坦增加随机性
- **出现课程**: L86
- **标注状态**: ❌ 缺失标注

### 核采样 (Nucleus Sampling)
- **英文全称**: Nucleus Sampling
- **缩写**: Top-p
- **定义**: 只保留累积概率 ≥ p 的最小 token 集合再采样，候选集大小随模型置信度自适应调整
- **出现课程**: L86
- **标注状态**: ❌ 缺失标注

## 量化

### 量化 (Quantization)
- **英文全称**: Quantization
- **缩写**: 无
- **定义**: 将浮点权重压缩为低精度整数（如 INT8）以节省显存的技术，以量化误差换取内存效率
- **出现课程**: L01, L87
- **标注状态**: ❌ 缺失标注

### 反量化 (Dequantization)
- **英文全称**: Dequantization
- **缩写**: 无
- **定义**: 将量化后的整数表示还原为浮点数的逆操作：(x_q − zero_point) × scale
- **出现课程**: L87
- **标注状态**: ❌ 缺失标注

### 量化误差 (Quantization Error)
- **英文全称**: Quantization Error
- **缩写**: 无
- **定义**: 量化引入的近似误差，约等于 scale/2，比特数越少误差越大
- **出现课程**: L87
- **标注状态**: ❌ 缺失标注

## 检索增强生成 (RAG)

### 词频逆文档频率 (TF-IDF)
- **英文全称**: Term Frequency-Inverse Document Frequency
- **缩写**: TF-IDF
- **定义**: 综合考虑词在单个文档内的频率（TF）和跨文档稀有程度（IDF）的词权重指标
- **出现课程**: L88
- **标注状态**: ✅ 已标注（首次标注于 L88）

### 词频 (Term Frequency)
- **英文全称**: Term Frequency
- **缩写**: TF
- **定义**: 词 t 在文档 d 中的出现次数除以文档总词数，反映词在文档内的相对重要性
- **出现课程**: L88
- **标注状态**: ✅ 已标注（首次标注于 L88）

### 逆文档频率 (IDF)
- **英文全称**: Inverse Document Frequency
- **缩写**: IDF
- **定义**: log((N+1)/(df+1))+1，包含该词的文档越多权重越低，用于抑制常见词干扰
- **出现课程**: L88
- **标注状态**: ✅ 已标注（首次标注于 L88）

### 余弦相似度 (Cosine Similarity)
- **英文全称**: Cosine Similarity
- **缩写**: 无
- **定义**: 两向量夹角余弦值，衡量方向相似度而与向量模长无关，用于 TF-IDF 检索排序
- **出现课程**: L88, L89
- **标注状态**: ❌ 缺失标注

### 稀疏表示 (Sparse Representation)
- **英文全称**: Sparse Representation
- **缩写**: 无
- **定义**: 向量中大多数维度为零的表示方式，TF-IDF 向量是典型的稀疏表示
- **出现课程**: L88
- **标注状态**: ❌ 缺失标注

### 词汇表 (Vocabulary)
- **英文全称**: Vocabulary
- **缩写**: vocab
- **定义**: 检索系统或语言模型所使用的全部唯一词汇集合，决定向量空间维度
- **出现课程**: L88
- **标注状态**: ❌ 缺失标注

### 检索增强生成 (RAG)
- **英文全称**: Retrieval-Augmented Generation
- **缩写**: RAG
- **定义**: 将外部知识库检索结果注入提示词上下文以辅助生成的技术框架，解耦记忆检索与语言生成
- **出现课程**: L89, L90
- **标注状态**: ✅ 已标注（首次标注于 L89）

### 文档切片 (Document Chunking)
- **英文全称**: Document Chunking
- **缩写**: 无
- **定义**: 将长文档切割为较小文本块（chunk）的预处理步骤，通常带有重叠以保持上下文连贯
- **出现课程**: L89, L90
- **标注状态**: ❌ 缺失标注

### 提示词 (Prompt)
- **英文全称**: Prompt
- **缩写**: 无
- **定义**: 输入给语言模型的指令和上下文文本，RAG 中包含系统提示、检索片段、历史和当前问题
- **出现课程**: L89, L90
- **标注状态**: ❌ 缺失标注

### 来源归因 (Source Attribution)
- **英文全称**: Source Attribution
- **缩写**: 无
- **定义**: 将生成答案追溯到具体检索来源文档和时间戳的机制，提高答案可验证性
- **出现课程**: L90
- **标注状态**: ✅ 已标注（首次标注于 L90）

### 会话记忆 (Conversation Memory)
- **英文全称**: Conversation Memory
- **缩写**: 无
- **定义**: 保存历史对话轮次以维持上下文连贯性的机制，通常结合滑动窗口限制 token 用量
- **出现课程**: L90
- **标注状态**: ❌ 缺失标注

### 滑动窗口 (Sliding Window)
- **英文全称**: Sliding Window
- **缩写**: 无
- **定义**: 只保留最近 k 轮对话历史的截断策略，防止历史过长超出 LLM 的上下文长度限制
- **出现课程**: L90
- **标注状态**: ❌ 缺失标注

### 幻觉 (Hallucination)
- **英文全称**: Hallucination
- **缩写**: 无
- **定义**: 语言模型生成与事实不符内容的现象，RAG 通过注入真实检索片段来抑制幻觉
- **出现课程**: L90
- **标注状态**: ❌ 缺失标注

### 嵌入 (Embedding)
- **英文全称**: Embedding
- **缩写**: 无
- **定义**: 将文本映射为稠密实数向量的表示学习技术，向量间距离反映语义相似度
- **出现课程**: L90
- **标注状态**: ❌ 缺失标注

### 向量索引 (Vector Index)
- **英文全称**: Vector Index
- **缩写**: 无
- **定义**: 存储文档嵌入向量以支持快速近似最近邻检索的数据结构
- **出现课程**: L90
- **标注状态**: ❌ 缺失标注
