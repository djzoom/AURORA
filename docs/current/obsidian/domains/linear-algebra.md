## 向量与基本运算

### 向量 (Vector)
- **英文全称**: vector
- **缩写**: 无
- **定义**: 有序数列，表示空间中的方向和大小，NumPy 用一维 array 表示
- **出现课程**: L09–L21
- **标注状态**: ❌ 缺失标注

### 线性组合 (Linear Combination)
- **英文全称**: linear combination
- **缩写**: 无
- **定义**: 用若干系数对若干向量加权求和，覆盖所有可能结果构成向量空间
- **出现课程**: L09
- **标注状态**: ❌ 缺失标注

### 标量缩放 (Scalar Scaling)
- **英文全称**: scalar scaling
- **缩写**: 无
- **定义**: 将向量每个分量乘以同一标量 c，方向不变（c>0时），长度变为原来 |c| 倍
- **出现课程**: L09
- **标注状态**: ❌ 缺失标注

### 点积 (Dot Product)
- **英文全称**: dot product
- **缩写**: 无
- **定义**: 两等维向量逐元素相乘后求和，结果为标量；几何上等于 |a|·|b|·cos(θ)
- **出现课程**: L10, L12, L13, L19, L21
- **标注状态**: ❌ 缺失标注

### 余弦相似度 (Cosine Similarity)
- **英文全称**: cosine similarity
- **缩写**: 无
- **定义**: 点积除以两向量模之积，衡量方向对齐程度，范围 [-1, 1]，用于推荐系统和特征匹配
- **出现课程**: L10, L11
- **标注状态**: ❌ 缺失标注

### 投影 (Projection)
- **英文全称**: projection
- **缩写**: 无
- **定义**: 将向量沿某方向分解，得到该方向上的分量
- **出现课程**: L10
- **标注状态**: ❌ 缺失标注

### 注意力机制 (Attention Mechanism)
- **英文全称**: attention mechanism
- **缩写**: 无
- **定义**: 神经网络中用 QK 点积打分的加权聚合机制，L10 中作为点积的 Aurora 应用场景提及
- **出现课程**: L10
- **标注状态**: ❌ 缺失标注

## 范数与距离

### L2 范数 (L2 Norm)
- **英文全称**: L2 norm / Euclidean norm
- **缩写**: L2
- **定义**: 各分量平方和再开根，几何上等于向量的欧氏长度
- **出现课程**: L11, L13, L14
- **标注状态**: ✅ 已标注（首次标注于 L11）

### L1 范数 (L1 Norm)
- **英文全称**: L1 norm
- **缩写**: L1
- **定义**: 各分量绝对值之和，对离群值比 L2 更鲁棒
- **出现课程**: L11
- **标注状态**: ✅ 已标注（首次标注于 L11）

### L∞ 范数 (L-infinity Norm)
- **英文全称**: L-infinity norm
- **缩写**: L∞
- **定义**: 各分量绝对值中的最大值
- **出现课程**: L11
- **标注状态**: ✅ 已标注（首次标注于 L11）

### 归一化 (Normalization)
- **英文全称**: normalization
- **缩写**: 无
- **定义**: 将向量除以其 L2 范数，使输出长度为 1，消除幅度影响，仅保留方向信息
- **出现课程**: L11, L13
- **标注状态**: ❌ 缺失标注

### 欧氏距离 (Euclidean Distance)
- **英文全称**: Euclidean distance
- **缩写**: 无
- **定义**: 差向量的 L2 范数，度量两点在欧几里得空间中的直线距离
- **出现课程**: L11
- **标注状态**: ❌ 缺失标注

### 说话人识别 (Speaker Recognition)
- **英文全称**: speaker recognition
- **缩写**: 无
- **定义**: 通过比对音频特征向量（d-vector）的方向（余弦相似度）来判断身份的任务
- **出现课程**: L11
- **标注状态**: ❌ 缺失标注

### 梅尔倒谱系数 (MFCC)
- **英文全称**: mel-frequency cepstral coefficients
- **缩写**: MFCC
- **定义**: 将每帧音频压缩成低维特征向量（如13维），用于语音识别和说话人识别
- **出现课程**: L09, L11, L21
- **标注状态**: ⚠️ 部分标注（L09 出现中文形式，其他处仅用缩写 MFCC，两者从未在同一处配对）

## 矩阵运算

### 线性变换 (Linear Transformation)
- **英文全称**: linear transformation
- **缩写**: 无
- **定义**: 保持加法和标量乘法性质的映射，矩阵是其数值表示
- **出现课程**: L12, L19
- **标注状态**: ❌ 缺失标注

### 旋转矩阵 (Rotation Matrix)
- **英文全称**: rotation matrix
- **缩写**: 无
- **定义**: 正交矩阵的一类，行列式为 +1，将向量旋转指定角度且保持长度不变
- **出现课程**: L12, L13, L19
- **标注状态**: ❌ 缺失标注

### 正交矩阵 (Orthogonal Matrix)
- **英文全称**: orthogonal matrix
- **缩写**: 无
- **定义**: 列向量两两正交且单位长度的方阵，满足 Q^T·Q = I，转置即逆，保持向量长度不变
- **出现课程**: L13, L17, L20
- **标注状态**: ❌ 缺失标注

### 酉矩阵 (Unitary Matrix)
- **英文全称**: unitary matrix
- **缩写**: 无
- **定义**: 正交矩阵的复数推广，满足 Q^H·Q = I；DFT 矩阵属于此类，确保 ifft(fft(x))=x 成立
- **出现课程**: L13
- **标注状态**: ❌ 缺失标注

### 对称矩阵 (Symmetric Matrix)
- **英文全称**: symmetric matrix
- **缩写**: 无
- **定义**: 满足 A = A^T 的方阵，特征值全为实数，特征向量两两正交；协方差矩阵必须对称
- **出现课程**: L13, L17, L20
- **标注状态**: ❌ 缺失标注

### 正定 (Positive Definite)
- **英文全称**: positive definite
- **缩写**: PD
- **定义**: 对所有非零向量 x 均有 x^T·A·x > 0 的矩阵，特征值全为正，可用 Cholesky 分解检验
- **出现课程**: L13
- **标注状态**: ❌ 缺失标注

### 协方差矩阵 (Covariance Matrix)
- **英文全称**: covariance matrix
- **缩写**: 无
- **定义**: 描述多维随机变量各分量间线性相关性的对称半正定矩阵，PCA 和白化的基础
- **出现课程**: L13, L16
- **标注状态**: ❌ 缺失标注

### 逆矩阵 (Inverse Matrix)
- **英文全称**: inverse matrix
- **缩写**: A⁻¹
- **定义**: 满足 A·A⁻¹ = I 的矩阵，将变换撤销；仅在 det ≠ 0 时存在；正交矩阵的逆等于其转置
- **出现课程**: L13, L16
- **标注状态**: ❌ 缺失标注

### 行列式 (Determinant)
- **英文全称**: determinant
- **缩写**: det
- **定义**: 方阵变换对体积的缩放比例；det = 0 时矩阵奇异不可逆，det ≠ 0 时可逆
- **出现课程**: L16, L17, L18
- **标注状态**: ❌ 缺失标注

### 余子式 (Minor)
- **英文全称**: minor
- **缩写**: 无
- **定义**: 划去矩阵第 i 行第 j 列后的子矩阵的行列式；L16 中以"子式/minor"形式标注
- **出现课程**: L16
- **标注状态**: ✅ 已标注（首次标注于 L16）

### 代数余子式 (Cofactor)
- **英文全称**: cofactor
- **缩写**: 无
- **定义**: 余子式乘以符号因子 (-1)^{i+j}，用于行列式按行/列展开及伴随矩阵构造
- **出现课程**: L16
- **标注状态**: ✅ 已标注（首次标注于 L16）

### 伴随矩阵 (Adjugate Matrix)
- **英文全称**: adjugate matrix
- **缩写**: adj
- **定义**: 由各代数余子式转置排列成的矩阵，满足 A⁻¹ = adj(A)/det(A)；L16 中以"伴随 adjoint"形式标注
- **出现课程**: L16
- **标注状态**: ✅ 已标注（首次标注于 L16）

### 白化 (Whitening)
- **英文全称**: whitening
- **缩写**: 无
- **定义**: 利用协方差矩阵逆平方根 Σ^{-1/2} 对特征去相关并归一化方差，要求协方差矩阵可逆
- **出现课程**: L16
- **标注状态**: ❌ 缺失标注

## 线性系统

### 线性系统 (Linear System)
- **英文全称**: linear system
- **缩写**: 无
- **定义**: 形如 Ax = b 的方程组，A 为系数矩阵，x 为未知向量；LPC 声音建模即为此形式
- **出现课程**: L15
- **标注状态**: ❌ 缺失标注

### 高斯消元 (Gaussian Elimination)
- **英文全称**: Gaussian elimination
- **缩写**: 无
- **定义**: 通过初等行变换将增广矩阵化为行阶梯形，逐列消去系数以求解线性方程组
- **出现课程**: L15, L20
- **标注状态**: ❌ 缺失标注

### 增广矩阵 (Augmented Matrix)
- **英文全称**: augmented matrix
- **缩写**: 无
- **定义**: 将系数矩阵 A 和右端向量 b 横向拼接得到的 [A|b] 矩阵，用于高斯消元
- **出现课程**: L15
- **标注状态**: ❌ 缺失标注

### 行阶梯形 (Row Echelon Form)
- **英文全称**: row echelon form
- **缩写**: REF
- **定义**: 经初等行变换后每行最左非零元素位于上一行之右的矩阵形式，便于回代求解
- **出现课程**: L15
- **标注状态**: ❌ 缺失标注

### 最小二乘法 (Least Squares Method)
- **英文全称**: least squares method
- **缩写**: LS
- **定义**: 当方程组超定（无精确解）时，求使残差 ||Ax-b||² 最小的 x；LPC 系数估计依赖此方法
- **出现课程**: L15
- **标注状态**: ❌ 缺失标注

### 超定系统 (Overdetermined System)
- **英文全称**: overdetermined system
- **缩写**: 无
- **定义**: 方程数多于未知数（m>n）的线性系统，一般无精确解，用最小二乘法求近似解
- **出现课程**: L15
- **标注状态**: ❌ 缺失标注

### 线性预测编码 (LPC)
- **英文全称**: linear predictive coding
- **缩写**: LPC
- **定义**: 用线性方程组 Ax = b 建模声音，通过最小二乘法估计滤波器系数
- **出现课程**: L15
- **标注状态**: ❌ 缺失标注

### 秩 (Rank)
- **英文全称**: rank
- **缩写**: 无
- **定义**: 矩阵线性无关行（或列）的数目，等于非零奇异值的个数，决定线性方程组解的类型
- **出现课程**: L15, L18
- **标注状态**: ❌ 缺失标注

## 特征值与分解

### 特征值 (Eigenvalue)
- **英文全称**: eigenvalue
- **缩写**: λ
- **定义**: 满足 Av = λv 的标量 λ，衡量矩阵沿特征方向的缩放比例；决定递归滤波器稳定性
- **出现课程**: L01, L14, L17, L18
- **标注状态**: ❌ 缺失标注

### 特征向量 (Eigenvector)
- **英文全称**: eigenvector
- **缩写**: 无
- **定义**: 满足 Av = λv 的非零向量 v，经矩阵变换后方向不变，仅长度按 λ 缩放
- **出现课程**: L14, L17
- **标注状态**: ❌ 缺失标注

### 特征多项式 (Characteristic Polynomial)
- **英文全称**: characteristic polynomial
- **缩写**: 无
- **定义**: det(A - λI) 关于 λ 的多项式，其根即为矩阵 A 的特征值
- **出现课程**: L17
- **标注状态**: ❌ 缺失标注

### 特征方程 (Characteristic Equation)
- **英文全称**: characteristic equation
- **缩写**: 无
- **定义**: det(A - λI) = 0，求解此方程得到所有特征值
- **出现课程**: L17
- **标注状态**: ❌ 缺失标注

### 对角化 (Diagonalization)
- **英文全称**: diagonalization
- **缩写**: 无
- **定义**: 将矩阵写成 A = P·Λ·P⁻¹ 的分解，使得矩阵幂次计算退化为对角线上标量的幂次
- **出现课程**: L17, L20
- **标注状态**: ❌ 缺失标注

### 正交对角化 (Orthogonal Diagonalization)
- **英文全称**: orthogonal diagonalization
- **缩写**: 无
- **定义**: 用归一化特征向量构成正交矩阵 P，使得 P^T·A·P = Λ；仅对对称矩阵保证成立
- **出现课程**: L17
- **标注状态**: ❌ 缺失标注

### 特征分解 (Eigendecomposition)
- **英文全称**: eigendecomposition
- **缩写**: 无
- **定义**: 将方阵分解为 A = P·Λ·P⁻¹，P 为特征向量矩阵，Λ 为特征值对角阵
- **出现课程**: L09, L14, L17, L20
- **标注状态**: ❌ 缺失标注

### 奇异值 (Singular Value)
- **英文全称**: singular value
- **缩写**: σ
- **定义**: SVD 分解中矩阵 Σ 的对角元素，衡量矩阵在各方向上的拉伸强度，全为非负实数
- **出现课程**: L14, L18, L20
- **标注状态**: ❌ 缺失标注

### 奇异值分解 (SVD)
- **英文全称**: singular value decomposition
- **缩写**: SVD
- **定义**: 将任意矩阵分解为 A = U·Σ·V^T，U 和 V 为正交矩阵，Σ 为奇异值对角阵
- **出现课程**: L09, L14, L18, L20, L21
- **标注状态**: ⚠️ 部分标注（L20 中首次出现完整标注"奇异值分解 SVD"，其他处仅用缩写）

### 低秩近似 (Low-Rank Approximation)
- **英文全称**: low-rank approximation
- **缩写**: 无
- **定义**: 截断 SVD 中最小奇异值对应的分量，用前 k 个奇异值重建矩阵，是 LoRA 和推荐系统的核心操作
- **出现课程**: L14
- **标注状态**: ❌ 缺失标注

### 主成分分析 (PCA)
- **英文全称**: principal component analysis
- **缩写**: PCA
- **定义**: 通过协方差矩阵特征分解找到数据变化最大的正交方向，用于降维和可视化
- **出现课程**: L13, L14
- **标注状态**: ⚠️ 部分标注（仅以缩写 PCA 出现，中文全称在本模块从未明确写出）

## 可逆性与数值稳定性

### 零空间 (Null Space)
- **英文全称**: null space
- **缩写**: 无
- **定义**: 矩阵 A 映射到零向量的所有输入向量构成的集合，对应 SVD 中零奇异值的右奇异向量
- **出现课程**: L18
- **标注状态**: ❌ 缺失标注

### 谱半径 (Spectral Radius)
- **英文全称**: spectral radius
- **缩写**: 无
- **定义**: 矩阵所有特征值绝对值的最大值；谱半径 < 1 是迭代格式收敛的充要条件
- **出现课程**: L18
- **标注状态**: ❌ 缺失标注

### 严格对角占优 (Strictly Diagonally Dominant)
- **英文全称**: strictly diagonally dominant
- **缩写**: s.d.d.
- **定义**: 每行对角元绝对值严格大于该行其余元素绝对值之和，是 Jacobi/Gauss-Seidel 迭代收敛的充分条件
- **出现课程**: L18
- **标注状态**: ✅ 已标注（首次标注于 L18）

### 条件数 (Condition Number)
- **英文全称**: condition number
- **缩写**: κ (kappa)
- **定义**: 最大奇异值与最小奇异值之比（σ_max/σ_min），衡量矩阵数值不稳定程度；越大求逆误差越大
- **出现课程**: L14, L15, L16, L20
- **标注状态**: ❌ 缺失标注

### 递归滤波器 (Recursive Filter)
- **英文全称**: recursive filter
- **缩写**: 无
- **定义**: 输出依赖前序输出的数字滤波器，其稳定性由系统矩阵特征值的模是否全小于 1 决定
- **出现课程**: L17
- **标注状态**: ❌ 缺失标注

### 状态空间模型 (State Space Model)
- **英文全称**: state space model
- **缩写**: 无
- **定义**: 用矩阵方程描述动态系统的内部状态与输入输出关系，aurora.dsp 极点分析依赖此框架
- **出现课程**: L17
- **标注状态**: ❌ 缺失标注

## 矩阵分解

### LU 分解 (LU Decomposition)
- **英文全称**: LU decomposition
- **缩写**: LU
- **定义**: 将矩阵分解为下三角矩阵 L 与上三角矩阵 U 之积，是高斯消元的矩阵形式
- **出现课程**: L20
- **标注状态**: ⚠️ 部分标注（L20 以"A=LU（高斯消元的矩阵形式）"出现，LU 为标准符号但无单独中文全称）

### QR 分解 (QR Decomposition)
- **英文全称**: QR decomposition
- **缩写**: QR
- **定义**: 将矩阵分解为正交矩阵 Q 与上三角矩阵 R 之积，基于 Gram-Schmidt 正交化过程
- **出现课程**: L20
- **标注状态**: ⚠️ 部分标注（L20 以"A=QR（Gram-Schmidt正交化）"出现）

### 谱定理 (Spectral Theorem)
- **英文全称**: spectral theorem
- **缩写**: 无
- **定义**: 对称矩阵可正交对角化为 S = Q·Λ·Q^T，特征值全为实数，特征向量两两正交
- **出现课程**: L20, L21
- **标注状态**: ❌ 缺失标注

### 离散傅里叶变换 (DFT)
- **英文全称**: discrete Fourier transform
- **缩写**: DFT
- **定义**: 将离散时域信号变换到频域的线性变换；L21 中出现中文"离散傅里叶变换"但无 DFT 标注
- **出现课程**: L10, L12, L13, L21
- **标注状态**: ⚠️ 部分标注

### mel 滤波器组 (Mel Filterbank)
- **英文全称**: mel filterbank
- **缩写**: 无
- **定义**: 按 mel 刻度排列的一组三角滤波器，构成矩阵 M，将 FFT 幅度谱压缩为 mel 频谱
- **出现课程**: L12, L21
- **标注状态**: ❌ 缺失标注
