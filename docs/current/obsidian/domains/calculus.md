## 微分基础

### 导数 (Derivative)
- **英文全称**: derivative
- **缩写**: 无
- **定义**: 函数在某点输入微小变化时输出的变化比值，即曲线在该点的斜率
- **出现课程**: L01, L22, L23, L24, L25, L26
- **标注状态**: ❌ 缺失标注

### 切线 (Tangent Line)
- **英文全称**: tangent line
- **缩写**: 无
- **定义**: 函数曲线在某点处的局部线性近似，斜率等于该点导数值
- **出现课程**: L22, L26
- **标注状态**: ❌ 缺失标注

### 切线斜率 (Tangent Slope)
- **英文全称**: tangent slope
- **缩写**: 无
- **定义**: 切线与水平轴所成角度的正切值，等于该点的导数值
- **出现课程**: L22, L26
- **标注状态**: ❌ 缺失标注

### 极限定义 (Limit Definition)
- **英文全称**: limit definition
- **缩写**: 无
- **定义**: 用极限形式 lim_{h→0}(f(x+h)-f(x))/h 严格定义导数的方式
- **出现课程**: L22
- **标注状态**: ❌ 缺失标注

### 泰勒展开 (Taylor Expansion)
- **英文全称**: Taylor expansion
- **缩写**: 无
- **定义**: 将函数在某点展开为多项式级数，用于分析差分近似的截断误差阶数
- **出现课程**: L22
- **标注状态**: ❌ 缺失标注

## 数值微分

### 数值微分 (Numerical Differentiation)
- **英文全称**: numerical differentiation
- **缩写**: 无
- **定义**: 用有限差分近似代替解析求导的数值计算方法
- **出现课程**: L22, L56
- **标注状态**: ❌ 缺失标注

### 解析微分 (Analytical Differentiation)
- **英文全称**: analytical differentiation
- **缩写**: 无
- **定义**: 按导数公式直接推导出精确闭合表达式的求导方式
- **出现课程**: L22
- **标注状态**: ❌ 缺失标注

### 中心差分 (Central Difference)
- **英文全称**: central difference
- **缩写**: CD
- **定义**: 用 (f(x+h)-f(x-h))/(2h) 近似导数，截断误差为 O(h²)，优于单侧差分
- **出现课程**: L22, L23, L24, L25, L26
- **标注状态**: ❌ 缺失标注

### 前向差分 (Forward Difference)
- **英文全称**: forward difference
- **缩写**: 无
- **定义**: 用 (f(x+h)-f(x))/h 近似导数，截断误差为 O(h)，精度低于中心差分
- **出现课程**: L22
- **标注状态**: ❌ 缺失标注

### 截断误差 (Truncation Error)
- **英文全称**: truncation error
- **缩写**: 无
- **定义**: 因差分公式有限阶近似而引入的系统性误差，随步长 h 缩小而减小
- **出现课程**: L22
- **标注状态**: ❌ 缺失标注

### 舍入误差 (Rounding Error)
- **英文全称**: rounding error
- **缩写**: 无
- **定义**: 浮点数有限精度导致的计算误差，步长 h 过小时因相减消去而急剧增大
- **出现课程**: L22
- **标注状态**: ❌ 缺失标注

### 浮点数 (Floating-Point Number)
- **英文全称**: floating-point number
- **缩写**: 无
- **定义**: 计算机中用有效位和指数表示实数的数据格式，双精度有效位约 16 位
- **出现课程**: L22
- **标注状态**: ❌ 缺失标注

### 梯度检验 (Gradient Check)
- **英文全称**: gradient check
- **缩写**: 无
- **定义**: 将解析梯度与数值梯度逐元素比对，验证反向传播实现正确性的标准工具
- **出现课程**: L22, L23
- **标注状态**: ❌ 缺失标注

## 多元微分

### 梯度 (Gradient)
- **英文全称**: gradient
- **缩写**: grad / ∇
- **定义**: 多元函数对每个输入变量的偏导数组成的向量，指向函数值增长最快的方向
- **出现课程**: L01, L22, L23, L24, L25, L26
- **标注状态**: ❌ 缺失标注

### 偏导 (Partial Derivative)
- **英文全称**: partial derivative
- **缩写**: ∂f/∂x
- **定义**: 多元函数对某一变量的导数，计算时将其他变量视为常数
- **出现课程**: L23, L24
- **标注状态**: ❌ 缺失标注

### 梯度向量 (Gradient Vector)
- **英文全称**: gradient vector
- **缩写**: 无
- **定义**: 所有偏导数按顺序排列成的向量，方向指向函数值增大最快的方向
- **出现课程**: L23
- **标注状态**: ❌ 缺失标注

### 多元函数 (Multivariate Function)
- **英文全称**: multivariate function
- **缩写**: 无
- **定义**: 接受两个或更多输入变量的函数，如神经网络损失关于所有权重的函数
- **出现课程**: L23
- **标注状态**: ❌ 缺失标注

### 等高线 (Contour Line)
- **英文全称**: contour line
- **缩写**: 无
- **定义**: 二维函数值相等的点连成的曲线，梯度方向始终与等高线垂直
- **出现课程**: L23, L26
- **标注状态**: ❌ 缺失标注

### 梅尔滤波器 (Mel Filterbank)
- **英文全称**: mel filterbank
- **缩写**: 无
- **定义**: 按梅尔频率尺度排布的三角形滤波器组，用于将频谱映射到感知相关的频率轴
- **出现课程**: L22, L23
- **标注状态**: ❌ 缺失标注

### MFCC (Mel-Frequency Cepstral Coefficients)
- **英文全称**: Mel-Frequency Cepstral Coefficients
- **缩写**: MFCC
- **定义**: 梅尔频率倒谱系数，通过梅尔滤波器和 DCT 提取的音频特征，需要验证其求导实现
- **出现课程**: L22, L23
- **标注状态**: ⚠️ 部分标注（首次标注于 L22）

### 数值梯度 (Numerical Gradient)
- **英文全称**: numerical gradient
- **缩写**: 无
- **定义**: 对每个参数维度施加 ±h 扰动、用中心差分计算得到的近似梯度
- **出现课程**: L22, L23
- **标注状态**: ❌ 缺失标注

### 解析梯度 (Analytical Gradient)
- **英文全称**: analytical gradient
- **缩写**: 无
- **定义**: 由导数公式直接推导的精确梯度，与数值梯度比对用于验证求导实现
- **出现课程**: L22
- **标注状态**: ❌ 缺失标注

## 反向传播

### 链式法则 (Chain Rule)
- **英文全称**: chain rule
- **缩写**: 无
- **定义**: 复合函数求导规则：dy/dx = f'(g(x))·g'(x)，是反向传播的数学基础
- **出现课程**: L01, L23, L24
- **标注状态**: ❌ 缺失标注

### 计算图 (Computation Graph)
- **英文全称**: computation graph
- **缩写**: 无
- **定义**: 将复合函数的运算表示为有向无环图，每条边缓存局部导数供反向传播使用
- **出现课程**: L23, L24
- **标注状态**: ❌ 缺失标注

### 局部梯度 (Local Gradient)
- **英文全称**: local gradient
- **缩写**: 无
- **定义**: 计算图中每个节点缓存的当前层输出对输入的偏导，用于链式法则连乘
- **出现课程**: L24
- **标注状态**: ❌ 缺失标注

### 损失函数 (Loss Function)
- **英文全称**: loss function
- **缩写**: 无
- **定义**: 衡量模型预测与真实值差距的标量函数，训练目标是对其最小化
- **出现课程**: L23, L24, L25
- **标注状态**: ⚠️ 部分标注（首次标注于 L23）

### 反向传播 (Backpropagation)
- **英文全称**: backpropagation
- **缩写**: BP
- **定义**: 沿计算图连乘局部偏导将梯度从损失端传回所有权重的算法，是链式法则的计算实现
- **出现课程**: L22, L23, L24, L25
- **标注状态**: ⚠️ 部分标注（首次标注于 L23）

### autograd (自动微分)
- **英文全称**: automatic differentiation
- **缩写**: autograd
- **定义**: 自动微分引擎，通过记录前向传播操作构建计算图后自动执行反向传播
- **出现课程**: L23, L24
- **标注状态**: ⚠️ 部分标注（首次标注于 L23）

## 梯度下降与优化

### 梯度下降 (Gradient Descent)
- **英文全称**: gradient descent
- **缩写**: GD
- **定义**: 沿梯度反方向迭代更新参数使损失函数减小的基础优化方法：x_new = x − lr × grad
- **出现课程**: L01, L24, L25, L26
- **标注状态**: ❌ 缺失标注

### 学习率 (Learning Rate)
- **英文全称**: learning rate
- **缩写**: lr
- **定义**: 控制每步梯度下降移动距离的超参数；过大则震荡，过小则收敛慢
- **出现课程**: L25, L26
- **标注状态**: ❌ 缺失标注

### 权重更新 (Weight Update)
- **英文全称**: weight update
- **缩写**: 无
- **定义**: 用梯度和学习率计算新参数值的操作：w_new = w − lr × ∂L/∂w
- **出现课程**: L25
- **标注状态**: ❌ 缺失标注

### 线性回归 (Linear Regression)
- **英文全称**: linear regression
- **缩写**: 无
- **定义**: 用直线 y = wx + b 拟合数据的最简单监督学习模型，可用梯度下降求解
- **出现课程**: L25
- **标注状态**: ⚠️ 部分标注（首次标注于 L25）

### 均方误差 (MSE)
- **英文全称**: mean squared error
- **缩写**: MSE
- **定义**: 预测值与真实值之差的平方的均值，最常用的回归损失函数
- **出现课程**: L25
- **标注状态**: ⚠️ 部分标注（首次标注于 L25）

### 动量 (Momentum)
- **英文全称**: momentum
- **缩写**: 无
- **定义**: 优化器中累积历史梯度方向以加速收敛并抑制震荡的机制
- **出现课程**: L25
- **标注状态**: ❌ 缺失标注

### 自适应步长 (Adaptive Step Size)
- **英文全称**: adaptive step size
- **缩写**: 无
- **定义**: 根据每个参数的历史梯度自动调整学习率的优化器策略，如 Adam、RMSProp
- **出现课程**: L25
- **标注状态**: ❌ 缺失标注

### SGD (Stochastic Gradient Descent)
- **英文全称**: stochastic gradient descent
- **缩写**: SGD
- **定义**: 随机梯度下降，每步用单个样本或小批量估计梯度，是最基础的优化器
- **出现课程**: L25
- **标注状态**: ⚠️ 部分标注（首次标注于 L25）

### Adam (Adaptive Moment Estimation)
- **英文全称**: Adaptive Moment Estimation
- **缩写**: Adam
- **定义**: 结合动量和自适应步长的常用优化器，是梯度下降 x = x − lr·grad 的变体
- **出现课程**: L25
- **标注状态**: ⚠️ 部分标注（首次标注于 L25）

### RMSProp (Root Mean Square Propagation)
- **英文全称**: Root Mean Square Propagation
- **缩写**: RMSProp
- **定义**: 用梯度平方的移动平均自适应调整每个参数学习率的优化器
- **出现课程**: L25
- **标注状态**: ⚠️ 部分标注（首次标注于 L25）

## 损失曲面

### 鞍点 (Saddle Point)
- **英文全称**: saddle point
- **缩写**: 无
- **定义**: 梯度为零但非极小值的点，在高维损失曲面中比局部极小值更为常见
- **出现课程**: L26
- **标注状态**: ❌ 缺失标注

### 局部极小值 (Local Minimum)
- **英文全称**: local minimum
- **缩写**: 无
- **定义**: 函数在邻域内取得最小值的点，梯度下降可能停留于此而非全局最优
- **出现课程**: L26
- **标注状态**: ❌ 缺失标注
