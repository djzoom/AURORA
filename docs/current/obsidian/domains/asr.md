## 评估指标

### 词错率 (Word Error Rate / WER)
- **英文全称**: Word Error Rate
- **缩写**: WER
- **定义**: ASR 评估核心指标，WER = (S+D+I) / N，以词为单位衡量替换、删除、插入错误之和占参考词总数的比例
- **出现课程**: L66, L67, L72, L73
- **标注状态**: ✅ 已标注（首次标注于 L66）

### 替换/删除/插入 (S/D/I)
- **英文全称**: Substitution / Deletion / Insertion
- **缩写**: S / D / I
- **定义**: WER 三类错误：S=输出了错误词，D=漏掉了参考词，I=多输出了多余词
- **出现课程**: L66, L73, L74
- **标注状态**: ✅ 已标注（首次标注于 L66）

### 编辑距离 (Edit Distance / Levenshtein)
- **英文全称**: Levenshtein distance / edit distance
- **缩写**: 无
- **定义**: 把一个序列变成另一个序列所需的最少插入、删除、替换操作数，用动态规划 O(m×n) 求解，WER 的数学基础
- **出现课程**: L66, L67, L73, L74
- **标注状态**: ✅ 已标注（首次标注于 L66）

### 幻觉词 (Hallucination)
- **英文全称**: hallucination
- **缩写**: 无
- **定义**: ASR 模型凭空生成真实音频中并不存在的词，体现为大量插入错误 I，WER > 1.0 的主要原因
- **出现课程**: L73, L74, L75
- **标注状态**: ❌ 缺失标注

### 重尾分布 (Heavy-Tail Distribution)
- **英文全称**: heavy-tail distribution
- **缩写**: 无
- **定义**: 逐句 WER 的统计分布形态：大多数句子 WER 接近 0，少数句子 WER 极高，均值被长尾拉高，中位数更能描述典型表现
- **出现课程**: L73
- **标注状态**: ❌ 缺失标注

## 传统 ASR 流水线

### 语音识别 (ASR)
- **英文全称**: automatic speech recognition
- **缩写**: ASR
- **定义**: 将语音音频自动转录为文字的技术；Whisper 是目前最广泛使用的 ASR 模型
- **出现课程**: L01, L66
- **标注状态**: ❌ 缺失标注

### 声学模型 (Acoustic Model)
- **英文全称**: acoustic model
- **缩写**: AM
- **定义**: 传统 ASR 中建模声学特征与音素关系的模块，HMM-GMM 系统中负责输出每帧 HMM 状态后验概率
- **出现课程**: L66, L68
- **标注状态**: ❌ 缺失标注

### 语言模型 (Language Model)
- **英文全称**: language model
- **缩写**: LM
- **定义**: 对词序列概率 P(词序列) 建模的模块，传统系统中为 N-gram，端到端系统中隐式融入 Transformer 解码器
- **出现课程**: L66, L71, L72
- **标注状态**: ❌ 缺失标注

### 流水线 (Pipeline)
- **英文全称**: pipeline
- **缩写**: 无
- **定义**: 传统 ASR 的模块化处理链路：MFCC 特征提取 → GMM/HMM 声学模型 → 发音词典 → N-gram 语言模型
- **出现课程**: L66, L68
- **标注状态**: ❌ 缺失标注

### 发音词典 (Pronunciation Dictionary)
- **英文全称**: pronunciation dictionary / lexicon
- **缩写**: 无
- **定义**: 传统 ASR 流水线中将词映射到音素序列的查找表，连接声学模型与语言模型
- **出现课程**: L66
- **标注状态**: ❌ 缺失标注

### 端到端 (End-to-End)
- **英文全称**: end-to-end
- **缩写**: E2E
- **定义**: 将原本分段独立训练的声学模型、语言模型合并成一个大网络，直接学习 P(文字|音频)，梯度可在全链路反传
- **出现课程**: L66, L68, L70, L72
- **标注状态**: ❌ 缺失标注

### 音素 (Phoneme)
- **英文全称**: phoneme
- **缩写**: 无
- **定义**: 语音的最小区分意义的声音单位，声学模型的基本建模对象，口音差异主要体现为音素混淆
- **出现课程**: L66, L74
- **标注状态**: ❌ 缺失标注

### 词表 (Vocabulary)
- **英文全称**: vocabulary
- **缩写**: 无
- **定义**: 模型所有可能输出 token 的集合，Whisper 多语言模型词表大小为 51865，CTC 模型词表含 blank 符号
- **出现课程**: L66, L68, L69, L70
- **标注状态**: ❌ 缺失标注

### 后验概率 (Posterior Probability)
- **英文全称**: posterior probability
- **缩写**: 无
- **定义**: HMM-GMM 声学模型每帧输出各 HMM 状态的后验概率 P(state|frame)，是传统系统帧级打分的基础
- **出现课程**: L66
- **标注状态**: ❌ 缺失标注

### 动态规划 (Dynamic Programming)
- **英文全称**: dynamic programming
- **缩写**: DP
- **定义**: 通过拆分为重叠子问题并存储中间结果来高效求解最优化问题的算法范式，编辑距离、CTC 前向算法均依赖此技术
- **出现课程**: L66, L67, L68, L69, L73, L74
- **标注状态**: ❌ 缺失标注

### 特征提取 (Feature Extraction)
- **英文全称**: feature extraction
- **缩写**: 无
- **定义**: 将原始波形转换为模型输入特征的预处理步骤，ASR 中通常指 STFT → Mel 滤波器组 → log 变换
- **出现课程**: L66, L70
- **标注状态**: ❌ 缺失标注

## CTC

### CTC (Connectionist Temporal Classification)
- **英文全称**: Connectionist Temporal Classification
- **缩写**: CTC
- **定义**: 一种无需帧级对齐标注的序列训练目标，引入 blank 符号，对所有压缩后等于目标序列的路径概率求和作为训练信号
- **出现课程**: L68, L69, L75
- **标注状态**: ⚠️ 部分标注（首次完整标注于 L75）

### 贪婪解码 (Greedy Decode)
- **英文全称**: Greedy Decode
- **缩写**: 无
- **定义**: 每步直接取概率最大的 token 作为输出，复杂度 O(T×V)，速度最快但只考虑单条最优路径
- **出现课程**: L68, L71
- **标注状态**: ✅ 已标注（首次标注于 L68）

### 压缩规则 (Collapse)
- **英文全称**: collapse
- **缩写**: 无
- **定义**: CTC 路径压缩：先去掉相邻重复符号，再去掉所有 blank，得到最终输出序列
- **出现课程**: L68, L75
- **标注状态**: ✅ 已标注（首次标注于 L68）

### 格 (Lattice)
- **英文全称**: lattice
- **缩写**: 无
- **定义**: CTC 动态规划用到的有向无环图结构，节点为（时间步, 符号）对，前向算法在格上高效求所有合法路径的概率之和
- **出现课程**: L69, L75
- **标注状态**: ✅ 已标注（首次标注于 L75）

### 前向算法 (Forward Algorithm)
- **英文全称**: forward algorithm
- **缩写**: 无
- **定义**: CTC 中用动态规划在扩展标签格上递推计算所有合法路径概率之和的算法，复杂度 O(T×2L)
- **出现课程**: L68, L69
- **标注状态**: ❌ 缺失标注

### 前向变量 (Forward Variable)
- **英文全称**: forward variable
- **缩写**: α (alpha)
- **定义**: CTC 前向算法中 α[t][s] 表示时刻 t 所有到达扩展标签位置 s 的路径前缀概率之和（log 域存储）
- **出现课程**: L69
- **标注状态**: ❌ 缺失标注

### 扩展标签 (Extended Label)
- **英文全称**: extended label
- **缩写**: 无
- **定义**: CTC 前向算法中在每个目标字符之间插入 blank 构成的序列 l'，长度 S = 2L+1，定义合法路径的对齐约束
- **出现课程**: L69
- **标注状态**: ❌ 缺失标注

### 标签折叠 (Label Collapse)
- **英文全称**: label collapse
- **缩写**: 无
- **定义**: CTC 路径压缩操作：将含 blank 的帧级序列通过去相邻重复再去 blank 映射到目标标签序列
- **出现课程**: L68
- **标注状态**: ❌ 缺失标注

### 数值下溢 (Numerical Underflow)
- **英文全称**: numerical underflow
- **缩写**: 无
- **定义**: 多个小概率连乘导致浮点数趋近于零的问题，CTC 前向算法在 log 域进行递推以避免此问题
- **出现课程**: L69
- **标注状态**: ❌ 缺失标注

## Whisper 架构

### 编码器 (Encoder)
- **英文全称**: encoder
- **缩写**: 无
- **定义**: Whisper 中处理 log-Mel 帧序列并压缩为上下文向量的 Transformer 部分，输出 (1500, d_model) 形状的序列
- **出现课程**: L66, L70, L71, L72, L75
- **标注状态**: ❌ 缺失标注

### 解码器 (Decoder)
- **英文全称**: decoder
- **缩写**: 无
- **定义**: Whisper 中以自回归方式逐步输出 BPE token 的 Transformer 部分，每步通过交叉注意力从编码器获取声学信息
- **出现课程**: L66, L70, L71, L75
- **标注状态**: ❌ 缺失标注

### 自注意力 (Self-Attention)
- **英文全称**: self-attention
- **缩写**: self-attn
- **定义**: Transformer 编码器中每帧可关注全局所有帧的注意力机制，Whisper 编码器不使用因果掩码
- **出现课程**: L70
- **标注状态**: ✅ 已标注（首次标注于 L70）

### 前馈网络 (FFN)
- **英文全称**: Feed-Forward Network
- **缩写**: FFN
- **定义**: Transformer 每层中自注意力之后的两层线性变换加激活函数子网络
- **出现课程**: L70
- **标注状态**: ✅ 已标注（首次标注于 L70）

### 交叉注意力 (Cross-Attention)
- **英文全称**: cross-attention
- **缩写**: 无
- **定义**: Whisper 解码器每步生成 token 时，Q 来自解码器当前状态，K/V 来自编码器 1500 步输出，将声学信息注入解码过程
- **出现课程**: L70, L71, L75
- **标注状态**: ✅ 已标注（首次标注于 L70）

### 下采样 (Downsampling)
- **英文全称**: downsampling
- **缩写**: 无
- **定义**: Whisper 编码器用两层 Conv1D（stride=2）将 3000 帧压缩为 1500 步，减少 Transformer 计算量
- **出现课程**: L66, L70
- **标注状态**: ❌ 缺失标注

### 自回归 (Autoregressive)
- **英文全称**: autoregressive
- **缩写**: AR
- **定义**: 解码方式：每步用已生成的历史 token 预测下一个 token，直到输出结束符
- **出现课程**: L70, L71, L75
- **标注状态**: ❌ 缺失标注

### log-Mel 频谱 (Log-Mel Spectrogram)
- **英文全称**: log-Mel spectrogram
- **缩写**: 无
- **定义**: 经 Mel 滤波器组和 log 变换后的声学特征矩阵，Whisper 标准输入为 80 维、3000 帧，形状 (80, 3000)
- **出现课程**: L66, L70, L72
- **标注状态**: ⚠️ 部分标注

### 归一化 (Normalization)
- **英文全称**: normalization
- **缩写**: 无
- **定义**: Whisper 输入的 log-Mel 归一化将动态范围压缩到 [-1, 1]；WER 计算前的文本归一化处理大小写和标点一致性
- **出现课程**: L70, L72, L73
- **标注状态**: ❌ 缺失标注

### 注意力权重 (Attention Weight)
- **英文全称**: attention weight
- **缩写**: α
- **定义**: 经 softmax 归一化后的缩放点积注意力分数，决定解码每个 token 时各编码帧的贡献比例
- **出现课程**: L70, L75
- **标注状态**: ❌ 缺失标注

## 微调与评估

### LoRA (Low-Rank Adaptation)
- **英文全称**: Low-Rank Adaptation
- **缩写**: LoRA
- **定义**: 冻结原始权重 W，额外引入低秩矩阵 A@B 表示增量 ΔW，只训练后者，参数量降幅可达 98%
- **出现课程**: L72
- **标注状态**: ✅ 已标注（首次标注于 L72）

### 微调 (Fine-Tuning)
- **英文全称**: fine-tuning
- **缩写**: 无
- **定义**: 在预训练模型基础上，用特定领域或语言数据继续训练以适配下游任务的训练策略
- **出现课程**: L72, L73
- **标注状态**: ❌ 缺失标注

### 过拟合 (Overfitting)
- **英文全称**: overfitting
- **缩写**: 无
- **定义**: 模型在训练集上损失持续下降但验证集上性能不再提升甚至下降的现象
- **出现课程**: L72
- **标注状态**: ❌ 缺失标注

### 对齐 (Alignment)
- **英文全称**: alignment
- **缩写**: 无
- **定义**: 将帧级输出序列与目标标签序列对应起来的映射关系，CTC 通过路径求和隐式学习对齐，交叉注意力热力图可视化对齐质量
- **出现课程**: L68, L69, L73, L74, L75
- **标注状态**: ❌ 缺失标注

### 混淆矩阵 (Confusion Matrix)
- **英文全称**: confusion matrix
- **缩写**: 无
- **定义**: 统计模型将哪些词错误替换成哪些词的频次矩阵，揭示音素混淆、形近词混淆等系统性错误模式
- **出现课程**: L74
- **标注状态**: ❌ 缺失标注

### 集外词 (OOV)
- **英文全称**: Out-of-Vocabulary
- **缩写**: OOV
- **定义**: 训练词表中未出现的词，如专有名词、技术术语，模型遇到时无法正确识别，是替换错误的主要来源之一
- **出现课程**: L74
- **标注状态**: ❌ 缺失标注

### 语音活动检测 (VAD)
- **英文全称**: Voice Activity Detection
- **缩写**: VAD
- **定义**: 检测音频中包含语音的片段与静音片段的技术，VAD 效果差会导致句末词被截断从而产生删除错误
- **出现课程**: L74
- **标注状态**: ❌ 缺失标注

### 热词增强 (Hotword Boosting)
- **英文全称**: hotword boosting
- **缩写**: 无
- **定义**: 在解码阶段对指定关键词/专有名词概率进行提权的技术，用于改善 OOV 词汇的识别准确率
- **出现课程**: L74
- **标注状态**: ❌ 缺失标注

### 热力图 (Heatmap)
- **英文全称**: heatmap
- **缩写**: 无
- **定义**: 用颜色深浅表示交叉注意力权重矩阵的可视化图，对角线清晰表示对齐良好，权重弥散则预示幻觉风险
- **出现课程**: L75
- **标注状态**: ❌ 缺失标注

### 熵 (Entropy)
- **英文全称**: entropy
- **缩写**: H
- **定义**: 注意力分布的不确定性度量 H = -Σ(α·log(α))，熵越高说明模型对当前 token 对应帧的定位越模糊
- **出现课程**: L75
- **标注状态**: ❌ 缺失标注
