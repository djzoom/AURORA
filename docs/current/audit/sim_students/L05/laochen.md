# 老陈 — L05 学习日志

## 我理解了什么（简要，3 行以内）

复数 `a+bj` 就是平面上一个箭头：箭头长度叫模（用勾股定理算，√(a²+b²)），箭头方向叫相位。
FFT 给每个频率发一个复数当"身份证"：模 = 多响，相位 = 从哪起步。这个"两个自由度打包"的说法我服气。
我自己用 `z.real / z.imag / np.sqrt / np.arctan2` 写出了 `magnitude_phase`，断言过了，3+4j 算出模 5、相位约 0.9273。

## 遇到的困难和问题

### 问题 1 【推演】"乘以 i = 逆时针旋转 90°"是从哪儿推出来的？

位置：第 1 节"复数的基本操作"，原文："i 的定义就是'乘以 i = 逆时针旋转 90°'"，接着用 `i² = -1（转了两次 90° = 转了 180° = 乘以 -1）` 收尾。

我高中学的定义是 i² = -1，课程这里反过来把"旋转 90°"当定义、把 i²=-1 当推论。可是**为什么复数乘法对应旋转**，一个字都没讲。我自己试着拿 (a+bj)·i = -b+aj 在纸上画，画出来点确实转了 90°，但这是我瞎蒙着画的，课程没带我做这一步。更别说"乘以任意复数 = 转它的相位角"这种一般结论了。

希望补充：给一个两三行的小推演或一个代码/画图 cell，算 (3+4j)·1j 前后的坐标，让我亲眼看到 (a,b) 变 (-b,a)、并说明这就是转 90°。不然"旋转"这个核心直觉全靠信。

希望补充：要么本课只用 `np.cos(theta) + 1j*np.sin(theta)` 构造单位圆点（这个我看得懂），把 `np.exp` 留给 L06；要么在第一次出现处明说"暂时当黑盒，L06 解释为什么"，并把白板问 4 挪到 L06。

### 问题 3 【理解】arctan2 是个什么东西？和我高中学的 arctan 差在哪？

位置：第 2 节推理路线："θ = arctan2(imag, real)"；手动实现规则："np.arctan2(b, a) — 反正切，返回 (-π, π]"。

高中我只学过 arctan 一个参数的版本。这里突然变成两个参数，课程只给了"返回 (-π, π]"一句，没说：(1) 为什么需要两个参数——我猜是 arctan(b/a) 在 a 是负数时分不清象限？但这是我猜的；(2) 参数顺序为什么是 (b, a)（虚部在前、实部在后）——我这种老是记反行和列的人，写成 arctan2(a, b) 一点都不奇怪，而且对 3+4j 这种两个都是正数的测试用例，写反了得 0.6435 也照样是个"像模像样"的数，断言报错时我懵了好一会儿才发现是顺序问题。

希望补充：一小段"arctan vs arctan2"对比——举一个第二象限的例子（比如 -3+4j）演示 arctan(4/-3) 给错答案而 arctan2(4,-3) 给对；再明确提醒"顺序是 (y, x)，即 (虚部, 实部)，和坐标 (x, y) 的习惯相反，容易写反"。

### 问题 4 【实践】"参数实验"的 markdown 写法和代码不一致，照文字敲出来对不上号

位置："参数实验：单位圆上 8 点"一节，原文："构造 `z = np.exp(1j * np.linspace(0, 2*np.pi, 8))`"，并说相位"以 π/4 等步长均匀分布"。

我先照 markdown 的原文自己敲了一遍，得到的相位步长是 0.8976（我后来才明白是 2π/7），跟文字说的 π/4≈0.7854 对不上，一度以为自己敲错了。往下看代码 cell 才发现它多了个 `endpoint=False`。我用 python3 两种写法都跑了对比，确认差异就出在这个参数上：不带它时 linspace 把 0 和 2π 都算进 8 个点里，间隔就成了 2π/7。

希望补充：把 markdown 里的构造式改成和代码一致的 `np.linspace(0, 2*np.pi, 8, endpoint=False)`，最好顺带一句解释 endpoint=False 是"不含终点 2π，避免和起点 0 重复"——代码注释里其实写了这句，但读文字先动手的人会先掉坑。

### 问题 5 【推演】FFT 预览里"相位 ≈ -π/2，因为 cos 相位领先 π/2"——为什么基准是 cos？

位置："FFT 输出预览"代码 cell 末尾的 print："相位 ≈ -π/2，因为 sin 波在 t=0 时值为 0，cos 相位领先 π/2。"

模 ≈ 33 那句明说了"L40 推导，现在接受即可"，我就接受了，没意见。但相位这句不一样：它给了个半截解释。我知道 sin 落后 cos 90°（L04 学的），可是**为什么 FFT 量出来的相位是"相对 cos"的**？信号明明是我用 sin 造的，凭什么相位不是 0 而是 -π/2？这一步推不出来，又没像模那句一样标"先接受"。

希望补充：一句到位的铺垫，例如"FFT 把每个频率的成分写成 A·cos(2πft+φ) 的形式（原因见 L06/L37），所以 sin = cos 落后 π/2，读出的 φ 就是 -π/2"。哪怕先当结论记，也得告诉我基准是 cos、以及去哪课找推导。
