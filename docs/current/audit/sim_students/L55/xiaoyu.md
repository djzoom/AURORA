# 小雨 — L55 学习日志

## 我理解了什么

**前向和反向两套算术**：每个 `*` `+` `**` 操作都产生一个新节点，记住"这一步怎么算的"（前向），然后在反向时按相反的链式法则把梯度传回去。比如 `a*b` 前向得 `out = a*b`，反向时 `a.grad` 得到 `b` 的值乘上来的梯度。

**三个新算子的导数公式**（虽然推不全）：`x^n` 的导数是 `n·x^(n-1)`；ReLU 就是"正数过，负数变0"；tanh 有个巧妙的公式能直接用输出值算。

**数值试验**：二维感知机的例子 `tanh(w1·x1 + w2·x2 + b)` 能看出梯度是怎么从输出逆向流向参数的。
