# 老陈 — L55 学习日志

## 我理解了什么

- **算子 = 前向计算 + 反向闭包**：每个加法、乘法、幂运算、激活函数都是一个"积木"，前向时计算 `out.data`，同时把"怎么传梯度回来"写进 `_backward` 函数里。
- **三个新算子的导数公式**：`__pow__(n)` 导数是 `n·x^(n-1)`；`relu` 是分段的 `1 if x>0 else 0`；`tanh` 有个妙招——导数可以直接用前向算出的 `out.data` 算，不用再调 `math.tanh`。
- **感知机端到端反向**：能用这些算子搭 `output = tanh(w1*x1 + w2*x2 + b)` 这样的计算图，调 `.backward()` 后梯度自动流回到 `w1`, `w2`, `b`。
