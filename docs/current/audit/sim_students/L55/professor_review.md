### 小雨问题 5（链式法则三项缩成两项）→ 改动

**位置**：cell `f412af9d`（"参数实验：二维感知机前向传播与链式法则验证"），插在"设 loss = `output`"这句话之后、"通过'计算图'的路径"这句话之前——正是学生原文卡住的地方。

**改动内容**：新增《插播：为什么下面的推导只有两项，标准链式法则不是三项吗？》段落：
1. 先给出学生见过的"完整版"三项链式法则 `∂L/∂w = (∂L/∂output)·(∂output/∂z)·(∂z/∂w)`，说明三项对应计算图上连续三步跳跃的"敏感度连乘"。
2. 用生活类比铺垫："加薪 → 存款 → 心情"链条，如果心情就是直接盯着存款看（没有额外心理加工），那么"存款对心情的影响"这一项恰好是 1。
3. 回到数学：说明本课 `loss = output` 是恒等函数（identity function），恒等函数的导数是 1（类比 `y=x` 直线斜率为 1），因此 `∂L/∂output = 1`。
4. 明确写出代入过程：`∂L/∂w = 1 · (∂output/∂z) · (∂z/∂w) = (∂output/∂z) · (∂z/∂w)`，点明"不是漏项，是这一项算出来是 1，乘 1 不变，所以隐身"。
5. 加提醒框：说明这只是本课简化，接上真正损失函数（MSE/交叉熵）后 `∂L/∂output` 不再是 1，三项链式法则会重新缺一不可，并预告 L56 会讲到。
6. 紧接着在原有"链式法则说：`∂output/∂w1 = ...`"一句后追加括注，明确"这里 output 就是 L，所以省去了恒为 1 的 ∂L/∂output 这一项"，把前面的铺垫和后面的具体推导直接勾连起来。
