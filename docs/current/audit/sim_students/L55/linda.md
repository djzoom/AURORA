### 遇到的困难和问题

#### 【推演】tanh 导数公式怎么推出来的？

**位置**：概念 2 — "d(tanh(x))/dx = 1 - tanh(x)²"

我看到这个公式就懵了。Notebook 只说"其导数"就是这个，但没告诉我怎么从 `tanh(x) = (e^{2x} - 1) / (e^{2x} + 1)` 推出来的。高中学过商法则和链式法则吗？我忘了。需要课程一步步推导：先算分子分母的导数，然后用商法则，最后简化成 `1 - tanh²`。这样我才能真正记住这个公式，而不是死背。

#### 【理解】为什么 tanh 梯度可以只用"前向输出值"计算？

**位置**：概念 2 最后，"不需要再存中间变量"这一段

Notebook 说 `out.data` 就是已经算好的 tanh 值，反向就能直接用。但为什么？这是说，前向时我算出了 `t = tanh(x)`，存在 `out.data` 里，反向时我需要梯度 `∂L/∂x = (1 - t²) * ∂L/∂out`，所以只需要 t，不需要再知道原来的 x 是什么。这个逻辑上讲得通，但数学上为什么"导数恰好能用 tanh 的输出值表达"呢？是巧合还是有某种深层对称性？需要课程解释一下这背后的数学。

#### 【理解】什么是"次梯度"（subgradient）？为什么 ReLU 在 0 处不选 undefined？

**位置**：概念 3 — "次梯度取 0"

Notebook 说"在 x=0 处次梯度取 0"，然后用代码 `out.data > 0` 来判断。但"次梯度"这个词我第一次见。ReLU 在 0 处不可导，我理解，但为什么要选 0？能选 1 吗？能选 0.5 吗？如果这是凸分析里的概念，那高中数学里没有，课程应该给个直观解释：比如"在实践中，x 正好等于 0 的概率极小，所以选哪个都无所谓，我们就选 0 因为方便"或者"选 0 能让梯度在负数区停止传播，防止不必要的计算"之类的。

#### 【理解】感知机里 `w1.grad = x1 * tanh'(z)` 是怎么来的？

**位置**：参数实验部分，反向传播的输出

代码自动算出来了，但我想理解。`output = tanh(z)` 其中 `z = w1*x1 + w2*x2 + b`。反向时，`∂output/∂w1 = ?`。我知道要用链式法则，但 notebook 没展开：应该是 `∂output/∂w1 = (∂output/∂z) * (∂z/∂w1)`，其中 `∂output/∂z = tanh'(z) = 1 - tanh²(z)`，`∂z/∂w1 = x1`，所以 `∂output/∂w1 = (1 - tanh²(z)) * x1`。对吗？Notebook 能不能把这个链式过程写出来，而不是只看最后的代码结果？
