# 小雨 — L67 学习日志

## 我理解了什么
编辑距离就是把一个字符串变成另一个，用替换、插入、删除三种操作，每个操作算1步，求最少需要几步。用一个二维表格从左上往右下填，相同字符就从左上直接抄值，不同就看左上、上、左三个格子加1谁最小——这个思想用来处理"拆解成更小的子问题"来解决。WER就是这套东西的词级版本。

### 【计算】DP表的索引关系有点绕
- 递推用`a[i-1]`和`b[j-1]`对比字符，但`dp[i][j]`代表前`i`个和前`j`个字符。
- 这样i和j是"字符个数"，而数组索引是"第i-1个字符位置"，需要心算+1/-1。
- 能否在初始代码注释里明确标一下：`dp[i][j]对应把a[:i]变成b[:j]`，这样读起来不容易混？
