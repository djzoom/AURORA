# 老陈 — L20 学习日志

## 我理解了什么

通过这节课，我看懂了把一个复杂的矩阵变换分解成几步简单变换的idea：LU把消元分解成行操作的乘积，QR把斜的基正交化成垂直的，SVD把任意矩阵写成"旋转-拉伸-旋转"三步，对称矩阵有个特殊形式QΛQᵀ用特征值和特征向量。我还理解了秩1矩阵（比如列向量乘行向量得到的外积）在SVD里只有一个非零奇异值。

## 遇到的困难和问题

### 【理解】QᵀQ=I 到底为什么叫"正交"？

课程展示了`np.round(Q.T @ Q, 10)` 得到单位矩阵，但我卡在："正交"这个词在高中是直线垂直的意思，矩阵形式`QᵀQ=I`和"垂直"有啥关系？为什么两个矩阵相乘得到单位矩阵就叫正交了？能否补充一句话说明Q的**列向量两两垂直且长度为1**？

### 【理解】SVD中U、Σ、Vᵀ 各自干什么——几何上看不懂

课程说"Vᵀ把输入旋转，Σ沿方向拉伸，U再旋转到输出空间"，但我测试了：A是2×3矩阵（2行3列），U是2×2，Σ是2×3，Vᵀ是3×3。

- 为什么是 (2×2) × (2×3) × (3×3) ？
- Vᵀ（3×3）作用在哪个空间？是作用在A的**列**空间吗？
- 如果A代表坐标变换，输入向量和输出向量各多少维？

能否用一个具体例子（比如把二维平面的向量映射到三维空间），说清楚这三个因子的几何作用顺序？

### 【推演】为什么L的对角线总是1？

代码里 `L = np.eye(n)` 初始化，说"对角线是1"。课程解释说L是"行消元乘数"，但没解释**为什么消元矩阵的对角线必须是1**。这和高斯消元的哪一步对应？是说第i行消元不消自己吗？

### 【计算】条件数κ变大，为什么求解不稳定？

参数实验里，`κ=1000`时QR的R对角线元素是`[1000. 1.]`。课程说"条件数越大……求解越不稳定"，但没说清楚稳定性和R对角线差距大有啥直接关系。是说分母和分子相差太大会有舍入误差吗？

### 【实践】rank_from_svd 里的 `threshold` 是多少？

练习要求"设门限 `t = threshold * S[0]`"，代码注释给了`threshold=1e-10`。但为什么是`1e-10`？这个值怎么选？如果S[0]很小（比如`1e-5`），那t就成了`1e-15`，是不是太严了？

### 【理解】为什么秩1矩阵有2个零奇异值？

代码验证了外积矩阵秩为1时，3个奇异值中后两个是0。我能背下来"零奇异值个数=秩亏"，但没理解**为什么**。秩1意味着所有行都是同一行的倍数，这和奇异值有啥几何关系？

### 【理解】对称矩阵的 S=QΛQᵀ 和一般矩阵的 A=UΣVᵀ 什么关系？

课程并排展示这两个，对于对称矩阵S，Q的两个角色重合了（既是左奇异向量又是右奇异向量），这是巧合还是有深层原因？为什么对称矩阵特别而一般矩阵不行？
