# 小雨 — L20 学习日志

## 我理解了什么

矩阵分解就是把一个复杂的变换拆成几个简单变换的乘积。LU是消元、QR是正交化、SVD是旋转-拉伸-旋转三步。用SVD可以找出矩阵真正有效的秩：看奇异值有多少个比较大，其他都接近0。
