# 琳达 — L09 学习日志

## 我理解了什么

1. **向量 = NumPy数组**：shape为(n,)的一维数组，代表"一列有序的数字"，音频信号、特征都是这样的形式。

2. **两个基本操作很直观**：加法就是逐个位置加（混音），缩放就是每个数都乘同一个系数（调音量）。代码和实现都很简单：`a+b` 和 `c*v`。

3. **线性组合的技能掌握了**：虽然概念有点抽象，但代码逻辑清楚——就是把一组数字和向量配对，逐一缩放后加起来，`coeffs[i]*vecs[i]` 求和。

## 遇到的困难和问题

### 【理解】向量既是"点"又是"箭头"，什么时候用哪个含义？

课程说"音频信号 `[0.1, -0.3, 0.5, ...]` = N维空间里的一个点（或一条路径）"，这让我困惑。一串数字怎么既是坐标空间里的点，又是从原点出发的箭头？这两个视角什么时候切换？在音频处理里，我该把 MFCC 向量看作点还是箭头？

### 【推演】为什么 √(3²+4²) 就是向量长度？

范数公式 `‖v‖ = √(v₁² + v₂²)` 给了，是勾股定理的推广，但课程没解释：为什么定义"长度"要用这个公式？还是说长度本来就是这样定义的（约定），没有更深的推导？如果向量有负分量（如 `[3, -4]`），为什么还是用平方和再开根号，而不是直接加绝对值？

### 【理解】"任意2D向量都可由基向量 e₁, e₂ 的线性组合表示"，为什么一定可以？

代码展示了例子（`[2, -1] = 2·e₁ + (-1)·e₂`），但没有证明：凭什么任意向量都行？有没有某个向量找不到对应的系数？这个陈述在数学上是充要条件还是只是巧合？

### 【理解】矩阵 A 对向量的变换是什么意思？

在"参数实验"部分看到矩阵 A 对向量的作用（如 `A @ v`），课程说"矩阵对向量的作用将在 L12 中详细展开"。但已经在这里用了，为什么不现在说清楚？矩阵和向量的点积结果怎么理解，和向量加法、缩放的关系是什么？

### 【实践】线性组合的几何意义还是不太想象得出来

代数上我会算 `2·e₁ + (-1)·e₂ = [2, -1]`，但这在几何上或实际应用里意味着什么？为什么我要把向量写成基向量的组合？在音频或特征向量的实际场景里，这个操作有什么用？
