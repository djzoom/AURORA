# 老陈 — L09 学习日志

## 我理解了什么

1. **向量的三个基本操作**：向量加法就是逐元素相加（对应混音），标量缩放就是所有元素乘同一个数（对应调音量），线性组合把这两个操作叠加起来。

2. **代码实现很直白**：`scale(v, c)` 就是 `c * v`，`add_signals(a, b)` 就是 `a + b`，因为 NumPy 的广播机制自动处理。三个函数都通过了白板挑战。

3. **向量 = 数组**：NumPy 里 shape 是 `(n,)` 的数组就是向量，每个分量代表一个维度的坐标。
