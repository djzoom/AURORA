# 小雨 — L04 学习日志

## 我理解了什么（简要）

收音机三个旋钮的比喻我很喜欢：A 管多响、f 管多高、φ 管从哪里开始，公式 `x(t) = A·sin(2πft + φ)` 我能背下来也能代入数字算。`sinusoid` 练习照着"推理路线"三步写出来了（`A * np.sin(2*np.pi*f*t + phi)`），断言通过。"和弦 = 几个正弦波加在一起"这个画面也建立起来了。

## 遇到的困难和问题

### 问题 1【理解】复数和 e 突然出现，可是复数是下一课才教的

位置：「动手观察：四个特殊角度」那格代码 `z = np.exp(1j * angles)`，以及后面「小检查：同一个角度，同时看 cos、sin、复数」。

课程开头明明说本课是"为 L05（复数）和 L06（欧拉公式）打好基础"，结果这两格代码直接用了 `1j`、`np.exp(1j*角度)`、`z.real`、`z.imag`。`1j` 是什么？为什么 e 的多少次方会出来 cos 和 sin？我只在第 1 节的一句话里见过 `e^{iθ} = cosθ + i·sinθ`，但那句话也没解释。我只能把输出当成"魔法打印出了 0 和 ±1"，验证了个寂寞。

希望补充：要么把这两格挪到 L05/L06 之后，要么在格子前加一段"预告"：明确说"这行代码用了下一课的知识，现在你只需要知道 `z.real` 打印的就是 cos、`z.imag` 就是 sin，原因 L06 揭晓"。现在的写法让我以为是我漏学了什么。

### 问题 2【推演】"1.85 倍、1.41 倍"这两个数字我算不出来，也不知道从哪来

位置：「代码实验：遍历频率、振幅和相位」的说明文字："freq 加倍时相邻样本的最大变化幅度明显增大（f=1→2 约 1.85 倍，f=2→4 约 1.41 倍——正弦是弯的，并非精确翻倍）"。

代码只是把 8 个采样值打印出来，并没有任何一行去算"相邻样本的变化幅度"。我对着打印出来的两排小数，手动相减了半天才凑出 0.38 和 0.71 这两个最大差值，比值确实约 1.85——但这不该让读者自己发明步骤吧。而且"正弦是弯的"这五个字并没有解释为什么不是精确 2 倍。

希望补充：在代码里加一行 `print(np.max(np.abs(np.diff(wave))))` 之类（并解释 diff 是"相邻两数相减"），让说明里的数字能在输出里直接看到；"为什么不是 2 倍"哪怕给个一句话的图形直觉（比如"波太快时相邻两点已经跨过了一个峰"）也好。

### 问题 3【推演】过零次数为什么是 2f−1 而不是 2f？还有三个没教过的函数

位置：「参数实验：只改一个旋钮」："规律约为 2f−1（f=1,2,4,8 对应 1,3,7,15）"，以及它上面那格代码 `np.sum(np.diff(np.signbit(x)) != 0)`。

我的直觉是：1 秒里有 f 个完整周期，每个周期穿过零线 2 次，应该是 2f 次才对，为什么全都少 1 次？课程只报了数字没说原因。另外这格代码一口气用了 `np.signbit`、`np.diff`、布尔值求和三个我完全没见过的东西，我只会 print/循环/列表，这一行对我就是天书。还有个小别扭：这格代码出现在解释它的文字**前面**，而且文字让我"把 phi 改到 np.pi/2"，可这格代码里根本没有 phi 这个变量，我不知道该改哪里。

希望补充：(a) 用一张标了过零点的波形图解释那个 −1（是不是因为 t=0 那个点正好等于 0、没被算作"穿过"？）；(b) 给 signbit/diff 各配一句人话注释；(c) 把说明文字挪到代码前面，并把示例代码改成真的带 phi 参数。

### 问题 4【理解】单位圆的"影子"比喻很棒，但没有图！

位置：第 1 节「sin 与 cos：同一个圆的两个侧面」——"把一个点放在单位圆上……cos 是它在 x 轴上的影子"。

我是看图才能懂的人。这一节用文字描述了一个转动的点和两个影子，紧接着的代码却只画了 sin/cos 两条曲线，那个"圆"从头到尾没出现。而且"cos 超前 sin 恰好 π/2"这句话让我纠结了很久：公式里写 `cos(θ) = sin(θ + π/2)` 是加号，可从图上看，cos 的峰在 sin 的峰**左边**——"超前"到底是曲线往左移还是 θ 加了一个数？这两种说法在我脑子里打架。

希望补充：画一张单位圆 + 转动点 + 两条投影虚线的静态图（或分成几帧），并在 sin/cos 曲线图上用箭头标出"同一个峰，cos 早了 π/2 出现"，把"超前 = 曲线左移 = 自变量加 π/2"三件事对齐讲一次。

### 问题 5【计算】和弦那格：261.6 Hz 的波，1 秒只采 200 个点，画出来的能是真的吗？

位置：「参数实验：和弦 = 多个正弦波相加」代码 `sr, dur = 200, 1.0` 配 `f=261.6 / 329.6 / 392.0`。

白板挑战问 3 刚教会我"每周期采样点数 = sr / f"。我照着一算：200 / 261.6 ≈ 0.76 个点——一个周期连一个采样点都摊不到！可图上画出来的 C4 却是一条悠悠的慢波，1 秒里根本数不出 261 个周期。图的标签写着 261.6 Hz，图的样子却完全不是。到底是我的"点/周期"公式用错了，还是这张图画的根本不是 261.6 Hz？课程对此一个字都没提。

希望补充：把 sr 改成足够大（比如 4000），或者明确加一句说明"sr 太低时波形会失真变样（以后 Nyquist 那课会讲），这里图只是示意"。现在这样，我刚学会的检验方法一用就把课程自己的图检验"翻车"了，很打击信心。
