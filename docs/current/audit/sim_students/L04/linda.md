# 琳达 — L04 学习日志

## 我理解了什么（简要）

正弦波就三个旋钮：A 管多响、f 管多高、φ 管从哪开始，公式 `A·sin(2πft+φ)` 我能背下来也能写成代码；`sinusoid` 练习我写一行 `return A*np.sin(2*np.pi*f*t+phi)` 就过了断言，白板挑战四问用高中三角知识都算出来了（sin(5π/4)=−√2/2 那步靠提示撑住）。"和弦=几个正弦波相加、DFT 反过来拆开"这个大方向我 get 到了。

## 遇到的困难和问题

### 1.【理解】复数 `np.exp(1j * angles)` 突然出现，没有任何铺垫

位置："动手观察：四个特殊角度"一格（`z = np.exp(1j * angles)`），以及后面"小检查：同一个角度，同时看 cos、sin、复数"一格。

课程开头明明说复数是 L05、欧拉公式是 L06 的内容，"本课为它们打好基础"。结果这两格代码直接用 `np.exp(1j*θ)`，然后让我"比对 z.real 和 z.imag"。我知道 Python 里 `1j` 是虚数单位（写爬虫时见过报错），但**为什么对一个虚数取 exp，实部就变成 cos、虚部就变成 sin？** 这一步一个字都没解释，我只能"信"输出。第 1 节末尾提了一句 `e^{iθ} = cos θ + i·sin θ`，但那是"后面 L05 会讲"的口吻。希望课程二选一：要么把这两格挪到 L05/L06 之后，要么在格前加一段"现在先把 `np.exp(1j*θ)` 当成一台自动同时算出 (cosθ, sinθ) 的机器，原理下两课讲"——现在这样夹在中间，我不确定是不是我漏学了什么。

### 2.【实践】和弦 demo 的 sr=200 跟本课刚教的"每周期采样点"公式直接打架

位置："参数实验：和弦 = 多个正弦波相加"代码格（`sr, dur = 200, 1.0`，`c4 = sinusoid(t, A=1.0, f=261.6)`）。

白板挑战问 3 刚教我"每周期采样点 = sr/f"（8000/440 ≈ 18.18 个点）。我照这个公式一算：200/261.6 ≈ 0.76、200/392 ≈ 0.51——**每个周期连 1 个采样点都不到**，一秒 200 个点怎么可能画出每秒振 392 次的波？我实际跑了：392 Hz 那条"波"1 秒内只过零 16 次，看起来像一条 8 Hz 的慢波，画出来的四条曲线跟 C4/E4/G4 根本对不上。结尾那句"DFT 的任务就是从和弦这条曲线分解回 C4/E4/G4"在这个采样率下应该也做不到吧？希望课程把 sr 改大（比如 8000），或者明确告诉我"这里点太稀、画出来的是假象，为什么会这样 L__ 讲"——现在这样我用本课学的公式自查，反而查出教材自己不一致，很动摇。

### 3.【推演】过零次数"规律约为 2f−1"只给结论，推不出来

位置："参数实验：只改一个旋钮"（"规律约为 2f−1（f=1,2,4,8 对应 1,3,7,15）"）。

我自己想：1 Hz 的正弦一个周期里上穿一次、下穿一次，过零 2 次，那 f Hz 一秒该过零 2f 次才对，为什么代码数出来总是少 1 次（2f−1）？是 t=0 那个零点不算？还是 `np.diff(np.signbit(x))` 这种数法漏了哪个？课程只报了结论没说原因。另外那句"频率翻倍时过零次数大约翻倍，并不是简单的 4 倍正比"我读了三遍没懂——f 从 1 到 4 是 4 倍、过零从 1 到 7 是 7 倍，"4 倍正比"指什么？希望补一两句：先说理论值是 2f，再解释代码数出来为什么恰好少 1（起点/端点怎么算），并把那句"4 倍正比"改成人话。

### 4.【推演】烟雾测试格的"1.85 倍、1.41 倍"数字从天而降

位置："代码实验：遍历频率、振幅和相位"上方文字（"f=1→2 约 1.85 倍，f=2→4 约 1.41 倍——正弦是弯的，并非精确翻倍"）。

我跑了代码，这两个数确实对得上，但课程只用"正弦是弯的"六个字带过。我的直觉本来是"频率翻倍 → 相邻点变化翻倍"，现在被告知不是，却不知道为什么不是，也不知道 1.41（看着像 √2？）是巧合还是有名堂。这段是让我"逐个检查数字"的验证格，可我验证完只确认了"数字没印错"，没获得任何解释。希望加两三句直白的说明（比如点稀到接近波峰时增量会饱和之类），或者干脆删掉具体倍数、只说"会变陡但不是严格翻倍"，免得我盯着 1.85 猜出处。
