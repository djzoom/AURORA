# 老陈 — L22 学习日志

## 我理解了什么
1. **导数 = 割线斜率的极限**：用两个离 x 很近的点（间距 h）算斜率，h 越小越接近真实的"切线斜率"。计算机用中心差分公式 `(f(x+h)-f(x-h))/(2h)` 近似，h≈1e-5 时误差小到约 1e-11。
2. **为什么中心差分更好**：左右对称地扰动，比只向前扰动多消去一阶误差项，所以误差从 O(h) 降到 O(h²)——能少 5 个数量级。
3. **h 的最优区间**：从实验看，h=1e-5～1e-7 误差最小。h 太大截断误差大，h 太小两个接近的浮点数相减会"灾难性消去"，精度剧降。

## 遇到的困难和问题

### 问题1：为什么 h 太小会导致误差反而变大？【理解/原理】
> Notebook 说"h < √ε 时分子相减有效位数丧失"，但具体机制没讲透。

从我的实验看，h=1e-14 时误差跳到 0.003，但我不理解"有效位数丧失"的细节：
- 浮点数精度大约是 1e-16。f(1+1e-14) 和 f(1-1e-14) 作为浮点数存储时，它们和 f(1) 的差在末位才能看出。相减时，高位的 1 抵消，只剩末位的差异，导致"有效数字"严重损失。
- **希望补充**：能否用一个具体的例子（比如二进制浮点数）说明 f(x+h)-f(x-h) 在 h≈1e-15 时如何丧失有效数字？

### 问题2：中心差分为什么是 O(h²) 而前向是 O(h)？【推演】
> Notebook 提了一句"泰勒展开消去一阶项"，但没有铺垫泰勒展开。

我高中时没学过泰勒级数，现在看这句话一头雾水。具体想知道：
- f(x+h) 和 f(x-h) 的泰勒展开式怎么写？  
- 两式相减时为什么一阶项（h 倍的导数项）会抵消？  
- **希望补充**：用最简单的例子（比如 f(x)=x²）直接展示：
  ```
  f(x+h) = (x+h)² = x² + 2xh + h²
  f(x-h) = (x-h)² = x² - 2xh + h²
  相减 → (f(x+h)-f(x-h))/2h = 2xh/2h = 2x  （完全精确！）
  ```
  然后对一般的 f(x) 说明为什么三阶项不会相消，误差就是 O(h²)。

### 问题3：为什么 h≈1e-5 是"最优"的平衡点？【理解/参数】
> 知道 h 太大截断误差大，h 太小舍入误差大，但"最优值"的来源不清楚。

实验结果表明最优 h≈1e-5～1e-7，但为什么不是 1e-4 或 1e-8？
- Notebook 提到"h≈1e-5"但没说这个数怎么算出来的。  
- 听说与 `√(机器精度)` 相关，但具体怎么推导？  
- **希望补充**：说明"截断误差 ∝ h²"和"舍入误差 ∝ 1/h"如何平衡，得出最优 h 的数学式。

### 问题4：极限定义与有限 h 的关系不够直观【理解/概念】
> "f'(x) = lim_{h→0}" 和 "实际用 h=1e-5" 之间的缝隙有点大。

数学上 h 要趋向 0，但计算机实际用 h=1e-5 停下来。这怎么正当化？
- 是说 h=1e-5 足够小使得误差接受？  
- 还是说极限的定义只是"理想情况"，计算机就是用有限 h 凑合？  
- **希望补充**：给出一个量化的标准："当误差 < 某某时，认为 h 足够小"之类的。

### 问题5：前向差分为什么 O(h) 而不是 O(h²)？【推演】
> 想理解两种差分方法的本质区别。

我能看出实验结果中前向差分误差比中心差分大，但不懂理由。我猜：
- 前向：f'(x) ≈ (f(x+h)-f(x))/h，只用了一侧信息？  
- 中心：左右对称，"配对"消去了某个错误项？  
- **希望补充**：用泰勒展开对比两种方法，说明为什么对称能多消一个 h 的因子。
