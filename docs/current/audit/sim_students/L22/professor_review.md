## 3. 最优 h≈1e-5 是怎么推导出来的（√ε 从何而来）

- **提出者**：老陈（问题3）、琳达（问题3）、小雨（问题4）
- **卡点**：三人都问"截断误差∝h²、舍入误差∝1/h"这句话如何具体平衡出 1e-5 这个数字；小雨还追问 √ε 是否是来源。
- **改动位置**：紧接上一条插入的 markdown cell（`3abc675f`，`### 最优 h≈1e-5 到底怎么算出来的？`）：把总误差写成 `E(h) ≈ f'''(x)h²/6 + ε|f(x)|/h`，对 h 求导令其为 0，推出 `h_opt ≈ (3ε)^(1/3) ≈ 8.7e-6 ≈ 1e-5`，与实验结果吻合；并专门澄清"√ε≈1e-8"只是前向差分场景下更粗糙的经验门槛，中心差分因为收敛更快，真正最优点是 ε 的**三次方根**量级而不是平方根——解决了小雨发现的数字对不上的疑惑。随后 code cell（`7e49e615`）用 `np.finfo(float).eps` 实际算出 `h_opt` 数值验证。
