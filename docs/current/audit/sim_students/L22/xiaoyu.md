# 小雨 — L22 学习日志

## 我理解了什么

导数就是曲线在某点的"坡度"——用两个离得很近的点，连一条割线，割线的斜率就近似了切线的斜率。中心差分公式 `(f(x+h) - f(x-h))/(2h)` 就是这个想法的代码实现。关键学到的是 h 不能随意选择——太大了计算不准，太小了计算机的浮点数精度不够用，最优的 h 大概是 1e-5 左右。

## 遇到的困难和问题

### 【理解】极限符号和 lim 的含义
**卡在**："极限定义来自 lim_{h→0}" 这句话。  
**问题**：lim 这个符号看起来像某种数学操作，但不太明白怎么用。课程说"h 越接近 0，割线就越接近切线"，这个意思明白了，但直接跳到公式 `lim_{h→0}` 有点快。能不能用更多日常语言解释一下 lim 是什么意思，比如"逐渐逼近"那样的词？

### 【推演】中心差分为何误差是 O(h²)，不是 O(h)
**卡在**："泰勒展开消去一阶项"这一句。  
**问题**：课程说中心差分比前向差分高一阶精度（O(h²) vs O(h)），但没有展开推导。我用 Python 验证了确实是这样（h 从 1e-1 减到 1e-5，误差减小了 100 倍，符合平方关系），但不知道为什么会这样。能补充一下"泰勒展开"是什么概念吗？或者用一个简单例子手算展开一下 f(x+h) 和 f(x-h)，看看为什么相减后一阶项消失？

### 【理解】"灾难性消去"(catastrophic cancellation) 的具体含义
**卡在**："当 h < √ε ≈ 1e-8 时，f(x+h)-f(x-h) 两个近似相等的数相减，有效位数丧失"。  
**问题**：这句话比较抽象。能给一个具体数字例子吗？比如"如果 f(x+h) = 1.23456789012345 和 f(x-h) = 1.23456788900000，它们相减得到什么"之类的。另外，为什么最优 h 是√ε 这个平方根？这个 ε（机器精度约 1e-16）怎么算出来的？

### 【理解】h 最优值 1e-5 从何而来
**卡在**："最优 h ≈ 1e-5：不太大不太小，平衡截断误差和舍入误差"。  
**问题**：虽然我用 Python 跑了一遍确实看到 h=1e-5 误差最小，但课程没解释如何预先计算出这个值。是不是有什么公式可以推导？比如"误差函数 = 截断误差 + 舍入误差"，然后求导找最小值之类的？

### 【解答】常见导数表的推导
**卡在**：表格直接列了"xⁿ → n·xⁿ⁻¹"、"sin x → cos x"等导数。  
**问题**：这些是背下来的公式吗？还是之前在某个 cell 或前面的课推导过？xⁿ 的导数用中心差分能验证，但 sin x 为什么导数是 cos x，这背后有什么直观理由吗？
