# 琳达 — L22 学习日志

## 我理解了什么
导数就是函数曲线在某点的斜率。计算机不能取h→0的极限，所以用两个很接近的点（x±h）做割线来近似切线，中心差分公式 `(f(x+h) - f(x-h)) / (2h)` 就能算出这个斜率，h取1e-5左右最准。实现出来的 `numeric_derivative` 函数用处很大——以后验证反向传播有没有算错就靠它。

## 遇到的困难和问题

### 【理解】为什么中心差分误差比前向差分小一阶（O(h²) vs O(h)）？
notebook说"泰勒展开消去一阶项"，但没有给完整推导。我是高中数学水平，看不出来展开式怎么消项的，这块是个黑盒。能不能补上具体的展开过程？或者至少说明白：前向差分和中心差分分别展开到哪一项，为什么中心差分那一项会消失？

### 【理解】"灾难性消去"(catastrophic cancellation) 具体怎么发生的？
我理解了结论——h太小时，`f(x+h)` 和 `f(x-h)` 都接近 `f(x)`，两个相近的大数相减会丧失有效位数。但"有效位数丧失"到底是怎么回事？浮点数为什么在减法时特别容易出问题？能举个具体例子（比如二进制或十进制形式）说明两个0.999999和1.000001相减会怎样吗？

### 【推演】最优的 h≈1e-5 是怎么推出来的？
notebook说"平衡截断误差和舍入误差"，我看出了实验结果（h=1e-5误差最小），但没看到推导。是不是说截断误差≈h²某个系数，舍入误差≈1/h某个系数，两个相等时总误差最小？那 1e-5 这个具体数字从哪来？是√(machine epsilon) 吗？为什么？

### 【理解】极限定义为什么必须这样写？
"f'(x) = lim_{h→0} (f(x+h)-f(x))/h"——为什么是这个式子定义导数，而不是其他形式？我理解"h→0时割线接近切线"这个几何直觉，但从"直觉"跳到"这个公式"的逻辑链条还是缺一块。

### 【实践】怎么验证 h 的最优值跟函数有关吗？
我看到对 sin(1.0) 的扫描，h=1e-5 最优。但如果换个函数（比如 x³ 或 e^x），最优 h 会变吗？notebook 没试过别的函数。
