# 老陈 — L11 学习日志

## 我理解了什么

**范数就是"向量长度"的三种量法。** L2 是我熟悉的勾股定理（直线距离），L1 是沿街道走的距离（横纵分别加），L∞ 是国际象棋中王的走法（最长一步）。**核心操作是 normalize**：把向量除以它的 L2 长度，得到长度为 1 的单位向量，这样方向不变、长度变成 1。

## 遇到的困难和问题

### 【理解】向量距离的几何含义
**卡在哪里？** 课程说"两向量的距离 = 差向量的长度"，给的公式是 `dist(a, b) = ‖a - b‖`，但没有图解为什么是这样。

**具体困惑：** 为什么 a 和 b 的距离不是直接用坐标算，而要先算 a - b 这个差向量，再取它的长度？这个逻辑关系是什么？

**希望补充：** 一张图或者一个 2D 例子：点 (1, 2) 到点 (4, 6)，画出向量 a、b、a-b，标注长度 5，说明为什么这就是距离。

### 【理解】L∞ 范数的名字"切比雪夫"
**卡在哪里？** 课程表格第二列写的是"L∞（切比雪夫）"，然后说"最大分量"，但没解释这个人名是怎么来的。

**具体困惑：** 为什么叫"切比雪夫距离"？是某个数学家的名字吗？这个名字重要吗，还是只是个历史术语？

**希望补充：** 一句话背景介绍，比如"L∞ 由切比雪夫定理命名"或者"这是...方面最常用的范数"。

### 【推演】L1 和 L2 梯度的推导
**卡在哪里？** 代码块"正则化含义：L1 vs L2"说"L1 梯度为 λ·sign(w)、L2 梯度为 2λw"，但怎么算出来的？

**具体困惑：** 
- L1 正则项是 `λ‖w‖₁ = λ·Σ|wᵢ|`，要对 w 求偏导，但绝对值在 w=0 时不可导，怎么处理？
- L2 正则项是 `λ‖w‖₂²`（平方），对 w 求偏导怎么求？为什么出现 2λ 这个系数？

**希望补充：** 展开 L1 和 L2 的偏导计算过程（哪怕只是针对一个分量 wᵢ）。

### 【理解】梯度下降中"为什么 L1 导致稀疏"
**卡在哪里？** 代码说"L1 梯度恒为 ±λ，无论权重多小都受同等力推向零 → 稀疏"，但没有说明梯度下降的过程。

**具体困惑：** 梯度下降公式是 `w_new = w - lr * grad`，如果 `grad = λ` 恒定，那 w 会一直减小，什么时候停止？什么时候会精确等于 0？

**希望补充：** 一个简单的例子或数值模拟：初始权重 w=0.3，lr=0.1，λ=0.1，跑 10 步梯度下降，显示 w 是怎么逐步逼近 0 的。

### 【计算】通用范数公式中的 p 值
**卡在哪里？** 代码块 cell-l11-0078 有公式 `|x1|^p + |x2|^p = 1`，注释说这是"Lp 单位球"。

**具体困惑：** 
- p 值是什么东西？在 `np.linalg.norm(v, p)` 中，p=1 时得 L1 范数、p=2 时得 L2 范数，这是怎么关联的？
- 为什么单位球是 `‖v‖_p = 1` 的所有向量的集合？这对学 normalize 函数有什么帮助？

**希望补充：** 解释通用范数公式 `‖v‖_p = (Σ|vᵢ|^p)^(1/p)`，然后说明为什么 p=1, 2, ∞ 时分别得到 L1、L2、L∞。

### 【实践】零向量的处理约定
**卡在哪里？** 代码说"零向量应返回全零向量，用 `if length == 0: return v` 提前拦截"。

**具体困惑：** 为什么一定要这样处理？是不是直接抛异常也行？返回全零向量这个约定在后续应用中有什么用处？

**希望补充：** 解释一下：normalize 的后续应用中（比如计算余弦相似度），为什么对零向量返回全零是"正确"的约定？

### 【解答】"MSE 损失 = 误差向量的 L2 范数平方"
**卡在哪里？** 课程说"MSE 损失就是误差向量的 L2 范数平方"。

**具体困惑：** 
- MSE 公式是 `mean((y_pred - y_true)²)`
- L2 范数平方是 `‖v‖₂² = Σ vᵢ²`
- 这两个为什么等价？mean 和 Σ 哪里对应上的？

**希望补充：** 并排写出 MSE 公式和"L2 范数平方"公式，标明对应关系。

### 【解答】"归一化后余弦相似度 = dot(a, b)"
**卡在哪里？** 课程说"MFCC 归一化后，余弦相似度等价于 `np.dot(a, b)`"。

**具体困惑：** 余弦相似度是 `cos(θ) = (a·b) / (‖a‖·‖b‖)`，如果 a、b 都归一化到 ‖a‖=‖b‖=1，那分母就是 1，所以 cos(θ) = a·b？为什么要特别提这个？

**希望补充：** 一个数值例子：原始向量 a=[3,4]、b=[6,8]，计算原始余弦相似度 vs 归一化后的 dot 积，对比看看是否相等。
