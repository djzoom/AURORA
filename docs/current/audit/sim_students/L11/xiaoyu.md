# 小雨 — L11 学习日志

## 我理解了什么
三种范数是量向量"长度"的三把尺子：L2 用勾股定理（直线距离），L1 是分量绝对值和（街区距离），L∞ 只看最大分量（国王走法）。归一化就是把向量缩放到长度为 1，通过 `v / ‖v‖₂` 实现。两个向量距离 = 差向量的长度。

## 遇到的困难和问题

### 【理解】符号 ‖v‖₂ 和 |v| 的区别没讲清
在"符号入口"这块，课程突然引入了 `‖v‖₂` 这个记号，说"L2 范数写作 `‖v‖₂`"。但问题是，我从高中数学里知道 `|v|` 也表示长度（绝对值），现在又来一个"两竖"符号 `‖v‖`，为什么要新建一个符号？两竖和一竖的区别是什么？课程没有解释为什么对向量的"长度"要用 `‖ ‖` 而不继续用 `| |`。

**希望补充**：在引入 `‖v‖` 之前，加一句：绝对值符号 `| |` 用于数字，范数符号 `‖ ‖` 用于向量和矩阵，因为它们的"长度"定义更复杂。

---

### 【推演】为什么 v/‖v‖₂ 的长度一定等于 1（没看到证明）
课程说"用 `v / length` 把每个分量同比缩小……长度从 `length` 变成 `length/length = 1`"。这是对的，我也验证过 [3,4] 除以 5 得 [0.6, 0.8]，新长度确实是 1.0。但这是一个论证，不是严格证明。

具体卡点：为什么 `‖v/c‖₂ = ‖v‖₂ / c` 这个性质成立？这涉及一个叫做"范数的齐次性"的数学性质，但课程完全没提。我看到的只是一个例子验证，没有一般性的推导。

**希望补充**：在"推理路线"之前，加一个小推导框：
```
如果 v = [v₁, v₂, ..., vₙ]，c > 0，那么：
‖v/c‖₂ = √((v₁/c)² + (v₂/c)² + ... + (vₙ/c)²)
       = √(1/c² · (v₁² + v₂² + ... + vₙ²))
       = √(1/c²) · √(v₁² + v₂² + ... + vₙ²)
       = (1/c) · ‖v‖₂
所以当 c = ‖v‖₂ 时，‖v/‖v‖₂‖₂ = 1
```

---

### 【理解】零向量为什么要返回全零，而不是报错
课程说："零向量应返回全零向量。用 `if length == 0: return v` 提前拦截——非零向量走精确的 `v / length`（范数恰好为 1），零向量原样返回全零"。

我不理解这个选择的逻辑。如果输入是 `[0, 0]`，为什么要"原样返回"，而不是直接抛异常说"零向量无法归一化"（因为无法定义方向）？这是一个约定俗成的处理，还是有数学上的理由？

**希望补充**：一句解释——"零向量没有明确的方向，无法定义单位向量。我们的约定是：当输入是零向量时，输出也是零向量（保持'长度为零'），这样避免了 NaN，也符合直觉。"

---

### 【理解】第 4 节"梯度"的跳跃太陡
第 4 节突然讲"L1 正则化在损失函数中加 `λ‖w‖₁`，对参数 `w` 的梯度为 `λ·sign(w)`"。但小雨只学过高中数学，"梯度"（gradient）是什么都没有概念。为什么L1的梯度是常数，L2的梯度正比于 w？这需要更多上下文。

代码块（`e5f6a7b8`）展示了具体数值，这很有帮助，但对初学者来说，"梯度是什么"、"为什么 L1 梯度是 sign(w)"这些基础没有交代。这感觉像是在给已经学过优化论的人补充细节，不像是在给 L11 学生做铺垫。

**希望补充**：在讲梯度之前，加一句通俗的解释——"梯度是参数对损失函数的'敏感度'——告诉我们调整参数会如何影响损失。L1 和 L2 的梯度不同，所以它们对小权重的'推力'大小不同。"然后再展示代码。

---

### 【理解】L∞ 范数的几何比喻不如 L1、L2 清晰
课程说"国王走法"（棋盘上国王走法只需要看最远的那个分量）。这个比喻对了解国际象棋的人有帮助，但对小雨（文科生，可能没怎么玩过棋盘游戏）来说，还是比较陌生。

相比之下，L1 的"出租车沿街道走"和 L2 的"鸟飞直线"都很容易想象。但 L∞ 这个"只看最大分量"的几何意义就没那么直观——为什么对这个向量而言，"最远的那步"就能代表全部长度？

**希望补充**：用一个更日常的例子——比如"衣服的防伪检测"：衣服尺寸标记为 [胸围 100cm, 袖长 60cm, 腰围 80cm]，如果检测仪器要找出"最大偏差"，就会看这三个里面距离标准最远的，这就是 L∞ 范数。
