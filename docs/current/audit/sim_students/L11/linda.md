# 琳达 — L11 学习日志

## 我理解了什么

三种范数是量"长度"的不同方法：L2 就是勾股定理的推广（直线距离），L1 是把各分量的绝对值加起来（街区距离），L∞ 只看最大的分量。关键任务是实现 `normalize(v)` = `v / np.linalg.norm(v)`，把向量缩放成长度为 1，这样就能保证两个向量的点积直接等于它们的余弦相似度（不用再除以幅度了）。
