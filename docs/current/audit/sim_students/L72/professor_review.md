## 修订的教学理念

1. **因材施教**：根据每位学生的思维风格（老陈偏数学推导、琳达偏概念理解、小雨偏生活类比）提供多角度说明

2. **不删减，只增加**：所有原有内容保留，新增内容都是**前置铺垫或深入补充**，不改变课程难度进阶

3. **从"不能做"到"能做"**：
   - 练习 2（simple_wer）：从"不知道编辑距离是什么"→ 补充伪代码 → 学生能实现
   - 中文 CER：从"代码看不懂"→ 逐行注释 → 学生理解设计意图

4. **科普风的坚守**：每条新增说明都遵循"问题 / 类比 / 故事 → 公式 → 例子"的流程，避免"显然""不难看出"这类绊脚石词汇
