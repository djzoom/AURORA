# 小雨 — L72 学习日志

## 我理解了什么（简要，3 行以内）

- **LoRA 的核心想法**：冻结原权重 W，旁边只加两个小矩阵 B 和 A（只有 0.5% 的参数），通过 B·A 的乘积作为"补丁"来微调模型，就像给壁画贴滤镜而不是重画。
- **词错误率 WER**：计数方式是 (替换词数 + 删除词数 + 插入词数) / 参考文本总词数。
- **中文适配**：换一个参数就能让模型从识别英文改成中文，用字符级错误率 CER 而不是词级 WER。
