# 小雨 — L27 学习日志

## 我理解了什么

掷骰子很多次，出现"6"的比例会慢慢稳定在 1/6 附近——这就是大数定律。代码里用 `rng.integers(1, 7, size=n)` 生成随机数，然后算 `np.mean(rolls == 6)` 就能估计概率。固定 `seed` 能让实验结果一样，这样调试才有意义。

## 遇到的困难和问题

### 【理解】求和符号∑ 出现得太突兀
在"大数定律正式表述"里写 $\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i$，我看到∑和上下标的 i=1、n 就紧张了。后面代码里隐约理解这是"把所有X加起来除以n"，但课程从未明确说过"∑_{i=1}^n 表示从第1项加到第n项"。能不能在符号第一次出现时加一句"这个∑符号表示求和：X_1 + X_2 + ... + X_n"？

### 【理解】"i.i.d." 这个缩写没有定义
出现了"$X_1, X_2, \ldots, X_n$ 是独立同分布（i.i.d.）随机变量"，但什么是 i.i.d.？虽然括号里有中文翻译，但这两个概念（"独立"+"同分布"）各是什么意思，课程没有单独解释。这对初学者是两个新词汇堆在一起。

### 【理解】代码 `np.mean(rolls == 6)` 为什么能算频率
这行代码，注释说"等价于 sum(rolls == 6) / n"。但 `rolls == 6` 产生的是 True/False 数组，怎样的 True/False 平均值就成了频率？我靠读代码不能心算明白——是 True 被当作1、False 当作0 吗？如果是，课程应该明说一次。

### 【推演】为什么误差量级是 O(1/√n)
表格清楚地显示：n=10 时误差 ±0.3，n=1000 时误差 ±0.03，缩小了10倍。下面说"误差收缩速度：O(1/√n)"，但课程没推导，也没解释 O 符号是什么。如果我算一下 1/√100=0.1、1/√10000=0.01，确实符合，但为什么概率的误差偏偏是 1/√n 这个形式呢？这对视觉型学习者来说，一张对比图会很有帮助。

### 【理解】条件概率公式的直观含义
"P(A|B) = P(A∩B) / P(B)"——这个符号堆砌我能跟着代码理解结果，但概念上不清楚。能不能用"在 B 这个条件下，A 的比例"这样通俗的说法先讲一遍，再写公式？∩ 符号也从未定义，我猜是"同时发生"，但没确认。

### 【理解】贝叶斯定理中的分母 P(B) 是怎么算的
在垃圾邮件例子中，P(free) = 0.9×0.3 + 0.01×0.7 = 0.277。代码里有这个计算，但课程文字没有一步步展开这个分母。为什么要用 P(free|spam)×P(spam) + P(free|ham)×P(ham)？课程应该解释一下"全概率公式"。

### 【实践】实现 `estimate_prob_six` 需要具体步骤
框架代码中 TODO 说"用 rng.integers(1,7,size=n) 生成样本，返回 (sample==6).mean()"，但我不知道具体是怎样写。是：
```python
def estimate_prob_six(n, seed=0):
    rng = np.random.default_rng(seed)
    rolls = rng.integers(1, 7, size=n)
    return np.mean(rolls == 6)
```
这样吗？课程的"参考路线"很有帮助，但没有实际的代码示范。对完全初学者可能需要写出第一个示例。

**总体感受**：这课的故事讲得好（"骰子说谎但概率不说谎"），代码演示也清晰，但数学概念的定义和符号的介绍不够细致。特别是对已经有"数学符号畏惧"的初学者，看到∑、∩、i.i.d.、O() 一个接一个，会比较吃力。建议课程在首次出现每个新符号或术语时都花10秒钟明确定义一下。
