# 老陈 — L27 学习日志

## 我理解了什么

掷骰子很多次（比如 10000 次），出现"6"的频率会稳定在 1/6 ≈ 16.67% 左右，这个规律叫大数定律。掷的次数越多，频率越接近真实概率，误差大约按 1/√n 缩小。另外，两个独立事件同时发生的概率等于两个概率相乘（比如两枚骰子都出 6 的概率 = 1/6 × 1/6 = 1/36）。

## 遇到的困难和问题

### 问题1【理解】随机变量的定义缺失
大数定律的正式陈述开头说"设 X₁, X₂, ..., Xₙ 是独立同分布的随机变量"，但课程里没有定义什么是"随机变量"。掷骰子的例子中，Xᵢ 是代表"第 i 次掷的结果（1～6 中的一个数字）"吗？还是有其他含义？这个概念在学 Python 代码时能看懂，但数学定义不清楚。

### 问题2【推演】条件概率公式 P(A|B) = P(A∩B)/P(B) 的推导
代码里计算 `p_A_given_B = (A & B).sum() / B.sum()` 我能理解这是"B 发生的情况中，A 也发生的比例"，但正式公式为什么是"P(A∩B) 除以 P(B)"？能从样本频率的角度推导一遍吗？比如从 n(A∩B)/n(样本总数) 和 n(B)/n(样本总数) 这两个频率怎么化简到那个公式？

### 问题3【理解】全概率公式从何而来
垃圾邮件的例子中，课程直接给出 `P(free) = P(free|spam)·P(spam) + P(free|ham)·P(ham) ≈ 0.277`。这个公式为什么要把"spam 情况下 free 的概率"和"ham 情况下 free 的概率"加起来？这个"加法法则"有名字吗？为什么不是乘法或其他？

### 问题4【理解】"似然"(Likelihood) 和"概率"的区别
贝叶斯定理里说 P(B|A) 是"似然"，但这不也是一个条件概率吗？为什么特意给它取个不同的名字"似然"？它和其他条件概率有本质区别，还是只是在贝叶斯框架下给它的一个别称？

### 问题5【实践】np.mean(bool_array) 怎么变成比例的
代码 `np.mean(rolls == 6)` 里，`rolls == 6` 产生一个 True/False 数组，然后 mean 就直接得到频率。这是因为 Python/NumPy 把 True 当 1、False 当 0 来计算吗？有没有一个更直白的理由解释为什么 mean 这样算就是频率？
