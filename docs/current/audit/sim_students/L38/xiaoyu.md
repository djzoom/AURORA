# 小雨 — L38 学习日志

## 我理解了什么

1. **FFT 就是一个加速算法**：它计算出来的频谱和 DFT 完全一样，但通过"按偶数/奇数拆开、再一层层合并"，把 N² 次乘法降到 N·log N 次。N=4096 时能快 200 倍左右。

2. **分治的层级结构**：[1,2,3,4,5,6,7,8] → 拆成 [1,3,5,7] 和 [2,4,6,8] → 再拆成 [1,5] 和 [3,7]……一直拆到单个数，然后逐层向上"蝶形合并"。每层合并用加减和一个叫"旋转因子"的乘法。

3. **蝶形操作很简单**：`top = a + W·b`、`bottom = a - W·b`，两个复数输入 → 两个输出。一个 DFT 分治拆的层数是 log₂N。
