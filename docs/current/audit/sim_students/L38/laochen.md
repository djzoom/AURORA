# 老陈 — L38 学习日志

## 我理解了什么

FFT 的核心是**分治法**：一个 N 点的频谱问题可以分解成两个 N/2 点的子问题（偶数下标和奇数下标分别做 DFT），然后用叫"蝶形"的公式把结果合并回去。蝶形就是对一对数值 (E[k], O[k]) 做 `a+W·b` 和 `a-W·b` 这样的加减组合。层数是 log₂N，每层 N/2 次乘法，所以从暴力的 N² 次乘法降到 N·log₂N 次——这就是 FFT 快 200 倍的全部原因。

### 【理解】半步练习中，E = DFT([1,1]) 和 O = DFT([0,0]) 怎么手算？

> 题目说"先在纸上写出 E = DFT([1,1])"。

我知道 DFT 的定义 X[k]=Σ_n x[n]·exp(-2πikn/N)，但手算起来：
- N = 2, E[0] = 1·exp(0) + 1·exp(0) = 2 ✓ 还可以
- E[1] = 1·exp(0) + 1·exp(-πi) = 1 + 1·(-1) = 0 ✓

但 O = DFT([0,0]) = [0, 0]，这太平凡了。真实数据应该不会这么巧合。**我想看一个不那么平凡的例子**，让我真正理解 E 和 O 的值是怎么算出来的。

### 【推演】蝶形公式 X[k] = E[k] + W_N^k·O[k] 是怎么推导出来的？

> 课程直接给了分治公式，但**没有给出推导过程**。

我想看到：给定 x[0..N-1]，按偶奇分：x_e = x[::2]，x_o = x[1::2]。然后从 DFT 的定义 X[k] = Σ x[n]·exp(-2πikn/N) 展开，把偶项和奇项分开，逐步化简，**最终得到 X[k] = E[k] + W_N^k·O[k]** 的形式。这个推导对我来说是个"黑箱"。

### 【理解】递归终止条件"N=1 时 DFT(x) = x"是公式证明吗？

> 课程说"单点信号没有任何频率结构，DFT 输出就是输入本身"。

这个直觉说法我理解，但作为递归的**基础情形**，我想看到严格的数学确认：
- DFT 定义：X[0] = Σ_{n=0}^{0} x[n]·exp(-2πi·0·0/1) = x[0]·1 = x[0] ✓

好的，这个确实对。但问题是：**怎么确保递归一直拆下去，最终所有叶子都收敛到 N=1？** 换句话说，我要确认 N 必须是 2 的幂次（例如 2, 4, 8, 16...）。这个**隐含前提**没有明说。

### 【计算】闭卷题"推导 T(N) = O(N log N)"的求和式展开

> 题目给 T(N) = 2T(N/2) + O(N)，要求展开 k = log₂N 层，写求和式，推导最终复杂度。

我知道**递推关系展开法**的思路：
- T(N) = 2T(N/2) + N
- = 2[2T(N/4) + N/2] + N = 4T(N/4) + N + N
- = 4[2T(N/8) + N/4] + 2N = 8T(N/8) + 3N
- ...
- 总项数 log₂N 层，每层贡献 N

所以 T(N) ≈ N·log₂N。但这个推导**怎么严格地写成求和式？** 我想看到一个标准的记号形式，比如：
```
T(N) = Σ_{i=0}^{log₂N} 2^i · (N/2^i) = ...
```
