# 小雨 — L52 学习日志

## 我理解了什么

从 WAV 音频到 MFCC 特征有一条完整的 6 步管线：分帧 → 加窗 → FFT → Mel 滤波 → 对数 → DCT。每一步都有 Python 函数实现，测试套件能用数学参考答案验证我写得对不对。课程还用代码验证了这些函数的形状、数值精度和数学性质。

## 遇到的困难和问题

### 【理解】RFFT 和 FFT 的区别
> 课程在 `X[k] = sum_{n=0}^{N-1} x[n] * e^{-j 2π k n / N}` 这里突然说"RFFT"，但没解释什么是 RFFT。后面代码注释只说输出是 `(n_frames, 513) complex`。我是高中数学，没学过"实数 FFT"这个概念。为什么信号是实数的，FFT 还是复数输出？为什么要用 RFFT 而不是 FFT？

### 【理解】虚数单位 j 和欧拉恒等式
> 公式中出现 `e^{-j 2π k n / N}`。我知道数学中虚数单位是 i，但这里是 j。更重要的是，我不理解为什么要用"虚数"来描述一个频率。高中没学过欧拉公式 `e^{iθ} = cos(θ) + i sin(θ)`，这里需要什么样的直观比喻或推导？

### 【理解】Hann 窗的周期和对称两种模式
> 课程代码验证了 `hann(8, periodic=True)` 等于 `hann(9, periodic=False)` 去掉最后一点，并说"Hann 周期窗"，但没解释为什么要分两种、什么时候用哪种。`w[n] = 0.5 * (1 - cos(2π n / N))` 这个公式，N 是什么？是窗长度吗？为什么 2π？

### 【理解】正交归一化系数
> 面试题 Q4 提到"DCT-II 正交归一化的归一化系数 w(k) 是什么？（k=0: 1/√N，k>0: √(2/N)；满足 D·Dᵀ=I）"。我不知道"正交"、"归一化"、"D·Dᵀ=I"是什么意思。高中数学没这些概念。课程需要在正文中解释：为什么要这样做？这样做有什么好处？

### 【推演】Mel 滤波器组的三角形构造
> 课程说"三角滤波器组，fb 形状 `(n_mels, N//2+1)`"，然后 `M[m] = sum_k fb[m,k] * |X[k]|²`。但没有画图或示例说明这个三角形长什么样。"顶点坐标在 mel 域均匀分布，转换回 Hz 后不均匀" —— 完全看不懂。需要：
>   1. 一张图，画出 3-4 个三角形叠起来的样子
>   2. 解释 mel 域是什么、Hz 域是什么、为什么要转换

### 【推演】STFT 帧数公式
> 课程说"帧数 = 1 + (T-N)//H（不填充时）；center=True 时变为 1 + T//H"。没有推导过程。高中数学应该能理解整除，但想知道：为什么这样算？什么时候帧数不够整数？

### 【实践】为什么频率维数是 N//2+1
> STFT 输出注释说 `(n_frames, 513)`，513 = 1024//2+1。课程只在"RFFT" 那一行说"保留 N//2+1 个非冗余频点"，没解释为什么恰好是这个数。对称性吗？

### 【理解】atol 数值精度设定
> 测试表格说"FFT `atol=1e-9`，DCT-II `atol=1e-12`"，对比代码解释说"浮点累积误差；log₂N 层 FFT 误差∝N·ε_machine"。完全不懂。什么是"机器精度"？为什么 log₂N 层？能否举个例子算一遍，算出 1e-9 这个数字是怎么来的？

### 【理解】对数压缩 eps 参数
> 讲解说 `log_mel = 10 * log10(M + eps)`。为什么要加一个小的 eps？如果 M 有 0 值，log(0) 会怎样？为什么加 eps 后 Mel 谱不会炸掉？高中数学知道对数没有定义在 0。
