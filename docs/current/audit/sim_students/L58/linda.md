# 琳达 — L58 学习日志

## 我理解了什么

- **训练循环五步是框架**：每轮必须 forward → 算 loss → backward 填梯度 → update 参数 → zero_grad 清零。这个流程很清楚。
- **不清零梯度会炸**：代码演示了不清零时梯度每轮翻倍，参数更新会失控。
- **Hinge Loss 的直观规则**：当 `y*score ≥ 1`（预测正确且置信）时 loss=0，否则 `loss = 1 - y*score`。
