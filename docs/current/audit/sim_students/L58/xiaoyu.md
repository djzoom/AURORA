# 小雨 — L58 学习日志

## 我理解了什么

**训练的 5 步循环**：前向传播算出预测 → 计算损失（数字越小越好）→ 反向传播求梯度 → 参数沿梯度反方向移动 → 清零梯度防止堆积。这个流程是机器学习的核心。

**Hinge Loss 的意思**：就是"如果你猜对了还很确信，损失就是 0；猜错了或不确信，损失就一点点增加"。有个图我看懂了。

**清零梯度的原因**：不清零的话梯度会像雪球一样越滚越大（第 1 轮 0.3，第 2 轮变成 0.6，第 3 轮变成 0.9...），参数会乱飞。
