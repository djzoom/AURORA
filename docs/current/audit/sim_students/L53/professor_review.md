## 本轮修复的实质问题：TODO 逆变换公式的转置错误

在核实"问题6/问题7（B 与 D 的关系）"时，发现 Cell 26 与 Cell 27（TODO 练习）存在**真实的数学错误**，而不仅仅是解释不够直白：

- **错误内容**：Cell 26 "TODO 中的矩阵乘法" 一节写的是 `log_mel_recon = dct_truncated @ B`，但配套的 Cell 27 TODO 注释和"参考代码框架"却写的是 `log_mel_recon = dct_truncated @ B.T`（多转置了一次），且 Cell 26 中间的推导段落也错误地把"多帧（行向量）情形"的逆变换写成 `S_recon = c · D^T`。
