# 琳达 — L18 学习日志

## 我理解了什么

**可逆矩阵的核心含义**：一对一映射的矩阵（det≠0、特征值都非0、满秩）能完美还原，就像ifft还原fft；反之，行列线性相关的矩阵是信息丢失，无法还原。

**严格对角占优（SDD）规则**：逐行检查 `|对角元| > 其他元素绝对值之和` 就能快速判断是否SDD，这对Jacobi迭代求解器很关键。

**三条可逆性判据**：det≠0、特征值全非0、满秩，这三个在数值上指向同一个事实。

## 遇到的困难和问题

### 【理解】零空间的SVD提取逻辑不清楚
在练习中说"零空间向量 = Vt最后一行"，但为什么最后一行对应零空间？课程说最小奇异值对应，但对满秩矩阵，最小奇异值也不是0，那不也会返回一个向量吗？这里需要说清楚：零空间只在秩亏矩阵才「真实存在」，满秩矩阵的零空间其实只有零向量。现在的表述容易让人误解。

### 【推演】Jacobi迭代公式从哪里来
代码给了 `x_new[i] = (b[i] - Σ_{j≠i} A[i,j]·x[j]) / A[i,i]`，但没有推导过程。我猜是从 Ax=b 的第i行逐项分离出来的（`A[i,i]·x[i] = b[i] - 其他项`），但课程应该明说这一步，别让初学者去猜。

### 【实践】Jacobi函数中的代码细节看不懂
`D_inv * (b - (A @ x - np.diag(np.diag(A)) @ x))` 这一行：
- 为什么要"A @ x - 对角部分" 而不直接写 `(A - D) @ x`？ 
- 这两个写法等价吗？NumPy广播规则下是否有性能差异？

代码注释应该拆开这一行，或者提前定义 `off_diag_A = A - D` 再用，这样更清晰。

### 【理解】"线性无关"这个前置概念没有铺垫
秩定义为"线性无关行数"，但课程没有定义什么叫"线性无关"。对于没上过线代的我来说，这是悬空的。至少应该说：两个向量线性无关 = 不存在常数k使得v₂=k·v₁；或者用例子（[[1,0],[0,1]]是线性无关的，[[1,2],[2,4]]是线性相关的）。

### 【理解】为什么SDD矩阵一定可逆
课程说"SDD → 可逆"，证据是M1可逆。但这只是一个例子，不是证明。这背后是什么数学原理？为什么对角线占优能保证没有行被"压扁"？一句话的几何直觉会很有帮助。

### 【理解】"谱半径"这个概念没有定义
在Jacobi收敛的理由里提到"谱半径 < 1"，但课程没有定义这是什么。这是特征值的最大绝对值吗？为什么SDD能保证谱半径 < 1？这段解释缺失了。
