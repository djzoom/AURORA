- **效果**：学生可以看到定量的精度对比，理解为什么代码中 `h=1e-5` 或 `1e-7` 能达到 `1e-9` 的误差

### 3. 【理解】"局部导数"和"全局导数"的区别
**提出者**：琳达、老陈  
**原问题**：这些术语和普通导数有什么区别？为什么要强调"局部"这两个字？

**修订内容与位置**：
- **新增 markdown cell**（Cell 915cb87c）："前置知识补充"中的**第 3 部分**——用表格明确对比：
  - "全局导数"= $\frac{dy}{dx}$（从输出到输入）
  - "局部导数"= 计算图上某条边的导数（如 $\frac{du}{dx}=2x$）
  - 反向传播 = 把全局拆成局部乘积

### 8. 【理解】三层复合的链式法则乘法顺序容易错
**提出者**：小雨  
**原问题**：从最外层反推 `sin → exp → x²`，怎么确定乘法顺序是"从外向内"而不是"从内向外"？

**修订内容与位置**：
- **新增 markdown cell**（Cell 737f689b）：在 3 层复合前插入推导表格
  - 四行表格明确标记顺序：最外层→中层→最内层→连乘
  - 一句规则：$\frac{dh}{dx} = [\text{最外层导数}] \times [\text{中层导数}] \times [\text{最内层导数}]$
  - 直觉说明：放大倍数从内向外累乘
