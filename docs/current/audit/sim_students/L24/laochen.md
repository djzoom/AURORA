# 老陈 — L24 学习日志

## 我理解了什么（简要，3 行以内）

1. **链式法则的核心**：复合函数 y=f(g(x)) 的导数 = f'(g(x)) · g'(x)，是一条"乘法链"。我用数值差分验证过 sin(x²)、sin(exp(x²)) 和 tanh(sin(x²+1)) 的情况，都吻合。

2. **任意深度的推广**：2 层、3 层、4 层复合函数都遵循同一个原则——按照每层的局部导数连乘。这个乘法链的思想就是反向传播的种子。

3. **代码实现与验证**：composite_derivative 的实现就是把链式法则公式直译成 NumPy 代码，中心差分验证了解析解的正确性。

## 遇到的困难和问题

### 【理解】为什么要用中心差分公式 (f(x+h)−f(x−h))/(2h) 来近似导数？
Cell 3332b773 和 dc8b8a9f 中，数值导数都是用中心差分计算的。我知道导数的定义是 lim_{h→0} [f(x+h)−f(x)]/h，但课程直接跳到中心差分，没有解释为什么这个"左右对称"的公式能逼近导数。h=1e−5 或 h=1e−7 这样的数值如何选择？太大或太小分别会怎样？

### 【理解】链式法则为什么一定是"乘法"而不是加法或其他运算？
Cell 48b17217 给出的直觉是"内层放大倍数 × 外层放大倍数"，这个类比好记。但能否用更严格的方式解释：为什么是相乘而不是相加？比如，为什么不是 dy/dx = dy/du + du/dx？课程没有从极限定义或泰勒展开的角度推导这个结论。

### 【理解】Cell b4185414 里"计算图"和"局部导数"的概念太模糊
代码把 y=sin(x²) 拆成中间节点 u=x²，分别计算 local_du_dx=2x 和 local_dy_du=cos(u)，然后相乘。这和链式法则公式 f'(g(x))·g'(x) 的对应关系是什么？"计算图的边"是什么意思？一条边=一个局部导数吗？课程多次提到"计算图"在 L54 反向传播会用到，但没有明确定义，这让我觉得理解不完整。

### 【理解】Cell 15a8e7d6 中 tanh 函数及其导数公式
4 层复合中突然出现 tanh 函数和导数公式 (1−tanh²(x))，但这在高中数学里根本没有学过。tanh(x) 是什么？它的导数 1−tanh²(x) 是怎样推导出来的？我能通过数值验证确认公式正确，但不知道它的来源。建议课程要么提前介绍 tanh，要么用高中范围内的函数（如 e^x）代替。

### 【实践】composite_derivative 实现完成
已实现并通过数值验证，结果正确。
