# 琳达 — L24 学习日志

## 我理解了什么

链式法则就是"外层导数 × 内层导数"：复合函数 y=f(g(x)) 的导数等于 f'(g(x))·g'(x)。比如 y=sin(x²)，我需要先求外层 f(u)=sin(u) 对 u 的导数（cos(u)），再求内层 g(x)=x² 对 x 的导数（2x），最后两个相乘就是 dy/dx。这个思想对 3 层、4 层嵌套函数也一样，就是一直乘下去。
