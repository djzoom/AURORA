# 小雨 — L24 学习日志

## 我理解了什么

链式法则就是"分层剥洋葱"：复合函数 y = f(g(x)) 的导数等于外层导数乘以内层导数，即 dy/dx = f'(g(x))·g'(x)。用 y = sin(x²) 的例子，把 x² 看成中间变量 u，然后外层对 u 求导（cos(u)）再代入 x²，内层对 x 求导（2x），最后乘起来。通过数值差分验证，这个公式在每个点都对得很准。
