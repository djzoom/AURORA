# 小雨 — L15 学习日志

## 我理解了什么（简要，3 行以内）

**秩判断法**：要判断 `Ax=b` 有没有唯一解、无穷多解还是无解，只需比较系数矩阵 A 的秩和增广矩阵 [A|b] 的秩。秩相等且等于未知数个数就唯一解，秩相等但小于未知数个数就无穷解，秩不相等就无解。**列的线性组合视角**：解矩阵方程 `Ax=b` 的本质就是找一组系数 x，使得 A 的各列加权组合能精确拼出向量 b。我实现了 `classify_system` 函数来自动判断方程组的三种命运，也看懂了初等行变换（交换行、行倍乘、行消元）如何逐步简化增广矩阵。
