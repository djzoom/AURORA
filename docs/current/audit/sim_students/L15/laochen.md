# 老陈 — L15 学习日志

## 我理解了什么

1. **Ax=b 是在找系数**：矩阵 A 乘向量 x 等于 b，其实就是要用 A 的各列加权组合拼出 b。比如 A 有两列 [2,0] 和 [1,1]，我要拼出 [5,3]，就是找到系数 x₀=1、x₁=3，使得 1×[2,0] + 3×[1,1] = [5,3]。

2. **用秩判断有几个解**：算出系数矩阵的秩 rank(A) 和增广矩阵的秩 rank([A|b])，跟未知数个数 n 比较，就能知道方程组是唯一解、无穷多解还是无解。规则很清楚。

3. **初等行变换能把方程组化简**：交换行、某行乘倍数、某行加上另一行的倍数这三种操作，都不改变方程组的解，可以用来消元。
