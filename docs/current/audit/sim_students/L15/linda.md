# 琳达 — L15 学习日志

## 我理解了什么

**1. 矩阵方程 Ax=b 的几何意思：** A 的各列向量组合起来能否精确拼出向量 b。这用矩阵乘法的列视角解释很清楚——x[0]×列₀ + x[1]×列₁ + ... = b。

**2. 用秩(rank)来分类方程组的解：** 计算 rank(A) 和 rank([A|b])，对比它们与未知数个数 n 的大小关系，就能判断唯一解/无穷解/无解。我能写出代码实现，也通过了四个例题的验证。

**3. 初等行变换的三种操作：** 交换行、某行乘倍数、某行加上另一行的倍数，这三种操作可以把增广矩阵逐步变形，目的是把未知数分离出来。代码演示很清楚。

## 遇到的困难和问题

### 【理解】秩(rank)的定义和含义缺失  
**卡在：** 课程使用 `np.linalg.matrix_rank(A)` 反复出现，但从未解释"秩"是什么。  
**具体问题：** rank=2、rank=3 这些数字表示什么意义？为什么有时候 rank 少于 n，就意味着有"自由变量"？秩是数行数、还是列数、还是别的什么？  
**希望补充：** 需要一句话定义秩（比如"秩=矩阵中线性独立的行数"或其他），或者用一个 2×2 的小例子展示秩的变化。

### 【理解】"列空间"(column space)概念突兀  
**卡在：** 课程说"b 不在 A 的列空间中"是无解的原因，但什么是列空间没有定义。  
**具体问题：** 是说 b 无法表示为 A 的列的线性组合吗？如果是，这和 rank 的关系是什么？为什么 rank([A|b]) > rank(A) 就等价于"b 不在列空间"？  
**希望补充：** 列空间的直观定义，以及它和秩的关系；或者用一个反例（比如 [[1,0],[0,1]] 的列空间是 ℝ²）来说明。

### 【推演】为什么"rank(A) < rank([A|b])"意味着无解  
**卡在：** 这是整个分类规则的关键，但课程没有给出推理链条。  
**具体问题：** 增广矩阵 [A|b] 比 A 多出一列，为什么它的秩会比 A 的秩更大？增加一列后秩为什么会增加？这在秩的定义层面应该说明白。  
**希望补充：** 一个具体例题（比如 [[1,2],[2,4]] 的 rank=1，但 [[1,2,3],[2,4,7]] 的 rank 会怎样），用行化简或列的线性关系来说明为什么秩会变。

### 【理解】初等行变换为何不改变方程组的解  
**卡在：** 代码演示了如何做行变换，但没有说明为什么这样变形后，新方程组的解和原方程组一样。  
**具体问题：** ER3 操作 R₂ → R₂ - kR₁，这对应原方程组的什么操作？是"第二个方程加上第一个方程的倍数"吗？为什么这样做解不变？  
**希望补充：** 从方程组的角度解释一遍（方程组两边同时乘某数、两方程相加，这些操作为什么保持解不变），这样理解才会深入。

### 【理解】行列式和奇异矩阵的定义  
**卡在：** 代码中出现 "det(A)=0 时矩阵奇异"、`np.linalg.LinAlgError`，但没有解释行列式和奇异的概念。  
**具体问题：** 行列式是什么？为什么 det=0 时 solve 会失败？"奇异"这个词是什么意思？  
**希望补充：** 行列式的定义（至少对 2×2 或 3×3 矩阵），以及为什么 det=0 对应不可逆。

### 【理解】lstsq 返回的 singular values 是什么  
**卡在：** `np.linalg.lstsq(A, b)` 返回四个值，其中 `s`（singular values）没有解释。  
**具体问题：** 为什么有两个奇异值 [5.0, 1.8e-16]？较小的奇异值（接近0）和矩阵秩的关系是什么？  
**希望补充：** 奇异值和秩的关系（我猜：非零奇异值个数=秩），或者这涉及 SVD（可能是 L14 内容），那就说一句"详见 L14"。
