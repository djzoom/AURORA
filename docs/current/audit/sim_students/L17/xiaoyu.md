# 小雨 — L17 学习日志

## 我理解了什么
特征分解 A=PDP⁻¹ 的核心就是换坐标系：在特征向量构成的坐标轴下，矩阵变成对角矩阵，矩阵乘法就只是各方向独立地缩放（缩放因子=特征值）。验证了特征方程 det(A−λI)=0 的根确实是特征值，用 char_poly 函数可以验证；矩阵幂次可以用对角化快速计算：Aⁿ = PDⁿP⁻¹。

## 遇到的困难和问题

### 【理解】为什么特征向量能当坐标轴？
课件说"换到特征向量坐标系，乘法退化成数乘"，但没有解释**为什么**选择特征向量作为坐标轴就能做到这一点。我理解 Av=λv 这个式子，但不明白这个式子怎么就能推出"用 P（特征向量按列拼接）当坐标变换矩阵，新坐标系下矩阵就是对角的"。这个从一维关系（Av=λv）跳到整个矩阵对角化（A=PDP⁻¹）的逻辑没有说明。

### 【推演】det(A−λI)=0 这个式子来得太快
课件说："Av=λv 等价于 (A−λI)v=0，有非零解当且仅当 A−λI 奇异"——这里卡住了。什么是"奇异矩阵"？为什么有非零解就代表矩阵奇异？我需要更详细的推理：齐次线性方程 (A−λI)v=0 什么时候有非零解，这个和行列式为零是什么关系？这个连接在高中数学里没学过，课件假设了我已经知道。

### 【理解】为什么对称矩阵特别（正交对角化 PᵀAP=D）
课件提到对称矩阵可以用正交矩阵对角化 PᵀAP=D，这和通用的 A=PDP⁻¹ 有什么区别？什么是"正交矩阵"？为什么对对称矩阵就能做到 Pᵀ=P⁻¹，但其他矩阵不行？课件没有定义"正交"，也没有解释为什么这个性质对对称矩阵独有。

### 【理解】旋转矩阵特征值为什么是复数
课件说"旋转矩阵的特征值是共轭复数"，但没有举例或解释**为什么**。一个实矩阵（纯几何变换）怎么会有复数特征值？这种情况什么时候出现？我需要具体例子。

### 【理解】"不是所有矩阵都能对角化"这句话太抽象
课件只说 [[1,1],[0,1]] 这样的矩阵"凑不齐 n 个独立方向"，但这个矩阵的特征值是什么？它的特征向量是什么？为什么凑不齐？这个反例没有详细说明。

### 【实践】Aurora 应用的连接不够具体
课件多次提到"递归滤波器稳定性""状态空间模型""极点分析""协方差矩阵"，这些术语在之前的课程里没出现过，我不知道这些概念是什么，无法理解为什么特征值很重要。建议补充：什么是递归滤波器？状态空间模型是什么样的数学结构？为什么 |λ|<1 就代表系统稳定？这需要背景知识铺垫。
