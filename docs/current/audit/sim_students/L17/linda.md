# 琳达 — L17 学习日志

## 我理解了什么（简要，3 行以内）

1. **特征值和特征向量的含义**：对矩阵 A，如果 `Av = λv`，说明 A 作用在向量 v 上的结果就是把它缩放 λ 倍。特征向量是矩阵"喜欢"的方向，特征值是沿着这个方向的缩放因子。

2. **特征分解 A = PDP⁻¹ 的用途**：把矩阵写成特征向量矩阵 P、特征值对角矩阵 D、和 P 的逆的乘积。好处是计算矩阵幂次变简单：`Aⁿ = PDⁿP⁻¹`，D 的幂次就是对角线各元素自己幂次。

3. **稳定性判据**：递归系统是否收敛只看特征值——如果所有 |λᵢ| < 1，则系统稳定，`Aⁿ → 0`；这对 Aurora 的递归滤波器很关键。
