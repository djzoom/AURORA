## 4. 正交矩阵是什么？为什么对称矩阵能用 PᵀAP=D 而不是 PDP⁻¹？

涉及：老陈（「正交矩阵与普通特征向量矩阵的区别」）、琳达（问题 4）、小雨（「为什么对称矩阵特别」）。

→ 在 `## 3. 对称矩阵对角化` 一节（cell `31a172a4`）追加大段说明：
  - "正交（orthogonal）"的人话定义：列向量都是单位向量，且两两垂直（标准正交基），并解释 `PᵀP` 第 (i,j) 项就是第 i、j 列的点积，因此对角为 1、非对角为 0 正好是 `PᵀP=I`。
  - 解释为什么这有用：一般矩阵求 `P⁻¹` 要单独算逆（慢、可能不稳定），但正交矩阵满足 `P⁻¹=Pᵀ`，转置几乎不用计算，这就是 `PᵀAP=D` 比 `PDP⁻¹` 更快更稳的原因。
  - 给出对称矩阵特征向量两两正交的证明：用 `v₁ᵀAv₂` 两种算法（一边用 `Av₂=λ₂v₂`，一边用 `Aᵀ=A` 展开 `(Av₁)ᵀv₂`）得到 `(λ₁-λ₂)(v₁·v₂)=0`，λ 不同则点积必为 0。
→ 配套代码 cell `a2284b13`：对课件给出的 v1,v2,v3 直接算三对点积，数值验证两两垂直。
