# 小雨 — L28 学习日志

## 我理解了什么

**均值、方差、标准差** 这三个概念讲得很清楚！通过"把所有数据的重量放在一条数轴上，均值是平衡点"这个比喻，我能理解 μ 就是求和再除以个数。方差就是每个点离均值有多远（平方后再平均），标准差就是方差开方。手算例子 [2,4,4,4,5,5,7,9] 也能跟上：μ=5，σ²=4，σ=2。

**z-score 标准化** 的目的很好理解——把不同量级的数据都拉到"均值=0、标准差=1"的同一个"语言"里，这样才能公平比较。公式 (x - μ) / σ 也简单，相当于先"往下移动μ单位"，再"缩放到以σ为单位"。

**实现思路** 很清楚：算出均值，算出标准差，然后用公式算 z-score，最后验证结果的均值接近0、标准差接近1。
