# 琳达 — L37 学习日志

## 我理解了什么

DFT 就是一个双循环：对每个频率索引 k（外循环），用 N 个旋转因子 `e^{-2πikn/N}` 逐点"询问"信号，最后求和。这个和就是 X[k] = 点积(信号, 旋转因子序列)。代码实现很直接：`for k in range(N): X[k] = dot(x, exp(-2j*pi*k*arange(N)/N))`。理解了物理频率就是 `f_k = k·sr/N`，所以 k 越大频率越高。

## 遇到的困难和问题

### 问题 1【推演】欧拉公式的具体计算
在手算 4 点 DFT 的例子里（`x=[1,0,-1,0]`），课程写了 `e^{-πi/2}`、`e^{-πi}` 等，并直接说 `e^{-πi}=-1`、`(-1)·(-1)=+1` 之类的。我高中学过 `e^{πi}=-1`，但一般的 `e^{iθ}` 怎么算？是不是应该用 `e^{iθ}=cos(θ)+i·sin(θ)`？课程为什么没有明确给出这个公式和一个计算示例，比如"e^{-πi/2} = cos(-π/2) + i·sin(-π/2) = 0 - i"？

### 问题 2【理解】酉矩阵和"正交变换"是什么
第 1 小节突然冒出"酉矩阵""正交变换""不同频率的基向量彼此垂直"这几个词。我没学过线性代数，完全不知道这些是什么。课程说"F 是酉矩阵（除以 √N 后）"，为什么要除以 √N？为什么这说明向量"垂直"？这些概念对于理解 DFT 是必需的吗，还是只是额外的数学性质？

### 问题 3【理解】负号代表顺时针还是逆时针
第 2 小节说"W_N^{k·n} 是单位圆上**顺时针**转了 k·n/N 圈的点"。但是在我印象中，复数乘以 `e^{iθ}` 是逆时针旋转（θ>0 时），乘以 `e^{-iθ}` 才是顺时针。这里 `exp(-2π i·...)` 的负号，为什么对应顺时针？课程只说"也就是 L35 的分析方向"，但我没看 L35，能否直接解释一下为什么是负号？

### 问题 4【理解】为什么实信号满足共轭对称
课程说"实信号必然满足共轭对称：X[k] = conj(X[N-k])"，但没有给证明。这是个事实还是有深层原因？我自己尝试推导了一遍（设 x[n] 是实数，展开 X[N-k] 和 conj(X[k]) 的定义式，发现它们相等），推导出来了，但课程为什么不主动给出这个推导？初学者是应该记住这个公式就行，还是需要理解它的来源？
