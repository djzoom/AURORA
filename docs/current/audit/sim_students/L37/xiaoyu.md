# 小雨 — L37 学习日志

## 我理解了什么

DFT 是用一种固定的"询问方式"（旋转因子）逐频点扫描信号，每个频点的答案是信号与该频率基向量的点积。双循环实现就是暴力问 N 个频点，每次问 N 个样本点，所以是 O(N²)。
