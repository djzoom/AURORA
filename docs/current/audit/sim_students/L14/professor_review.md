1. **"为什么从 Av=λv 这个等式得到？"**（位置：`## 1. 特征向量：Av=λv` 一节开场直接给公式，没有推导）

   从高中向量倍数（λ>0 同向、λ<0 反向）推到"共线即方向不变"，再推导 `(A-λI)v=0` 特征方程的来源，解释为何用等式而非 `Av ∝ v`。

2. **"秩（rank）是什么意思？"**（位置：全课反复使用"秩-k""低秩"却从未定义）
   → 在《🤔 为什么工程师要发明它？》一节后插入新 markdown+code cell《插播：秩到底是什么？》：
   用"重复联系人信息"类比给出秩的直观定义，配 `[[1,2],[2,4]]`（秩1）和单位矩阵（秩2，满秩）两个具体例子，并用 `np.linalg.matrix_rank` 验证；同时预告"秩=非零奇异值个数"这个后续会用到的等价说法。

3. **"U、V 各代表什么方向？对角矩阵 A=diag([2,3]) 的 U/V 为什么不是单位矩阵？"**（位置：`## 3.1 SVD：把矩阵拆成方向和强度`）
   → 在该 cell 后插入：
   - markdown《追问：为什么偏偏是这个顺序？为什么 U、Vᵀ 必须是正交矩阵？》，说明"转-拉-转"的分解动机，以及正交性保证 S 独自承担拉伸职责的原因；
   - code cell 用 2×2 矩阵逐步打印 `x → Vt@x → S*... → U@...` 三步过程，和直接算 `A@x` 对比；
   - markdown+两个 code cell，专门针对老陈的对角矩阵例子：`diag([3,2])`（已降序）时 U=Vt=单位矩阵；`diag([2,5])`（未降序）时 SVD 会重排顺序，U 不再是单位矩阵——直接回答"为什么不是单位矩阵"。

4. **"特征向量为什么有复数？"**（位置：运行 `np.linalg.eig(A)` 发现输出带 `+0.j`）

   说明 `np.linalg.eig` 统一以 `complex128` 返回（不论矩阵是否对称），对称矩阵的虚部恰好为 0 但外壳仍是复数；旋转矩阵 `[[0,-1],[1,0]]` 则有真正非零虚部；并介绍 `np.linalg.eigh` 作为对称矩阵的纯实数替代方案（用真实代码验证过三种情况的 dtype 和数值）。

5. **"Full SVD vs Thin SVD 在低秩近似中的区别，为什么还要 Thin？"**（位置：《符号入口：本节关键 shape》）
   → 在该 cell 后插入 markdown+code cell《追问：Thin SVD 到底省在哪？》：
