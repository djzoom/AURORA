# 琳达 — L14 学习日志

## 我理解了什么

1. **特征向量 = 矩阵最"偏爱"的方向**：矩阵 A 作用在特征向量 v 上，只会把它拉伸 λ 倍（Av = λv），方向根本不改变。对比普通向量[1,1]被[[2,0],[0,3]]拉成[2,3]就歪了。

2. **SVD = 矩阵的"指纹分解"**：任何矩阵都能拆成 A = U·S·V^T，其中S里的奇异值从大到小排列，代表每个方向的"重要性"。只保留前k个，就能快速压缩矩阵——这是音频去噪、推荐系统、LoRA的共同核心。

3. **低秩近似 = 扔掉不重要的方向**：秩-1近似虽然误差大，但能看出主要特征；秩-2能基本还原秩为2的矩阵；这简直就是现实数据的天然写法——因为真实数据大多"藏"在前几个方向里。
