# 小雨 — L39 学习日志

## 我理解了什么（简要）

**FFT 的核心想法**：不像傻瓜 DFT 一样把 N 个信号逐个跟 N 个频点全部相乘（N² 次乘法），而是用"分治"的思路——把信号按偶数下标和奇数下标拆成两半，分别递归做 FFT，再用"蝶形"公式合并结果。这样层数从 N² 降到 N·log N，当 N=1024 时快 200 倍。

**递归的结构**：长度 1 的信号直接返回（基础情形），然后不断拆分偶奇、缩小问题规模，最后在返回的路上一层层合并。
