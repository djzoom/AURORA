# 小雨 — L25 学习日志

## 我理解了什么

梯度下降就是"蒙眼下山"：在当前位置测梯度（斜率），然后沿着**反方向**走一小步 `x_new = x - lr * grad`，重复几十次就能逼近山谷。学习率 lr 是步长控制器——太小走得慢，太大容易冲过头反弹。
