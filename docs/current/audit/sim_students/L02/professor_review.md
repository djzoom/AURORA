### 主题 3：公式里的 2π 从天而降（阿凯 Q2 / 老陈 Q2 / 琳达 Q2）
- **改动 E**：任务 2（c12）在公式之前追加「2π 是从哪冒出来的？」——摩天轮
  类比 + 三步推演（f = 每秒转 f 圈 → t 秒共 f·t 圈 → 每圈 2π 弧度 →

### 主题 5：手算表 sin(0.785) 没法徒手算（阿凯 Q3 / 琳达 Q3 / 小美 Q2 / 小雨 Q3）
- **改动 G**：任务 2（c12）追加「弧度急救包」：π rad = 180° 换算、
  特殊角对照表（0.785=π/4=45°、1.571=π/2=90°、2.356=3π/4=135°，
  sin45°=√2/2≈0.71），并警告"别按 0.785 度算（≈0.014，差几十倍）"；
  原手算表增加"θ 换算"一列（π 的几分之几 + 度数），原有各列全部保留。

### 主题 6：make_time_axis 为何"不含终点"、linspace 陷阱（阿凯 Q4 / 老陈 Q5）
- **改动 H**：任务 1（c7）追加「为什么最后一个 < duration？」——含头不含尾
  的两个理由（两段拼接不重复计点；N = duration×sr 账目干净），老陈的切萝卜
  比喻（数的是"每小段的起点"），并提前排雷：`np.linspace(0, duration, N)`
  两头都含、间隔是 duration/(N-1) 而非 1/sr，会挂断言。

### 主题 7：RMS 无动机、A/√2 只能背（老陈 Q3 / 琳达 Q4 / 小美 Q3 / 小雨 Q4）
- **改动 I**：任务 3 之前新插 markdown（e3b51efc）：RMS 回答"平均有多响"；
  mean 正负抵消量不出响度，所以平方→平均→开根号；Root-Mean-Square 名字即
  步骤倒念；再用"sin² 在 0 与 1 之间关于 1/2 上下对称 → 平均值 = 1/2"
  推出 RMS = √(A²/2) = A/√2，注明严格证明需积分、后续课程会碰到。
- **改动 J**：其后新插可运行演示 cell（723f6982）：上下两幅图画 sin 与
  sin²（虚线标 1/2 平均线，回应小美"一张图就够"）；再遍历 4 组不同
  freq/sr/dur 打印 RMS 全部钉在 0.7071 附近（回应琳达"多举例让它不像魔法"），
  并说明非整数周期时的微小偏差。白板挑战 Q4 的提示同步加了回看指引。
