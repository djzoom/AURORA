### 【理解】normalize 的全局 mean/std 为什么不会"抹掉"时间变化？

normalize 做的是：
```
X_norm = (X - X.mean()) / (X.std() + eps)
```

其中 X 是整个 (40, 32) 的矩阵。但如果所有 40×32=1280 个格点都用同一个均值和方差，那不就是说：
- "yes" 的 Mel 图前半段有能量，后半段是零 → 平均后拉平
- 所有样本都用同一个均值/方差 → 不同关键词的能量对比被磨平？

white board 挑战说"逐帧会破坏帧间能量对比，全局保留时间动态"，但我看不出为什么。

**需要补充：** 一个例子：两个样本，一个前半段全是 1、后半段全是 0；另一个相反。用全局 normalize 和逐帧 normalize 各算一遍，对比结果，说明哪种能保留"前后的能量起伏"。
