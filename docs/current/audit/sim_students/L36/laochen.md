# 老陈 — L36 学习日志

## 我理解了什么

1. **窗函数的必要性**：截断信号边缘会产生虚假高频成分（频谱泄漏），加窗就是让信号边缘平滑过渡，从 0 缓慢升到最大再降回 0，这样 FFT 就不会把边缘跳变当成真实的频率成分。

2. **四种窗的权衡**：矩形窗两端为 1（不处理）→ 旁瓣高；Hann/Blackman 两端接近 0（彻底压低边缘）→ 旁瓣低但主瓣变宽；Hamming 两端约 0.08（保留一点能量）→ 折中方案。

3. **加窗的操作很直接**：就是逐点相乘 `signal * window`，窗的值在 0 到 1 之间，相当于给信号的每个点配一个"权重"。
