# 琳达 — L36 学习日志

## 我理解了什么

**窗函数是给信号两端做"淡入淡出"的权重**——时域上乘以一个数组 `signal * window`，边缘接近 0、中间接近 1，这样截断边界就不会突然跳变。矩形窗（全 1）会产生高频污染（旁瓣），而 Hann/Hamming/Blackman 这些平滑窗能压低旁瓣。Aurora 的 STFT 在每一帧都用 Hann 窗就是为了避免这个问题。
