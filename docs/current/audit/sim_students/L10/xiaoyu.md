# 小雨 — L10 学习日志

## 我理解了什么

点积就是两个向量逐个元素相乘，然后把乘积都加起来：`[1,2,3]·[4,5,6] = 1×4 + 2×5 + 3×6 = 32`。如果两个向量同向点积是正数、垂直是0、反向是负数，这样点积可以衡量"两向量有多像"。为了忽略向量长度的影响，除以两个向量长度的乘积，就得到余弦相似度，它固定在 -1 到 1 之间，用来比较歌曲推荐、频率匹配等场景。

## 遇到的困难和问题

### 【理解】复指数序列和 DFT 这一段太突兀了

> 原文位置："DFT 里，`X[k] = Σ x[n] · e^{-2πikn/N}` 就是信号数组与复指数序列的点积"

我看到了这个公式，但是：**复指数是什么？** `e^{...}` 我只知道 `e ≈ 2.718`，但这里指数是复数吗？从高中数学，我从没见过这个。课程说"注意力机制也是点积"，但没有解释复数参与计算会怎样。这对我来说是个黑盒，没有铺垫。  
**希望课程补充**：用最简单的例子（比如 1Hz 和 2Hz 的正弦波），解释为什么把信号与复指数做点积能提取频率；或者先讲"为什么用复数"。

### 【推演】点积的两种形式怎么相等？代数式 `Σaᵢbᵢ` = 几何式 `|a||b|cosθ`

> 原文位置："几何解读：`a·b = |a| · |b| · cos(θ)`"

课程说这两个是等价的，但怎么推导？特别是：
- 对 1000 维的向量，θ（两向量的夹角）还有几何意义吗？
- 为什么是 cos(θ) 不是 sin(θ) 或别的三角函数？  

纸上我推不出来，只能硬记住"这样定义"。  
**希望课程补充**：最少给 2D 例子推导一遍，说清楚为什么 cosine。

### 【理解】向量长度的三种记号搞混了

> 原文位置：多处混用 `|a|`、`||a||`、`np.linalg.norm(a)`

文中说"向量长度（下一课细讲）"，用的是 `|a|`；之后代码里是 `np.linalg.norm(a)`；有时候数学符号是 `||a||`（双竖线）。  
**我的困惑**：这三个一样吗？为什么要三种写法？数学教科书里看到的是 `||a||`，但代码里是 `np.linalg.norm()`，这个函数具体算的是什么？（平方和开方吗？）  
**希望课程补充**：明确说"在这个课程，`|a|` 就是 `||a||` 就是 `np.linalg.norm(a)`，都表示向量长度，计算方式是...(公式)"。

### 【理解】为什么零向量要 raise ValueError？

> 原文位置："若任一向量为零向量（模为 0），raise ValueError"

代码要求检查零向量并报错。但为什么？  
- 点积确实等于 0（所有元素都 0）  
- 余弦相似度 = 点积 ÷ (|a|×|b|)，如果 |a| = 0，分母是 0，无法计算  
- 但什么时候会遇到零向量？音乐推荐里会出现"没有任何特征"的歌吗？  

这个边界条件对我来说很人为，不知道实际用处。  
**希望课程补充**：给个真实场景例子，为什么零向量会出现，为什么不能处理。

### 【推演】投影的两个公式为什么分母不一样？

> 原文位置：第 4 节投影部分

标量投影：`a·b / |b|`  
向量投影：`(a·b / |b|²) · b`

为什么一个除以 `|b|`，一个除以 `|b|²`？特别是后面那个，为什么投影回向量 b 的方向就要多除一个 `|b|`？  
代码里直接计算 `np.dot(b, b)` 就是 `|b|²`，但这两个公式的推导逻辑不透彻。  
**希望课程补充**：从几何角度或代数推导说明为什么分母要这样设计，最好配个图示。
