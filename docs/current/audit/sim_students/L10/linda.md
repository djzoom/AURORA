## 我理解了什么

### 【推演】Cell 19：向量投影公式 (a·b / |b|²) · b 怎么推出来的？
课程给了标量投影 = a·b / |b| 和向量投影 = (a·b / |b|²) · b，但没有推导。我验证了数值是对的，但**不明白为什么分母突然从 |b| 变成 |b|²**。

逻辑应该是：
1. 标量投影告诉我投影的"长度"
2. 然后乘以 b 方向的单位向量才能得向量投影

所以应该是 (a·b / |b|) · (b / |b|) = a·b · b / |b|²，这样对；但公式直接写成 (a·b / |b|²) · b 没有说明推理。

**希望补充**：完整推导步骤，或者说明 |b|² = b·b 这个关键等式。
