# 老陈 — L10 学习日志

## 我理解了什么

点积就是两个向量对应位置的数字一个个相乘，最后全加起来，得到一个单独的数字。这个数字能看出两个向量的方向有多接近：同向是正数、垂直是零、反向是负数。余弦相似度就是把点积除以两个向量的"长度之积"，这样就不管向量有多大了，只看方向，结果总在-1到+1之间。投影就是把一个向量在另一个向量方向上的"影子"算出来。
