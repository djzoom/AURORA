# 琳达 — L40 学习日志

## 我理解了什么

FFT 输出的 bin 号需要通过公式 `k × sr / N` 转成实际的 Hz 频率。实信号的 FFT 左右对称，所以只需要取前一半（0 到 N/2）就能拿到全部信息，这叫"单边谱"。幅度谱、功率谱、dB 谱是对同一个 FFT 输出的三种不同展示方式。

## 遇到的困难和问题

### 【理解】频率分辨率公式为什么是这样

第 1 节说"FFT 将 `[0, sr)` Hz 均匀分成 N 份"，然后 Δf = sr/N。但这个公式是怎么推导的？**为什么FFT一定是从 0 到 sr 这个范围？为什么要除以 N？** 课程没有给出这背后的数学理由（可能是采样定理的推论？但L40没讲）。

### 【推演】共轭对称性 |X[k]| = |X[N-k]| 是怎么来的

第 2 节说实信号满足"共轭对称"，但没有推导过程。**为什么输入是实数就会导致这种对称？** 能给一个简单的例子吗？比如，一个长度为4的实信号 [1, 2, 3, 4]，它的 FFT 前两个和后两个系数具体是什么关系？

### 【推演】相位谱反对称性 φ[k] = −φ[N−k] 和共轭对称的关系

第 3b 节提到相位反对称，说"与幅度对称性完全一致"。**如果 X[k] = |X[k]|·e^{jφ[k]}，和 X[N-k] = X[k]*（共轭），这两个条件怎么推出相位反对称？** 我试着从 X[N-k] = X[k]* 推导，但半路卡住了。

### 【理解】"dB 谱把乘法变成加法"这句话什么意思

第 3 节说"dB 谱把乘法关系变成加法"。**具体指什么乘法关系？** 比如，如果两个信号的幅度要相乘（比如级联两个滤波器），为什么在 dB 下就变成相加了？这是 log 函数的性质吗？为什么这样做就能"更容易分辨细节"？

### 【实践】白板问 3："区分两个音符"需要 Δf ≤ 10 Hz，这个标准是怎么定的
