# 老陈 — L40 学习日志

## 我理解了什么

频率分辨率 **Δf = sr / N**，决定了FFT中相邻两个频率桶（bin）之间间隔多少Hz。完整频率轴是 `k × sr/N`（k=0 到 N-1），但实数信号满足共轭对称，只需看前半段（k=0 到 N//2），共 N//2+1 个点，从0 Hz到奈奎斯特频率 sr/2。幅度谱|X[k]|、功率谱|X[k]|²、dB谱20log₁₀(|X[k]|)是同一个数据的三种表现形式。

## 遇到的困难和问题

### 【理解】共轭对称为什么会发生？—— Cell 7-8

教材说"当输入 x[n] 是实数时，FFT满足共轭对称：|X[k]| = |X[N-k]|"，然后直接用这个事实。但这是个什么鬼规律呢？为什么实数输入就一定导致复数FFT的后半段是前半段的镜像？

**卡在的具体位置**："因此相邻两个 bin 之间的频率间隔为…" — 后面直接跳到性质本身，没有推导。

**希望的补充**：用一个 N=4 的小例子，比如 x = [1, 0, -1, 0]（实数），手算出四个FFT系数是复数还是什么，然后展示 X[0] 对应 X[0]、X[1] 对应 X[3]、X[2] 对应 X[2]... 这样的具体对应关系，或者从欧拉公式推导为什么一定有这个对称性。

---

### 【推演】bin 下标为什么对应 k × sr/N 这个频率？—— Cell 4-5

我能看出来"相邻两个 bin 间隔 Δf = sr/N"，也能看出来列表 `freqs = np.arange(N) * sr/N` 会产生 [0, sr/N, 2sr/N, ...]。但**为什么**第 k 个 FFT 系数 X[k] 就被说成对应这个频率呢？这是 FFT 的定义还是有数学推导？

**卡在的具体位置**："第 k 个 bin 对应的频率是…freqs[k] = k * sr / N"

**希望的补充**：简单解释一下离散傅里叶变换的定义——N 个时间采样点映射成 N 个频域点，频轴均匀分成 N 份，从 0 到 sr（近似）。这个对应关系是DFT定义的一部分，不是凭空来的。

---

### 【理解】dB 谱的系数为什么是 20 而不是 10？—— Cell 10

代码写的是 `D[k] = 20 * log10(|X[k]| + eps)`，讲解说"dB 把乘法变成加法"。我看到了后面说功率谱 P[k] = |X[k]|²，但没有明确说明：**为什么恰好是 20？**

**卡在的具体位置**："dB 谱 D[k] = 20 * log10(|X[k]| + eps)" — 没有解释这个 20

**希望的补充**：写清楚一条推导：如果功率 P = |X|²，那么 10·log₁₀(P) = 10·log₁₀(|X|²) = 20·log₁₀(|X|)。所以 20 是从功率的对数定义来的，不是随意选的。

---

### 【理解】相位谱在这里的作用是什么？—— Cell 13-16

代码计算了 `φ[k] = angle(X[k])`，验证了反对称性 φ[k] ≈ -φ[N-k]。但对于一个实信号（比如 440Hz + 880Hz混合），这些相位值通常是多少？用来做什么？

**卡在的具体位置**："对于单频正弦 sin(2π f₀ t + θ₀)，FFT 在 f₀ 对应 bin 处的相位就是 θ₀" — 这个说法很抽象

**希望的补充**：给个具体数值例子——比如 sin(2π·440t + π/6)，其 FFT 的相位应该是 π/6。或者说，相位信息对"时间拉伸（time-stretching）"这类操作为什么重要。现在只知道有这么个东西，不知道怎么用。

---

### 【计算】为什么要 round(440/31.25)？小数 bin 是什么意思？—— Cell 4

题目说"440 Hz 钢琴 A4 音：落在第 k = round(440/31.25) = 14 号桶"。我算出来 440÷31.25 = 14.08，round 到 14。但如果不 round，14.08 这个小数 bin 位置是什么意思？精度偏差？

**卡在的具体位置**："k = round(440/31.25) = 14"

**希望的补充**：解释一下频率分辨率的物理意义——当 Δf = 31.25 Hz/bin 时，440 Hz 的信号最接近第 14 号 bin（对应 437.5 Hz），但理论上可能偏移 ±15.625 Hz；若要定位得更精准，必须增大 N 来减小 Δf。这样能理解为什么 N 大很重要。
