# 小雨 — L16 学习日志

## 我理解了什么
- **2×2 行列式公式** `det = ad - bc` 很直观，是四个元素的交叉乘积之差
- **奇异矩阵的概念**：两行（列）成比例 → det=0 → 不可逆，这个逻辑清楚了
- **基本思路**：det≠0 → 变换可逆，det=0 → 信息丢失（被"压扁"了）

## 遇到的困难和问题

### 【理解】逆矩阵公式的来源是什么

课程直接给出 $A^{-1} = \frac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$，但**完全没有解释为什么是这样**。为什么主对角线互换、副对角线变号？这个公式是怎么推导出来的？我高中没学过，也想不出来。希望课程能补充：要么给一个简洁的推导，要么用矩阵乘法验证一次（比如 $A \cdot A^{-1} = I$，展开计算让学生看到为什么是这个形式）。

### 【理解】伴随矩阵（adjoint matrix）没有被定义清楚

课程在 3×3 部分说用"伴随矩阵公式" $A^{-1} = \text{adj}(A) / |A|$，但从来没有明确说：
- **伴随矩阵是什么**？怎么构造？
- **补充例题中的** $\begin{pmatrix}5&-3&1\\-3&3&-1\\1&-1&1\end{pmatrix}$ 是怎么来的？
- **代数余子式（cofactor）** 和 **子式（minor）** 有什么区别？

我看了笔记里提到"划掉第 i 行第 j 列"，但伴随矩阵中的元素和这些子式有什么关系？需要课程补充一个清晰的定义和小例子。

### 【推演】行列式 ad-bc 这个公式从哪里来的

课程说这是"两列向量张成平行四边形的有向面积"，但没有**推导过程**。为什么平行四边形面积等于 ad-bc？用高中的向量叉积还是行列式定义？我想理解这个，但课程跳过了。

### 【推演】"划掉行列"在代码中怎么实现

课程的补充例题代码写：
```python
M11 = np.linalg.det(A[1:,[1,2]])   # 划掉第0行第0列
```

这里 `A[1:,[1,2]]` 我看不懂：
- `[1:]` 是从第1行开始（所以划掉了第0行），我理解
- `[1,2]` 是什么意思？为什么不是 `:,[1,2]` 这样？这是 NumPy 的什么语法？

希望课程给一个简短的"NumPy 索引速查"或在这里加一句注释说明。

### 【理解】符号混乱：M_ij、C_ij、代数余子式

笔记里混用了：
- $M_{ij}$ = 子式（第 i 行第 j 列的 2×2 行列式）
- 代数余子式 = 子式 × (-1)^(i+j)？

但这两个在余子式展开公式中的地位不一样。3×3 补充例题用的是 $|A| = a_{11}M_{11} - a_{12}M_{12} + a_{13}M_{13}$，这里的"−"是符号，还是代数余子式的 (-1)^(i+j) 已经被吸收了？没有清楚说。

### 【实践】白板挑战题 4 的"翻转"没有可视化

问 4 说：如果 det(A) = −5，几何含义是"面积放大 5 倍 + 翻转"。

**我不理解"翻转"指的是什么**。二维平面怎么翻转？是镜像对称吗？为什么 det<0 就是翻转？课程应该补充一个 2×2 矩阵的例子（比如反射矩阵），画个图说明 det<0 时坐标系的方向反了。目前这个解释太抽象了。
