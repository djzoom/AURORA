# 老陈 — L16 学习日志

## 我理解了什么

行列式测量矩阵变换对空间的"缩放程度和方向"：det(A)≠0表示变换可逆，能从结果还原回原始状态；det(A)=0表示矩阵把空间压扁了维度（比如3D压成2D），信息丢失不可逆。用2×2公式ad−bc能手算，3×3可以通过划掉行列后递归用更小的行列式算（余子式展开）。

## 遇到的困难和问题

### 1. 【理解】2×2逆矩阵公式为什么是那样？
课程说了结论："主对角线互换，副对角线变号，整体除以 det"，然后给了公式。但这个公式是怎么推导出来的？是从某个矩阵方程解出来的吗？比如，为什么不是其他形式？希望课程补充：用矩阵乘法验证 A·A⁻¹=I，展开后看看每一项是怎么消掉的，反推出为什么逆矩阵必须长那样。

### 2. 【理解】有向面积里的"翻转"指的是什么？
课程说"det<0：翻转（左手变右手）"。这个翻转的定义有点模糊。比如旋转45°的矩阵，它也改变了向量的方向，但det>0。翻转和旋转的区别在哪儿？我猜翻转是指某种镜像变换（像照镜子），但课程没有具体例子。希望补充：给一个det<0的具体矩阵（比如 [[1,0],[0,-1]] 做y轴反射），图示它如何把平行四边形"翻过来"。

### 3. 【推演】为什么必须按第一行展开？其他行不行？
课程标题是"按第一行**余子式展开**"，并用第一行做了例题。但我想知道：能不能按第二行展开？按第三行展开？答案会一样吗？行列式有某种"对偶性"吗？希望课程补充：用同一个3×3矩阵，分别按三行展开演示一遍，让我看到答案都是一样的（或解释为什么数学上这是等价的）。

### 4. 【理解】伴随矩阵具体怎么构造？
单元4说"用伴随矩阵 adj(M) / det_3x3(M) 实现"逆矩阵。什么是伴随矩阵？它是从代数余子式排成的某个矩阵吗？行列式、子式(Minor)、代数余子式(Cofactor)、伴随矩阵(Adjoint)这些概念怎么一层一层嵌套的？希望补充一个对照表：
- Minor：划掉某行列后的行列式
- Cofactor：Minor乘以±1（符号规则是什么？）
- Adjoint：把所有Cofactor排成矩阵（怎么排？转置吗？）

### 5. 【实践】实现 inv_2x2 时，我应该怎么测试"det=0返回None"这个分支？
代码框架里有测试用例 `singular = [[1,2],[2,4]]`，但我自己写的时候怎么想到用这个例子？有什么规律吗？比如两行成比例就一定det=0？希望补充：给初学者一个"生成奇异矩阵"的方法论，比如"任意向量v，矩阵[[v], [k*v]]（两行倍数）一定奇异"。

**实学时长心得**：这一课的几何直觉（"det=0压扁了空间"）很清晰，公式背诵也容易。难点是课程跳过了推导细节——比如2×2逆矩阵公式是凭空出现的、伴随矩阵的构造也没说清。我需要的不是更多例题，而是多一层"为什么这样"的解释。
