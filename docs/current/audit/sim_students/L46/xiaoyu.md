# 小雨 — L46 学习日志

## 我理解了什么

人耳不是线性的，而是按倍数（对数）感知音高——100Hz 和 110Hz 我能听出差别，但 3000Hz 和 3010Hz 我听不出。Mel 频率尺度就是一个公式，把这种"倍数感知"拉直成等距。这样的话，低频段会被分得很细（滤波器窄又密），高频段被分得粗（滤波器宽又疏），正好符合耳蜗的物理结构。钢琴键的等宽排布对应频率翻倍，Mel 尺度让听感也变成等距，很聪明。
