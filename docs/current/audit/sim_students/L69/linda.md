# 琳达 — L69 学习日志

## 我理解了什么

CTC的核心问题是：目标序列对应指数多条路径（3^T条），无法逐一枚举。前向算法通过动态规划把它变成O(T·S)复杂度——用"前向变量"α[t][s]递推"在第t帧以位置s结束的所有合法路径概率和"，在扩展标签l'=[∅,a,∅,b,∅]上逐帧逐列递推。最后两个位置加一起就是答案。代码框架我懂（初始化、递推、求最后值），但细节卡住了。
