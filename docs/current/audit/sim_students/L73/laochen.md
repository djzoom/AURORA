# 老陈 — L73 学习日志

## 我理解了什么

WER 公式 WER = (S+D+I)/N，把"词错了"分成三类：替换（说成别的词）、删除（漏说了）、插入（多说了）。每类问题对应不同的修复方向——替换多说明数据不够多样，删除多说明信号弱，插入多说明模型"幻觉"了。按领域分别计算这三个比率，能定位模型的具体薄弱点。

```python
elif i > 0 and j > 0 and dp[i][j] == dp[i-1][j-1] + 1:  # 替换
```
