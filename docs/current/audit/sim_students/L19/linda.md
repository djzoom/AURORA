# 琳达 — L19 学习日志

## 我理解了什么（简要）

1. **矩阵×向量就是"列的线性组合"**：`A @ v = v[0]*A[:,0] + v[1]*A[:,1]` 这个拆解让我知道矩阵乘法其实在做什么了——它不是什么神秘公式，就是用向量的每个分量作系数去加权矩阵的各列。

2. **矩阵变换能通过三个特征来分类**：正交性（检查 `M @ Mᵀ ≈ I`）、行列式的符号（det = +1/-1/其他）、是否对角，用这三个条件就能把旋转、反射、缩放、剪切区分开，代码验证都通过了。

3. **矩阵×矩阵就是叠加多个秩1矩阵**：这个我能理解逻辑（每个"列×行"构成一个秩1块），虽然还不完全清楚为什么秩1分解这么重要。

### 【理解】正交矩阵为什么保长度

**位置**：第3、4节用 `R @ Rᵀ = I` 定义旋转矩阵（正交矩阵），说它保距（`||Rx|| = ||x||`）  
**具体问题**：高中数学没有学过矩阵的共轭转置、范数的正式定义。课程直接给出条件 `R @ Rᵀ = I` 就宣称它保距，但为什么？数学上怎么推导？给我看不懂的"代码断言"（assertions）不算解释——我需要知道 WHY。  
**期望补充**：加上一句几何直觉或简单的数学推导。比如"正交矩阵保距的原因：向量长度 `||x||² = x · x`，如果 `R @ Rᵀ = I` 那么 `||Rx||² = (Rx) · (Rx) = x · (Rᵀ R) · x = x · x = ||x||²`"（虽然我还要花时间理解点积的这个性质，但至少告诉我有一个推导路径）。

### 【理解】行列式的几何意义没有铺垫

**位置**：第4、5节大量使用 `np.linalg.det(A)` 来判断是旋转（+1）还是反射（−1）还是其他  
**具体问题**：我记得L16课讲了行列式的定义，但那一课的重点是"矩阵可逆当且仅当det≠0"，没有说det的符号有什么几何意思。为什么旋转必须det=+1？为什么反射det=-1？det的绝对值的大小代表什么（比如缩放矩阵 det=6.0，这个6来自哪里）？  
**期望补充**：在第4节或参数实验部分，加一句"det的几何意义"：det的绝对值是矩阵如何缩放面积，det的符号告诉你是否翻转（反转坐标系的定向）。旋转不翻转，所以det>0；反射翻转，所以det<0。
