# 老陈 — L19 学习日志

## 我理解了什么

1. **矩阵×向量有两种计算法**：一种逐行算点积（数值角度），一种看作矩阵各列的系数线性组合（几何角度）。两种方法结果完全一样，但"列的线性组合"这个观点能直观看出向量被变换到哪里。

2. **旋转/反射/缩放/剪切四种变换有不同的特征**：旋转满足 R@R.T=I 且 det=+1；反射也满足正交但 det=-1；缩放是对角矩阵；其他是剪切。

3. **矩阵×矩阵可以拆成秩1矩阵的和**：虽然还不完全明白"秩1"的深层含义，但知道这个分解形式存在。
