# 小雨 — L71 学习日志

## 我理解了什么

学到了 Whisper 如何逐词生成文字的两种方式：**贪婪解码**（每步选最有把握的词，快但容易卡住）和 **Beam Search**（同时思考多条路线，选累计最优的，更准但慢）。就像我写论文时，要么一直往一个方向写（贪婪），要么先想好3条思路、比较后再选最强的（beam）。还学到了用对数概率相加而不是概率相乘，能避免数字太小造成电脑算不出来（下溢）。
