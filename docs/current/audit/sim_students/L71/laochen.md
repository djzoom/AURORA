# 老陈 — L71 学习日志

## 我理解了什么（简要，3 行以内）

- **自回归解码**：语言模型不是一次吐出整句话，而是像说话一样一个词一个词地选。每次选的时候都看前面已经选好的词。
- **贪婪 vs Beam 的权衡**：贪婪每步选最可能的词，但可能走进"局部最优"陷阱。Beam Search 同时考虑多条路线，到最后选最长累计概率最高的那条。
