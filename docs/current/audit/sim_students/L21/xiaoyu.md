# 小雨 — L21 学习日志

## 我理解了什么

矩阵和向量的乘法本质是"行向量 · 列向量 = 点积"。FFT 和 Mel 滤波都不是什么黑盒，其实就是用特定系数构造的矩阵和信号/功率谱相乘。理解了矩阵形状：DFT 矩阵 (N,N) 乘信号 (N,) 得到频域 (N,)；Mel 矩阵 (n_mels, n_freq) 乘功率谱 (n_freq,) 得到梅尔特征 (n_mels,)。

## 遇到的困难和问题

### 【理解】DFT 矩阵公式的含义

课程说 "W[k,n] = e^{-2πi·kn/N}" 是 DFT 矩阵的定义，但没有任何解释这个公式从哪来、为什么要用复数指数函数。为什么不是其他函数？为什么是负号？为什么 2π？2π 和频率、圆有什么关系？对文科生来说，看到这堆符号就很害怕。希望课程能加一段：用类比或故事解释为什么要选这个特定的公式。

### 【理解】复数矩阵和共轭转置 Wᴴ

验证 DFT 矩阵"正交性"时，课程说"W·Wᴴ = N·I"。我不知道 Wᴴ 是什么——是共轭？是转置？都是？代码里怎么算？`W.conj().T` 就是吗？对复数矩阵来说，为什么要用共轭转置而不是普通转置？这个性质有什么用处？

### 【理解】酉矩阵和正交矩阵

课程提到"酉矩阵"和"(1/√N)·W 是正交矩阵"，但没定义"酉矩阵"是什么，为什么要分两个概念。是不是说：复数矩阵叫"酉"，实数矩阵叫"正交"？它们有什么区别和联系？

### 【推演】欧拉公式 e^{iθ} = cos(θ) + i·sin(θ)

计算 W[2,3] = exp(-3πi/2) 时，我试图用欧拉公式 e^{iθ} = cos(θ) + i·sin(θ)，但这个公式课程从没讲过，高中数学也没有。我从验证代码里"猜"出来的。希望能在某处（比如前面的复数章节）完整推导或至少正式介绍这个公式，否则我无法手算这类问题。

白板问题要求计算 cos(-3π/2) 和 sin(-3π/2)。我知道 -3π/2 弧度制，但要把它转换成角度（-270°），再想象单位圆上那个点…这太复杂了。课程有没有一个快查表或图形，说明常见角度（0, π/6, π/4, π/3, π/2, π, 3π/2, 2π）对应的 cos 和 sin 值？
