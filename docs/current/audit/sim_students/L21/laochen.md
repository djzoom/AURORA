# 老陈 — L21 学习日志

## 我理解了什么

看懂了一件事：**FFT 和 mel 特征都是矩阵乘法，没有黑盒**。FFT 就是把信号向量乘以一个 DFT 矩阵 `W`（每行都是复数权重），mel 特征就是把功率谱向量乘以一个滤波器矩阵 `M`（每行是三角形）。两个操作本质上都是**一堆点积的堆叠**。用 `np.outer` 能一行生成 DFT 矩阵，用 `np.linspace` 的端点法能生成 mel 矩阵。

## 遇到的困难和问题

### 【理解】DFT 公式为什么是负号？

课程说 `W[k,n] = exp(-2πi·kn/N)` 就这么写，但**没有解释为什么是负号**而不是正号？为什么一定要分子是 `-2π`？为什么分母是 `N`？之前 L06 学欧拉公式是在描述旋转，现在这里负号是什么物理或数学意义？是定义规范问题，还是有更深的原因？

### 【计算】exp(-3πi/2) 怎么算出来 = j？

白板挑战第1题说 `W[2,3] = exp(-2πi·6/8) = exp(-3πi/2)`，然后这等于 `j`（虚数单位）。我能看出 `-2π·6/8 = -3π/2`，但**怎么从 exp(-3πi/2) 直接得出结果是 j？** 欧拉公式 `e^(iθ) = cos(θ) + i·sin(θ)`，那 `exp(-3πi/2) = cos(-3π/2) + i·sin(-3π/2)` = ... = `0 + i·1`？我需要一步步验证 cos(-3π/2) 和 sin(-3π/2) 分别是多少，这里跳得太快了。

### 【理解】为什么 mel 矩阵要用 N_mels+2 个端点？

Cell-8 说"修复：使用 N_mels+2 个端点"。为什么不是 N_mels 个或 N_mels+1 个？逻辑是用 `centers[m], centers[m+1], centers[m+2]` 这三个端点来定义第 m 个滤波器的上升段和下降段。那如果用 N_mels 个端点，就没有 `centers[m+2]` 了，所以必须多两个？**为什么不能用其他办法（比如循环索引或边界扩展）？** 为什么这个"修复"是对的？

### 【理解】Mel 滤波器重叠程度如何看？

课程说"减少滤波器数量会使相邻频率 bin 被合并进同一个三角窗，高频细节分辨率下降"。我从参数实验代码里看到 `n_mels` 从 40 改到 128，矩阵行数会改变。但**怎样"观察滤波器重叠程度"？** 重叠程度体现在哪个指标上？是看非零区间的宽度？还是看相邻滤波器峰值间的距离？没有给出量化定义或可视化。

### 【实践】build_dft_matrix 的 np.outer 为什么就对？

题目提示"用 `np.outer` 一行构造"。我试了 `np.outer(n, n)` 生成一个 N×N 矩阵，每个 [i,j] 位置的值是 `i·j`，然后乘以 `-2πi/N` 再 `exp` 就得到 DFT 矩阵。**但我没有从原理上理解为什么 np.outer 的结果恰好是 W[k,n] 中的 k·n 部分。** np.outer 是什么数学概念？为什么两个向量外积的 [i,j] 位置就是 i·j？

### 【理解】DCT 部分的简化太大。

Cell-13 说"简化版，跳过 Mel；只取独立 bin"。我看到代码里：
- 构造了一个 `D` 矩阵（4 行，对应前 4 个 DCT 系数）
- 然后 `D @ spectrum[:N//2+1]` 得到 mfcc4

**但完整的 MFCC 管道应该是：信号 → DFT → 功率谱 → Mel 滤波 → log → DCT**，这里跳了 mel 和 log，直接在原始功率谱上做 DCT。** 这样做出来的 DCT 输出和真实 MFCC 还能比较吗？为什么要跳过？** 没有这些步骤，学生学到的只是"有这么一个矩阵"，没有看到完整链条。

### 【实践】build_simple_mel_matrix 参数不明确。

题目只说"参考 cell-8 的修复版逻辑"，但用的是线性刻度还是 mel 刻度？题目名叫 "build_simple_mel_matrix"，"simple" 是什么意思？** 是简化了频率刻度（用线性而不是 mel 刻度），还是简化了滤波器形状？** 没有明确说明函数的预期输入、输出、和与 `mel_filterbank()` 的关系。

### 【理解】酉矩阵性质 W·Wᴴ = N·I。

DFT 矩阵是"酉矩阵"，满足 `W·Wᴴ/N = I`（即 `W·Wᴴ = N·I`）。课程说这意味着 `(1/√N)·W` 是正交矩阵，但**为什么要特别关心这个性质？** 这个性质对 FFT 计算有什么直接的用处？是用来做快速计算，还是用来验证可逆性，还是别的？为什么要共轭转置而不只是转置？
