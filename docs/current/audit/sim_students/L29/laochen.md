# 老陈 — L29 学习日志

## 我理解了什么

这课讲了三种常用的概率分布：**均匀分布**（区间内处处等可能）、**正态分布**（中间多两边少，神经网络权重就这么初始化）、**伯努利分布**（硬币一样的0/1分布）。我学会了写一个 `gaussian_pdf` 函数来计算正态分布在某个点的"概率密度"，还知道了68-95-99.7这个规则——在正态分布里，±1个标准差包含68%的数据，±2个标准差包含95%。

## 遇到的困难和问题

### 【理解】正态PDF公式中，√2π 和 exp 是怎么来的？

课程直接给了公式 `f(x;μ,σ) = 1/(σ√2π) * exp(-(x-μ)²/(2σ²))`，但没说这个公式为什么是这个样子。课程提到"中心极限定理"推导出正态分布，但没有给任何推导过程。我高中数学里从没见过这样的公式，√2π 为什么是1/√2π 而不是别的数字？exp() 为什么要这样放指数？

**希望补充**：至少给个简化版的推导，或者说明这是从哪个数学定理推出来的关键步骤。

---

### 【理解】"概率密度"和"概率"到底差在哪？为什么PDF值可以 > 1？

课程说 `f(x)` 是"概率密度"，但我一开始理解不了这和"概率"有什么区别。标准正态分布 x=0 时 PDF ≈ 0.4，这是一个"密度值"，但怎么才能从它变成实际的概率？课程提到"PDF 在 ±∞ 上积分为 1"，但没解释为什么非要积分才能得到"概率"。如果我想知道 P(-0.5 < X < 0.5) 是多少，光看PDF值看不出来。

**希望补充**：用一个直观例子说明PDF和概率的区别。比如用"人口密度"做类比——密度本身不是人数，要乘以面积才是人数？

---

### 【理解】为什么σ增大，峰值就减小，但"面积始终为1"能保证？

课程实验中看到 σ=1 时峰值≈0.4，σ=2 时峰值≈0.2（减半了）。但为什么"曲线下面积始终为1"？这是怎么数学上保证的？分子里有 `1/σ`（当σ增大时分子变小），分母里也有σ（当σ增大时分母变大），这两个σ 是不是故意设计得能互相抵消，才保证了积分=1？还是只是巧合？

**希望补充**：用一个简单的公式或推导证明，为什么这个特定的归一化系数 `1/(σ√2π)` 能确保积分=1。

### 【推演】正态分布"没有闭合形式的CDF"是什么意思？

代码注释说"用数值积分近似"，暗示CDF不能写成一个"公式"。均匀分布的CDF 看起来很简单：F(x) = (x-a)/(b-a)。为什么正态分布的CDF 不能这样简单地写出来？PDF 是可以写公式的，为什么CDF 不行？

**希望补充**：解释什么是"闭合形式"，为什么正态分布特别，为什么要靠数值方法。

### 【理解】伯努利分布的PMF公式 P(X=k) = p^k · (1-p)^(1-k) 怎么理解？

这个公式比较巧妙，当 k=1 时变成 `p^1 · (1-p)^0 = p`，当 k=0 时变成 `p^0 · (1-p)^1 = 1-p`。我能看出这个"技巧"，但为什么这个公式是"对的"？为什么不是别的形式？

**希望补充**：直观解释为什么伯努利分布要写成这个形式，而不是直接写成两个分支 P(X=0)=1-p, P(X=1)=p。
