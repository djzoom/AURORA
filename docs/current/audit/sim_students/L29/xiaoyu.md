# 小雨 — L29 学习日志

## 我理解了什么

**1. 概率分布就是描述"随机事件的规律"的工具。** 就像天气预报说"明天有 80% 概率下雨"，概率分布是告诉我们某个数值出现的"可能性有多大"。正态分布（钟形曲线）中间的值最常见，两边很罕见——像我的考试成绩，大多数人在平均分附近，天才和不及格都少。

**2. 正态分布有两个旋钮（μ 和 σ），改变它们能调整曲线的位置和宽度。** μ 决定曲线在哪（向左或向右移），σ 决定曲线的"胖瘦"——σ 越大越矮越宽，σ 越小越尖越窄。

**3. "68-95-99.7 规则"很实用：大约 68% 的数据在中心±1 倍标准差内，95% 在±2 倍内，99.7% 在±3 倍内。** 这意味着如果知道平均分和标准差，就能估计有多少学生在什么分数区间。

### 【推演】gaussian_pdf 实现中，指数项为什么这样写？

实现提示是 `np.exp(-((x-mu)/sigma)**2 / 2) / (sigma * np.sqrt(2*np.pi))`。

对比公式 $\exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$，我发现提示里写的是 `-((x-mu)/sigma)**2 / 2`，而公式是 $-\frac{(x-\mu)^2}{2\sigma^2}$。这两个数学上相等吗？

$$-\frac{(x-\mu)^2}{2\sigma^2} \stackrel{?}{=} -\frac{1}{2} \left(\frac{x-\mu}{\sigma}\right)^2$$

让我验证一下：右边是 $-\frac{1}{2} \cdot \frac{(x-\mu)^2}{\sigma^2} = -\frac{(x-\mu)^2}{2\sigma^2}$。确实相等。但课程为什么提示用前一种形式？是为了代码简洁吗？还是有数值计算的原因？能否解释一下这个"缩放再平方"的技巧？
