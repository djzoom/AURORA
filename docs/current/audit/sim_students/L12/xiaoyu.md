# 小雨 — L12 学习日志

## 我理解了什么

**矩阵不是数字表格，是"坐标变换机器"**——把一个向量输进去，按规则改变它的方向和大小。矩阵乘向量的计算方式很清晰：**逐行点积**，即输出向量的每一个数 = 矩阵的那一行与输入向量的点积。**矩阵乘矩阵可以分解成：对右边矩阵的每一列分别调用一次矩阵×向量，再把列拼成新矩阵**——这就是函数复合的含义。

Cell 中说"等价于 Σⱼ A[i,j]·x[j]，**无需写内层循环**"。对，我不需要写内层循环（因为用 `np.dot`），但这行文字把"求和符号"和"点积"混淆了。一个高中毕业生看到求和符号会本能紧张，建议改成："等价于把矩阵第 i 行的每个元素与向量 x 对应位置相乘，再全部加起来，也就是点积"——这样更平易近人。
