# 琳达 — L12 学习日志

## 我理解了什么
1. **矩阵×向量 = 逐行点积**：矩阵的每一行与输入向量做点积，得到输出向量的对应元素（m×n 矩阵作用于 n维向量得到 m维向量）。
2. **矩阵×矩阵 = 对B的每列做matvec**：用A作用于B的每一列，再拼接结果，就得到A@B。
3. **矩阵是坐标变换机器**：对角矩阵拉伸、旋转矩阵转动、全1矩阵求和——都能通过相同的"点积"框架统一理解。

### 【理解】函数复合的含义和符号
notebook 第5部分演示 `(A @ B) @ x == A @ (B @ x)`，代码验证了相等，但"函数复合"这个概念我还是有点抽象。为什么矩阵乘法叫复合？是把两个变换打包成一个吗？如果先算 `B @ x` 再算 `A @(B@x)` 和先算 `A @ B` 再乘 `x`，计算量有什么区别吗？

💡 建议补充：用"管道"或"黑盒"的类比：矩阵是一个黑盒变换器，`B @ x` 是信号通过B，`A @ (B @ x)` 是先通过B再通过A。`A @ B` 就是把两个黑盒焊成一个大黑盒。这样新的大黑盒等价于什么都不算、直接用预先焊好的矩阵。
