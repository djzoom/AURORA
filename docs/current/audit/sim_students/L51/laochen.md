# 老陈 — L51 学习日志

## 我理解了什么

学完了一个 WAV 文件处理的完整流水线：读音频 → 切帧做 STFT → 压到 Mel 频域 → 再过 DCT 降维到 13 个 MFCC 系数。用四个图层看同一段语音：时域波形能看到"啊"的能量包络，频谱图能看到元音的等间距亮纹（谐波），Mel 图能看到共振峰，MFCC 最紧凑。还学了改参数（n_fft、hop、n_mels）对图的影响。

## 遇到的困难和问题

### 【理解】采样率与奈奎斯特频率的 1:2 关系  
**卡在哪里**："表格里 sr=16000 Hz → Nyquist=8000 Hz"。为什么一定要除以 2？  
**具体困惑**：课程没解释这个 1:2 的关系为什么成立。这是采样定理吗？能不能用更浅白的话说：为什么采样率翻倍才能捕住某个频率？  
**希望补充**：一个简单的例子（比如用 10 Hz 采样率采 5 Hz 的正弦波）能"看"到什么，再升到 10 Hz 采样率能"看"到什么，直观解释奈奎斯特。

### 【推演】中心化 padding 对帧数公式的推导  
**卡在哪里**："aurora stft 默认 center=True（两端各反射填充 n_fft//2），于是 n_frames = 1 + N // hop"。  
**具体困惑**：这个 `1 + ...` 公式怎么推出来的？两端各填充 n_fft//2 后，总采样点变成了 N + n_fft，那 n_frames 应该是多少？为什么不是 `(N + n_fft) // hop` 而是 `1 + N // hop`？  
**希望补充**：画个 timeline：原信号有 N 个点，padding 后长什么样，每一步能切几帧，为什么最后是这个数字。

### 【理解】频率分辨率公式 `sr / n_fft` 的含义  
**卡在哪里**：表格说"频率分辨率：sr/n_fft：31 Hz → 7.8 Hz"（当 n_fft 从 512 → 2048）。  
**具体困惑**：这个"分辨率"是什么意思？16000 ÷ 1024 = 15.6 Hz，这 15.6 Hz 代表什么？是说我能区分的最小频差是 15.6 Hz 吗？为什么 FFT 输出的频率间隔恰好是 sr/n_fft？  
**希望补充**：解释 FFT bin 的定义（为什么第 k 个 bin 对应的频率是 k·sr/n_fft），以及"频率分辨率"为什么等于相邻 bin 的频差。

### 【推演】元音合成中的谐波权重  
**卡在哪里**：元音 /a/ 用 `(1.0 / k)` 加权，元音 /i/ 用自定义的 `[1.0, 0.4, 0.8, ...]`。  
**具体困惑**：为什么 /a/ 是 1/k（第 k 个谐波权重 = 1/k）？这个 1/k 是从哪来的？/i/ 为什么权重完全不同？这背后是否有物理规律（比如声道共鸣）？如果我要自己造一个元音，怎么设这些权重？  
**希望补充**：讲一下人类语音的物理模型（声带产生基频 + 多个谐波，声道共鸣器选择某些谐波放大），然后解释这两个权重向量为什么长这样。

### 【理解】DCT 和 MFCC 系数 C0 的关系  
**卡在哪里**：白板挑战问 5 说"C0 = sum(log Mel energy) / √N"，但课程没讲 DCT 是什么。  
**具体困惑**：MFCC 是怎么从 Mel 能量变过来的？为什么要过 DCT？DCT 输出的 13 个系数分别代表什么？特别是 C0 为什么是总能量？（高中没学过 DCT，只学过 DFT 的基础概念，如果有的话）  
**希望补充**：一个很简单的例子（比如 3 个数 [1, 2, 1] 过 DCT 得到什么），展示 C0 就是把它们加起来并标准化。然后再讲为什么 ASR 中常常丢掉 C0（因为它只代表响度，与音素类别无关）。
