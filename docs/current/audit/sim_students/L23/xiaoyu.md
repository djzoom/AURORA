# 小雨 — L23 学习日志

## 我理解了什么

1. **偏导数**（∂f/∂x）是"固定其他变量，只改一个变量"时的变化率，可以用中心差分公式 `(f(x+h) - f(x-h))/(2h)` 数值计算。

2. **梯度**是把所有偏导数拼成一个向量，比如 f(x,y) 在某点的梯度 [∂f/∂x, ∂f/∂y]。

3. **实现思路**：梯度函数对每个维度单独做一次中心差分，就像在高维空间中依次戳戳 x、y、z 方向，记录函数的反应。

## 遇到的困难和问题

### 【理解】符号 ∂ 是什么？为什么不写 d？
题目写 ∂f/∂x，但从没解释这个符号叫什么。我只学过 f'(x)，看到 ∂ 有点陌生。高中数学里没有这个符号，突然出现让我有点紧张。希望课程能说明：∂ 叫"偏导符号"，表示只改一个变量；而 d 是全导数。

### 【理解】∇ 这个倒三角是什么符号？
课程用 ∇f 表示梯度向量，但没说这个符号叫什么。我猜是"倒三角"吗？发音是什么？对数学不自信的学生看到陌生符号容易产生心理障碍。

### 【理解】"等高线"和"法向量"是什么？
题目说"梯度垂直于等高线切线"、"梯度 = 等高线法向量"，但我没学过这俩概念。我知道等高线出现在地形图上，但没有视觉演示，不清楚数学意义。"法向量"更是第一次听。课程需要配一张图或类比：为什么圆的法向量是径向？

### 【推演】f(x,y) = x² + y² 的等高线为什么是同心圆？
题目说"等高线是以原点为圆心的同心圆"，但没推导。我想知道从哪里看出来的？是不是因为 x²+y² = c（常数）这个方程的图像是圆？但为什么等于同一个 c 就叫"等高"？

### 【计算】"点 (1,2) 处的等高线切线方向是 (-2,1)"——这怎么算？
对答案格中用切线 (-4, 3) 作为"垂直于梯度 (6,8) 的方向"。我能用点积验证：(-4)×6 + 3×8 = 0，确实垂直。但这个切线方向 (-2, 1) 怎么来的？是否有公式？还是只能死记硬背？

### 【理解】代码中 `np.zeros_like()` 和 `dtype=float` 的作用
题目写 `point = np.asarray(point, dtype=float)`，我不确定 dtype=float 的必要性。如果 point 已经是数组，为什么还要转换？转换成 float 是为了精度吗？还是因为后面要做除法？

### 【理解】什么是"单位扰动向量"？
题目说"构造单位向量 e"，然后 e[i] = 1.0。但为什么叫"单位"？我学的单位向量是长度为 1 的向量，这里 e = [1, 0] 的长度确实是 1，但这个命名让我疑惑——是不是有更清楚的说法，比如"只有第 i 位为 1、其他位为 0 的基向量"？

### 【计算】为什么数值梯度在 h=1e-5 时能这么精确？
代码中用 h=1e-5，结果误差在 1e-9 以内。但这个 h 怎么选？太大会怎样？太小会怎样？有没有通用的选择方法？
