# 琳达 — L23 学习日志

## 我理解了什么
- **偏导数**就是"固定其他变量，只沿一个方向改变"的导数，用中心差分 `(f(p+hᵢ) - f(p-hᵢ))/(2h)` 可以数值计算。
- **梯度**是所有偏导数拼在一起的向量，比如 f(x,y)=x²+y² 在 (3,4) 处的梯度是 [6, 8]，对应 ∂f/∂x=6 和 ∂f/∂y=8。
- **实现方法**：对每个维度构造单位扰动向量，用中心差分逐分量计算，就能得到梯度。

## 遇到的困难和问题

### 【理解】多变量求偏导的合法性
> 原文："把 y 视为常数，对 x 求导"

具体在哪卡住：课程说对 f(x,y)=x²+y² 求 ∂f/∂x 时，"把 y 视为常数"。这我能跟着算出答案 2x，但我想知道：**为什么可以把 y 当常数处理？** 这和一元函数求导有什么区别？为什么这样做就叫"偏"导？高中数学没有讲过多元函数，课程也没有从极限定义严格推导为什么这个操作合法。

希望课程补充什么：最好加一个"为什么偏导数可以这样算"的短段落，比如从 f(x,y) 在点 (x₀,y₀) 附近用泰勒展开，展示出"固定 y=y₀，只让 x 变化"时，函数的行为就像一元函数，所以一元求导法则就适用。


### 【理解】"梯度指向最陡上坡"的直观理由
> 原文："梯度向量指向**函数值增加最快**的方向"

具体在哪卡住：代码验证了"梯度⊥等高线"这个事实（两个向量点积接近 0），但我没想明白为什么垂直于等高线就意味着"最陡"。直观上，沿着等高线走，函数值不变（废话）；那沿着垂直方向走确实会变化，但为什么垂直方向比其他所有非等高线方向都陡？数学上怎么证明？

希望课程补充什么：加个几何图解（或用 contour plot 可视化），展示梯度向量和等高线的位置关系，以及为什么"垂直=最陡"。或者从"方向导数"的角度，说明"沿单位向量 u 的方向导数 = 梯度·u"，最大值在 u 与梯度同向时取到。


### 【推演】从偏导数定义推出计算规则  
> 原文：∂f/∂x = 2x（对 f(x,y)=x²+y²）

具体在哪卡住：课程直接给出了答案，但没有展示推导步骤。比如，对于 f(x,y)=x²+y²，我该怎么用极限或中心差分的定义从头推出 ∂f/∂x=2x？虽然我能从"把 y 当常数，对 x 用求导法则"猜出答案，但这不算真的理解推导过程。

希望课程补充什么：对第一个例子 f(x,y)=x²+y²，显式写出 ∂f/∂x = lim_{h→0} [(x+h)²+y² - (x-h)²-y²]/(2h) = ... = 2x 这样的完整步骤。


### 【概念】"单位扰动向量"的几何含义
> 代码：e = np.zeros_like(point); e[i] = 1.0

具体在哪卡住：我理解了这段代码的作用——在第 i 维加上 h，其他维度不变。但当 point 是 3 维或更高维时，这个"单位向量"的几何直观是什么？为什么选择这个方向做扰动能反映"沿 xᵢ 方向的变化"？这个直观很模糊。

希望课程补充什么：一个图例说明"在 3D 坐标中，沿 x 轴、y 轴、z 轴的三个单位向量分别是什么"，然后再展示"在点 p 附近，沿第 i 个单位向量方向改变 h 距离，函数值怎么变"。


### 【计算】中心差分公式中除以 2h 的原因
> 公式：∂f/∂xᵢ ≈ (f(point + h·eᵢ) - f(point - h·eᵢ)) / (2h)

具体在哪卡住：为什么分母是 2h？直观上，我从 x-h 走到 x+h，距离是 2h，所以"斜率 = 高度差 / 水平距离"... 但课程没有明确说这是"中心差分"还是别的什么，也没有与"向前差分"(f(x+h)-f(x))/h 或"向后差分"(f(x)-f(x-h))/h 比较为什么中心差分更好。我记得 L22 应该讲过，但现在看这一课感觉断层了。

希望课程补充什么：回顾一下"为什么用中心差分而不是向前/向后"（精度更高、对称性更好），或者至少在代码注释里说明一下。
