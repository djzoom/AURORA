# 老陈 — L23 学习日志

## 我理解了什么
1. **偏导数** = 固定其他变量，对某一个变量求导。就像做汤时，把其他配料都锁住，只改变盐的用量看味道变化有多快。
2. **梯度** = 把所有偏导数拼成向量。对 f(x,y) = x² + y²，在点(3,4)处的梯度是[6,8]（分别是∂f/∂x=6和∂f/∂y=8）。
3. **数值梯度的实现** = 对每个维度用中心差分法：先向这个方向加h，再减h，相减除以2h就能逼近偏导数。验证过了，误差到1e-10级别。

## 遇到的困难和问题

### 【理解】何谓"梯度指向最陡上坡"？课程没有清楚推导
在"梯度的方向是什么"这段，课程说梯度[6,8]"垂直于等高线"、"指向函数值增大最快的方向"。我能看代码验证确实垂直了，但没理解"为什么"垂直就意味着最陡。这里需要一个几何解释：
- 等高线上的点函数值相同
- 沿等高线走，函数值不变
- 垂直于等高线的方向，就是离开等高线最快的方向
- 因此垂直方向的函数变化率最大

**希望补充**：用泰勒展开或数值例子解释中心差分相比单侧为什么精度更高（大概是因为消除了一阶误差项？）
