# 小雨 — L13 学习日志

## 我理解了什么
1. **正交矩阵就是旋转/镜像变换**：列向量两两垂直且长度为1，可以用 Q^T Q = I 验证，最妙的是求逆只需转置，不用计算！
2. **对称矩阵和正定矩阵是两个概念**：对称矩阵满足 A^T = A（沿对角线对称），正定矩阵满足 x^T A x > 0（能量为正），判定方法是看特征值或 Cholesky 分解。
3. **FFT 精确可逆的原因是酉矩阵**：因为 DFT 矩阵是正交族（转置=逆），所以 ifft(fft(x)) 不损失信息。

## 遇到的困难和问题

### 【理解】内积和二次型公式没有推导过程

Cell 4 说"内积 (Ax)·y = x·(A^T y)"，但没有推导。我理解点积 u·v 是对应元素相乘再求和，但涉及矩阵的内积为什么是这样，凭啥推出来的？同样 x^T A x > 0 这个表达式，怎样理解成"能量为正"的？需要逐步计算示例（比如 2×2 矩阵，给具体数字）。

### 【理解】Cholesky 分解的含义和用途说不清

Cell 16 说"Cholesky 分解 A = L L^T"，但 L 是什么？怎样求？Cell 17 的代码直接调 `np.linalg.cholesky(A)`，分解成功就说 A 是正定。**问题是**：为什么"分解成功"就能证明矩阵正定？这两者的数学关系是什么？L（下三角矩阵）和 A 的特征值有什么关系？这里跳跃太大。

### 【理解】为什么对称矩阵的特征值必为实数

Cell 9 和 Cell 17 多次提到"对称矩阵特征值为实数"，用 `eigvalsh` 而不是 `eig`。**问题是**：为什么？这涉及复数和特征值定理，我的高中数学没学过复数的特征值理论。课程为什么要区分对称和非对称矩阵求特征值？这背后的代数原理没讲。

### 【实践】try-except 异常处理语法不懂

Cell 17 的代码出现了：
```python
try:
    np.linalg.cholesky(A)
    tag = '✅ 正定'
except np.linalg.LinAlgError:
    tag = '⬜ 半正定'
```
我只学过 print、变量、循环、列表，没学过 try-except。这是 Python 的异常处理，超出基础范围。**问题是**：这段代码"尝试分解，失败就捕获异常"，为什么失败会抛异常？应该怎样理解这个逻辑流？

### 【理解】Gram 矩阵 X^T X 为什么总是对称且半正定

Cell 16 提到"PCA 的协方差矩阵 C = (1/N) X^T X 必须是 PSD——由 A = M^T M 构造的矩阵天然半正定"。**问题是**：为什么 M^T M 天然半正定？这需要证明吗？为什么 X^T X 一定是对称矩阵？（虽然 (X^T X)^T = X^T X 看起来对，但为什么？）

### 【理解】"酉矩阵"和正交矩阵的区别

课程开头说"酉矩阵（unitary matrix）与正交矩阵的关系"，Cell 14 说"DFT 矩阵是酉矩阵（U^-1 = U^H）"。**问题是**：酉矩阵是什么？什么是 U^H（共轭转置）？正交矩阵是实数域的特殊情况吗？为什么 FFT 涉及复数，就从正交矩阵升级到酉矩阵？这个概念跳跃有铺垫吗？

### 【实践】np.allclose 的容差设置细节

Cell 13 和 Cell 20 用 `np.allclose(x, y, atol=1e-12)` 比较矩阵，但为什么选 `atol=1e-12`？浮点数误差的量级是多少？为什么不用默认参数？这对判定"几乎相等"的标准没有解释。
