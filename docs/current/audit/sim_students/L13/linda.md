# 琳达 — L13 学习日志

## 我理解了什么（简要，3 行以内）

正交矩阵 = 旋转/镜像，保持向量长度不变，用 `np.allclose(Q.T @ Q, np.eye(n))` 验证。对称矩阵特征值都是实数。正定矩阵能通过 Cholesky 分解判定（分解成功 = 正定）。

## 遇到的困难和问题

### 【理解】正交性条件"列向量两两垂直且单位长度"到底怎么对应到"QᵀQ = I"

课程说"QᵀQ 的每个元素是两列向量之间的点积"。我能跟随代码验证这是真的，但推导过程没有。具体来说：
- 为什么矩阵乘法 Qᵀ 乘 Q 的结果，它的 [i,j] 位置恰好就是第i列和第j列的点积？这背后的矩阵乘法定义是什么？
- 高中没系统学矩阵，这个映射对我不是"显然的"。

**希望补充**：手算一个 2×2 例子，明确说出 (QᵀQ)[0,0] = Q[:,0] · Q[:,0]、(QᵀQ)[0,1] = Q[:,0] · Q[:,1] 等，这样就能看清楚了。

---

### 【理解】正定矩阵定义"xᵀAx > 0"中的符号含义不清

课程定义正定为"对所有非零向量 x，xᵀAx > 0"，但没说这是什么运算：
- x 是列向量 (n,)，Aᵀ 是 (n,n)，那 xᵀ @ A @ x 是怎样的形状变化？(1,n) @ (n,n) @ (n,1) = 标量对吗？
- 课程之后用"所有特征值 > 0"或"Cholesky 分解成功"来判定正定，这两个条件我能验证，但"xᵀAx > 0"的直觉没有。

**希望补充**：给一个 2×2 矩阵和一个具体向量 x 的例子，手算 xᵀAx，展示它确实是标量且为正。或者用 NumPy 直接算给我看。

---

### 【理解】为什么对称矩阵的特征值"必为实数"——完全没有解释

课程说"对称矩阵特征值必为实数"，这似乎是数学事实。但：
- 特征值是什么？从 `det(A - λI) = 0` 的对称性能推出 λ 是实数，但这个推导超出了高中数学。
- 课程没有说"为什么"，只是说"eigvalsh 比 eig 更快更稳定"。

**希望补充**：简单说明一下——要么给出简洁的数学解释，要么就说"这是一个代数定理，我们暂时用着"。不要悬空。

---

### 【理解】突然出现"酉矩阵"概念，和正交矩阵什么区别

课程在讲正交矩阵时说"DFT 矩阵是酉矩阵（U⁻¹ = Uᴴ）"，引入了共轭转置 Uᴴ 的概念。但：
- 共轭转置是什么？为什么需要这个东西？复数矩阵是什么？高中没学过。
- 课程说"正交矩阵保持向量长度"，那"酉矩阵"也保持某种"长度"吗？
- 为什么 FFT 用的是酉矩阵而不是普通的正交矩阵？什么时候两者有区别？

**希望补充**：简化一下——要么说"FFT 处理复数，所以用共轭转置而不是普通转置"，要么就说"酉矩阵是正交矩阵在复数域的推广，细节后续补"。现在这一句话太密集了。

---

### 【理解】"协方差矩阵是半正定"这个说法什么时候用

课程在正定部分提了一句"PCA 的协方差矩阵 C = (1/N)XᵀX 必须是 PSD"。但：
- 协方差矩阵是什么？为什么必须是半正定？什么时候是半正定、什么时候是正定？
- 课程没有给出这些背景信息，只是点了一下。

**希望补充**：要么这节课就讲清楚"半正定 vs 正定"的实际区别和应用场景，要么就不提 PCA，留到 L16 再说。现在这句话对新手来说是"悬空的知识"。

---

### 【实践】np.allclose 的浮点容限什么时候该调

课程说要用 `np.allclose(Q.T @ Q, np.eye(n))` 而不是 `==` 因为浮点误差。我试过不同的代码，确实工作了，但：
- np.allclose 的默认 atol 和 rtol 是多少？什么情况下需要调参数？
- 课程有的地方用 `atol=1e-12`，有的地方用默认值——标准是什么？
- 如果我的项目里某个矩阵接近正交但不完全正交，怎样判定"够不够接近"？

**希望补充**：一句话说清楚——比如"np.allclose 默认 rtol=1e-5，atol=1e-8，对浮点运算的数值误差够了"。或者给出一个例子展示"如果容限太紧会怎样"。

白板挑战最后说"盖上屏幕，纸上作答"，但问1"Q.T @ Q = I"直接一行代码就能验证，为什么不鼓励边写边验？手算对初学者真的更有帮助吗？还是这只是教学仪式？

**希望补充**：要么说明手算的目的（比如"理解矩阵乘法的细节""培养直觉"），要么改成"先手推再用代码验证""对答案对不上就看推导哪错了"。

## 总体反馈

这课结构清晰、代码例子管用，是好课。但对"为什么"的解释不够系统——假设了读者已经懂矩阵运算，但实际初学者还在"矩阵乘法怎么算"的阶段。特别是正定性、特征值、复数矩阵这几处，建议加更多铺垫或明确说"后续课讲"。
