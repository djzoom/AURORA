# 老陈 — L35 学习日志

## 我理解了什么

欧拉公式把复数写成 `cos(θ) + i·sin(θ)` 的形式，这两个东西加起来就代表单位圆上某个位置。FFT 的旋转因子 `twiddle(k, n, N)` 就是套用这个公式，把特定角度 `-2πkn/N` 代入，得到一个转啊转的复数——每次转多少度取决于参数。实现函数很直白：前一个直接堆 cos 和 sin，后一个套用前一个就完事。

## 遇到的困难和问题

### 【理解】欧拉公式为什么成立？

课程用"单位圆上的秒针"做比喻，说实部是 cos、虚部是 sin，那它们加起来就是 $e^{i\theta}$。但这只是告诉我*结果长什么样*，没有说明**为什么指数函数和三角函数突然跑到一起了**。我的高中数学里知道 $e^x$ 是增长函数，知道 $\sin/\cos$ 是周期函数，但怎么就能用 $\cos\theta + i\sin\theta$ 来定义 $e^{i\theta}$？这背后的第一原理或推导链条是什么？是用泰勒级数吗？课程需要补充或至少提示一下。

### 【理解】复数乘法旋转的几何原理

代码演示了"乘以 1j 会转 90°"，这个规律是对的，但**为什么**乘以虚数单位就能旋转？这涉及复数乘法的极坐标形式（模和辐角），但高中数学里复数通常只用矩形坐标 $a+bi$ 讲，没有接触到"两个复数一乘，模相乘、角度相加"这个性质。我能背下来规律（因为代码验证了），但搞不清楚**机制**。希望课程补充一个快速的"复数乘法就像旋转加缩放"的说法。

### 【理解】旋转因子里的负号为什么必须是负？

课程说 `-2πikn/N` 中的负号是"DFT 的习惯约定，表示顺时针旋转"。但为什么 DFT 偏偏选顺时针而不是逆时针？这只是习惯吗，还是有数学上的必然原因——比如，这样定义能简化某个公式，或者对应信号处理的某个物理含义？课程没有说。我现在只能当规则背下来，但感觉理由不足。

### 【理解】参数 k、n、N 各自的作用和意义

课程用实验展示"改 N 从 8 到 16，点密度翻倍"，和"改 k 从 1 到 3，旋转速度加快"。这些**现象**我能看懂，但**物理意义**是什么？
- `k` 是频率编号，所以它控制"这次我在检测第几个频率"——旋转速度快就是检测更高的频率，对吗？
- `n` 是采样点编号，所以随着 n 增加，秒针要一圈圈转——这模拟信号随时间推进，对吗？
- `N` 是总采样数，缩小间距就是频率分辨率提高，对吗？

课程没有明确说这些映射关系，我是靠猜测和代码的现象推理出来的。最好课程能有一个小表格或图示，标清"参数→物理意义"的对应。

### 【推演】twiddle(1,2,8) = -1j 这个特例为什么非平凡？

课程强调这是"非平凡验证"，但没有说明**为什么这个特例很重要**。让我手算一下：
- 角度 = $-2\pi \cdot 1 \cdot 2 / 8 = -\pi/2$
- $\cos(-\pi/2) + i\sin(-\pi/2) = 0 - i = -i$ ✓

结果确实是 -1j。但这展示了什么？它是个"整数倍 90°"的特例，所以能精确出来不涉及浮点误差吗？还是它代表某个具体的 FFT 场景？课程需要一句话解释"这个特例的意义"。

### 【理解】图示 `aviz.twiddles(8)` 会看到什么？

代码是 `aviz.twiddles(8);` 但没有任何说明——会画什么图？为什么要看这个图？它与前面的 `euler` 圆形、参数实验的数字打印有什么区别？我现在略过了这行，因为我在做文本学习、没法跑代码看效果，但其他学生运行后可能也不知道这图想教什么。

课程最后说"先做过 L06 的话今天的推导会更流畅；没做过的话，完成本节后再回去看 L06"。但我现在不知道 L06 讲了什么，也不清楚两课的**分工**——L06 是"理论推导"而 L35 是"代码实现"吗？还是覆盖范围不同？建议课程在开头明确说一下"如果你没看过 L06，这里会有 X、Y、Z 三个概念需要补课"。
