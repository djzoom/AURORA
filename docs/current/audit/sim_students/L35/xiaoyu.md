# 小雨 — L35 学习日志

## 我理解了什么

**欧拉公式的秒针类比很清楚**：e^{iθ} = cos θ + i·sin θ 就是把一个角度 θ 变成单位圆上的坐标，实部是横向（cos），虚部是纵向（sin）。

**旋转因子 twiddle 是 FFT 的积木**：公式 e^{-2πikn/N} 说明了在某个频率 k、某个时刻 n、总长 N 的信号长度下，应该转到单位圆的哪个位置。

**模长恒为 1 的性质**：任何角度 θ，(cos θ)² + (sin θ)² 用勾股定理永远 = 1，所以 e^{iθ} 永远在单位圆上。

## 遇到的困难和问题

### 【理解】为什么一定要用虚数？

课程说"复指数是把 sin/cos 合并成单位圆上的坐标"。我理解了用实数对 (cos θ, sin θ) 能表示圆上一个点，但为什么非要用复数 cos θ + i·sin θ？实数对不也可以吗？虚数单位 i 到底代表什么物理意义？能补充一个对比例子吗？

### 【推演】负号从何而来

公式里的负号（-2πikn/N）课程说是"DFT 的习惯约定"。单位圆上逆时针是正方向，那"负号代表顺时针"这个说法如何推导？为什么 DFT 偏偏要用负号而不是正号？能举一个 k=1, n=0,1,2 具体的旋转过程说明吗？

### 【计算】负角度的三角函数

表格只给了 0, π/2, π, 3π/2, 2π 这些正角。白板挑战要求手算 twiddle(1,2,8) = euler(-π/2)，需要 cos(-π/2) 和 sin(-π/2) 的值。我知道 |-π/2| = π/2 对应圆上的 (0,1)，但 **-π/2 是逆时针还是顺时针？** 能补一张标注负角度的单位圆图吗？

### 【理解】乘以旋转因子 = 问"投影有多大"

课程说"乘以旋转因子相当于问 x[n] 在这个频率上的投影"，但从 x[n] × e^{-2πikn/N} 这个乘法运算看不出"投影"的含义。投影是什么？能用一个生活类比（比如灯光、或频率的角度）解释吗？

### 【实践】代码中 1j 的用法

实现 euler 时要返回 `np.cos(theta) + 1j * np.sin(theta)`。Python 里 1j 代表虚数单位很特殊（不是数学符号 i），numpy 函数是否对标量和数组都兼容？如果 theta 是数组，`np.cos(theta)` 和 `1j * np.sin(theta)` 直接相加会得到复数数组吗？
