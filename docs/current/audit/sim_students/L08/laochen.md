# 老陈 — L08 学习日志

## 我理解了什么

欧拉公式 e^{iθ} = cos θ + i·sin θ 把角度变成复数，对应复数平面上单位圆的旋转。共轭 z̄ 就是虚部取反，几何上是关于实轴做镜像。代码验证了这些性质：复数乘以它的共轭得到模的平方，两个共轭复数的模始终相等。

## 遇到的困难和问题

### 【理解】欧拉公式的推导铺垫缺失
笔记里直接给了 e^{iθ} = cos θ + i·sin θ，但没说这是怎么推出来的。我会 Python 算这个公式，验证结果对，但"为什么实部恰好是 cos、虚部恰好是 sin？"这个背后的原因课程没讲。高中数学没学过这种指数和三角函数的关系，所以这里只能把它当"已知事实"背了，没有真正理解透。希望前面的 L05/L06 能补充这个推导过程，或者至少说明是泰勒级数展开的结果。

### 【理解】"关于实轴对称"的几何直觉不清楚
共轭的定义"虚部取反"我看懂了，代码验证也通过了。但课程说"关于实轴对称反射"，这个说法有点抽象——具体是什么对什么对称？如果 z = 3+4j，那 z̄ = 3-4j，它们在平面上确实关于横轴（实轴）上下翻转，这我能画出来。但为什么要把这个操作叫"共轭"？有什么几何或物理意义吗？

### 【推演】DFT 旋转因子的负号从何而来
课程提了一句"旋转因子 W_N^k = e^{-i2πk/N}"，但为什么是**负号**？是 −2π 而不是 +2π？如果 k=1, N=8，那是 e^{-i·2π/8} = e^{-iπ/4}，这代表顺时针旋转 45°。但课程没说明：负号代表什么物理意义？顺时针 vs 逆时针有什么区别吗？（L06 应该解释过，但我需要再看一遍确认。）

### 【理解】"投影系数"和复数点积的具体含义
课程写"FFT 做的就是高效计算信号在每个频率分量上的投影系数"。这里的"投影"我理解成向量点积，但高中数学里点积是实数，两个向量的"几何"点积。现在换成复数了，"点积"还是同样的定义吗？复数 z1 和 z2 的点积是什么公式？如果涉及共轭（z1̄ · z2），课程要明说。
