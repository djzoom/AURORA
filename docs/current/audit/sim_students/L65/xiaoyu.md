# 小雨 — L65 学习日志

## 我理解了什么（简要，3 行以内）

训练不是一条直线：计算图用箭头记录了每一步计算，反向传播就沿着这些箭头往回送梯度。看两条线的图表就能发现过拟合——验证集损失开始上升是信号。混淆矩阵是个计分板，对角线是对的，其他地方是错的，颜色深说明这个错误很常见。
