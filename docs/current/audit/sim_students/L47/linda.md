# 琳达 — L47 学习日志

## 我理解了什么

- **5 步流水线**一下子清楚了：STFT 拿到复数 → 平方变功率 → 乘以三角滤波器组 → log 一压缩，就是我们要的 Mel 特征。矩阵乘法 `power @ fb.T` 实现了一帧多个能量值的批量计算。
- **log 压缩的作用**：跨越几十倍数量级的能量值不加 log 的话，神经网络学习困难；加了 log 后数值分布平衡，梯度稳定。这样想就有意义了。
- **eps = 1e-8 的用处**：防止 log(0) 变成 -∞，把最小值约束在 log(1e-8) ≈ -18.4，这是数值稳定的技巧。

## 遇到的困难和问题

### 【理解】Mel 尺度公式是如何推导的
课程说"人耳对数感知"就导出 `mel = 2595·log₁₀(1+f/700)`，但这个 2595 和 700 这两个魔法数字是怎么来的？是实验测出来的吗？还是有数学推导？这个公式没有推导过程，我卡壳了。

### 【理解】三角形滤波器和"50% 重叠"的物理-数学映射
课程说"耳蜗基底膜有 50% 空间重叠" → 设计滤波器时"相邻滤波器重叠 50%" → 这导致三角形形状。但这个因果链条怎么确定的？为什么一定是三角形而不是高斯形、梯形或其他平滑形状？如果设定 40% 或 60% 重叠，滤波器形状会变吗？这个从物理观察到数学设计的转换规则没有讲。

### 【计算】三角滤波器面积的解析推导
最后一个闭卷题要求推导分段函数的面积。三角形的数学定义有三段，我需要手算几何和。虽然高中数学可能够，但题目说的"面积"是求和 $$\sum_k H_m(k)$$ 吗？频率轴是离散 bin 还是连续的？这会影响面积公式的形式。另外"HTK 惯例"是什么术语？课程没有解释。

### 【实践】STFT 返回格式和复数处理
代码框架中 `S = stft(x, n_fft=1024, hop_length=256)` 返回什么？如果是复数矩阵，那么 `power = np.abs(S) ** 2` 是对复数的模求平方吗？还是说 np.abs 会自动把复数转为模（幅度），然后平方？我需要看 L44 确认 STFT 的具体输出格式。

### 【理解】为什么前 40 个 Mel bin 覆盖 0–1730 Hz
课程数据说"前 40 个 bin 集中在 0–1730 Hz，后 40 个 bin 平摊 1730–8000 Hz"。这和 Mel 公式的非线性有关，但具体怎么分配的？是用 mel_to_hz 反函数逐个计算吗？还是 Mel 尺度内均匀分割后自动得到的 Hz 分布？这个二者的关系没有弄透。
