# 小雨 — L41 学习日志

## 我理解了什么

加窗就是在做FFT之前，先把信号边缘"柔和地降低"，用Hann窗（边界是0，中间隆起的曲线）替换掉矩形窗（边界生硬断裂）。这样就不会因为信号被硬生生截断而产生频谱泄漏了。实现就三步：取帧 → 乘窗 → FFT。

## 遇到的困难和问题

### 【理解】时域乘法为什么会导致频域卷积？

课程说"时域乘以矩形窗 → 频域卷积 sinc"，但完全没有解释**为什么**。作为一个高中数学水平的学生，我听过傅里叶变换，但没学过卷积（convolution）是什么。能不能用图或类比解释：为什么信号在时间上被"切一下"，频率上就会"散开"？能不能告诉我卷积在这里具体是什么意思？

### 【理解】什么是 sinc 函数、主瓣和旁瓣？

课程说"矩形窗的旁瓣峰值比主瓣低 −13 dB"，但完全没有说：sinc 是什么函数？它长什么样？"主瓣"和"旁瓣"分别指哪些部分？用图示的话会不会更清楚？如果没有 sinc 的概念，−13 dB 这个数字对我就是个天书。

### 【推演】Hann 窗公式 w[n] = 0.5(1 − cos(2πn/N)) 是怎么设计出来的？

课程给了公式和它的优点（旁瓣低约 −31 dB），但没有说**为什么选择这个特定的余弦公式**。为什么不是别的函数？为什么是 0.5 × (1 − cos...)？这个公式是凭空创造的吗，还是有某种数学原理决定了它必须这样？能否补充一个"Hann窗设计思路"的段落？

### 【理解】周期型 (periodic) vs 对称型 (symmetric) 窗函数区别

课程在讲 Hann 窗时说"周期型（periodic），与对称型 N−1 分母不同"。但是：
- 什么是对称型？分母改成 N−1 会怎样？两种的图形差别大不大？
- 什么时候用哪种？在 STFT 里用的是哪种？
- 这会影响最后的 windowed_fft 实现吗？

### 【理解】参数实验中的 "排除主瓣 ±5 bin" 和旁瓣数值不匹配

实验中说要"排除主瓣 ±5 bin"来计算"最大旁瓣"，得到的矩形窗约 −24 dB、Hann 约 −55 dB。但开头说矩形窗旁瓣是 −13 dB、Hann 是 −31 dB。这两组数字为什么不一样？是测量位置不同吗？这个差异会不会给初学者造成混淆，让人怀疑教材的一致性？

### 【计算】频率不对齐 bin 时的计算看不懂

课程说"440 Hz 不是 fs/N 的整数倍"，所以泄漏严重。然后在参数实验中说"试试 437.5：恰好 112 个整周期"。我想问：
- 437.5 Hz 和 "112 个整周期"这个数字是怎么来的？中间的计算是什么？
- 为什么这个频率就"恰好对齐"了？

### 【理解】什么是奈奎斯特频率？

实验中说"别选 500 Hz：那是奈奎斯特频率，sin 在采样点上恰好全为 0"。这是什么？为什么 500 Hz 时 sin 全为 0？fs=1000，N=256，500 Hz 和这些参数有什么关系？
