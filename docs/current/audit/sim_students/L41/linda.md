# 琳达 — L41 学习日志

## 我理解了什么（简要，3 行以内）

1. **频谱泄漏的问题与解法**：截断信号的两端会产生陡峭的边界跳变，导致频域里有很多旁瓣噪音（频谱泄漏）。解法是用平滑衰减到零的窗函数（比如 Hann 窗）替代硬截断，这样边界就没有突兀的跳变，旁瓣就少很多。

2. **加窗 FFT 的三步流程**：从信号取一帧 → 逐点乘以窗向量 → 做 FFT。代码实现非常直白，就是 `X = fft(x_frame * window)`。

3. **Hann 窗的工程效果**：Hann 窗用公式 `w[n] = 0.5(1 - cos(2πn/N))` 定义，边界处自动变成 0，中间是 1；实验对比显示它能把旁瓣压低约 30 dB，比矩形窗（无窗或权重全1）好很多。
