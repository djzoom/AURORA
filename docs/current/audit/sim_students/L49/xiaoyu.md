## 我理解了什么

这节课要教的是一个叫 DCT-II 的数学变换，它能把 40 个相关的数字变成 13 个差不多独立的数字。类比就像把说了两遍的话压缩成一份。核心公式是 `X[k] = w(k) * Σ x[n] * cos(...)`，还要特殊处理 k=0 的权重。用这个变换可以从 log-Mel 特征提取到 MFCC 系数。

## 遇到的困难和问题

### 【理解】正交归一化权重为什么是这两个值？

课程说"k=0 用 $\sqrt{1/N}$，k>0 用 $\sqrt{2/N}$"，理由是"k=0 的基向量模长为 √N，而 k>0 的基向量模长为 √(N/2)，为使所有基向量在同一尺度下正交，k=0 需额外除以 √2"。

**卡在哪**："基向量的 L2 模长"和"权重应该是多少"之间的逻辑连接我推不出来。如果 k=0 的基向量模长是 √N，为什么就要除以 √2 得到 1/√N？这个比例关系来自哪里？课程假设我知道什么是 L2 模长、知道"正交"对应什么数值条件，但这些对我还是黑盒。

**希望补充**：用具体数值（比如 N=4）手算一遍，展示 k=0 和 k=1 两个基向量的 L2 模长分别是多少，然后说明为了让他们"在同一尺度"需要乘上什么权重。

---

### 【推演】为什么乘以这两个权重后矩阵就变成正交矩阵？

课程说"M @ M.T 的对角线为 N/2（k>0）或 N（k=0）"（这我能从演示代码理解），然后说"归一化后 D @ D.T = I"。

**卡在哪**：对角线是 N 或 N/2，乘以 sqrt(1/N)^2 = 1/N 或 sqrt(2/N)^2 = 2/N，这样对角线变成 1——我能算出来。但非对角元也要变成 0，课程没说为什么乘以这两个权重就能让非对角元保持 0。

**希望补充**：说明正交矩阵的条件是"行向量两两正交且长度为 1"，然后手算验证一下非对角元（比如第 0 行和第 1 行的内积乘以权重后是否为 0）。
