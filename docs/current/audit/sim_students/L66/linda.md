### 【推演】DP转移方程中"删除、插入、替换"的含义需要画图

卡在第4节DP转移方程。公式本身我能读：

```
if hypothesis[i-1] == reference[j-1]:
    dp[i][j] = dp[i-1][j-1]
else:
    dp[i][j] = 1 + min(
        dp[i-1][j],    # 删除
        dp[i][j-1],    # 插入  
        dp[i-1][j-1]   # 替换
    )
```

但这里"删除"、"插入"、"替换"**针对的是谁的操作**？我理解是：把hypothesis的前i个词变成reference的前j个词。那么`dp[i-1][j]`（删除）就是说"已经把hyp的前i-1个词变成ref的前j个词了，现在删除hyp[i-1]"——这对吗？还是说"保留hyp[i-1]，把ref[j]删掉"？

建议：画一个3×3的小矩阵例子，标清楚"删除谁、插入谁"，我就能确认理解是否正确。
