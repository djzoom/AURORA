# 琳达 — L26 学习日志

## 我理解了什么

终于把梯度下降"看见"了！原来导数就是函数在某一点的斜率，负梯度就是"最陡下坡"的方向。一维时沿着斜率反方向走，二维时垂直等高线向低处走，都是同一个逻辑。学习率越大步子越大，但太大了会在谷底左右弹跳甚至发散——这个从可视化图一下子就明白了。

## 遇到的困难和问题

### 【理解】中心差分的分母为什么要乘 2？
笔记用 `(f(x+h)−f(x−h))/(2h)` 来近似导数，但直观上，两个点间隔是 `h`，分母不应该就是 `h` 吗？为什么要 `2h`？那验证断言里"检查公式分母是否为 2h"的警告，说明有很多人会写成 `h`。需要课程讲清楚为什么中心差分特别要用 `2h`，不是只是给公式。

### 【推演】`f(x) = x²` 的导数为什么是 `2x`？
笔记里直接说"理论值 `2x`"，但怎么推出来的？这应该是前面 L22 讲过，还是这个课程假设我已经知道？没有推导或出处的话，我只能死记。

### 【理解】什么叫凸函数、什么叫鞍点？
说"梯度下降对凸函数必然收敛"和"深度学习中的非凸损失才会出现鞍点"，给了反例 `f(x,y)=x²−y²` 说它有鞍点，但没解释为什么那个函数有鞍点、什么情况下一个函数才是凸的。这两个概念对理解梯度下降的局限很重要，但完全是黑盒。

### 【实践】练习验证中"负梯度更新后函数值严格下降"
断言检查 `f(x - lr*slope) < f(x)`，假设这总是真的。但这是为什么？这是一个数学保证吗？什么时候会失败？课程只给代码没给原理。

### 【理解】梯度下降轨迹与 loss 曲线的对应关系
最后说"每个 epoch 的 loss 值对应参数空间里的一个位置，打印出来的是它在损失轴上的投影"。这个"投影"是什么意思？折线轨迹在二维(x,y)空间里，loss 曲线是一维的，怎样从轨迹读出 loss 值？

### 【计算】学习率多大会导致发散？
看到 `lr=0.5` 会振荡、`lr=0.1` 稳定，但对一般的函数怎样判断 lr 的安全范围？比如有没有一个公式，根据函数的二阶导数之类的算出 lr 上界？

### 【推演】二维梯度的分量怎么算？
函数 `f(x,y) = x² + 2y²`，梯度说是 `(2x, 4y)`。对 `x²` 求关于 x 的偏导是 `2x`，我勉强能接受（虽然没推导）；但对 `2y²` 求关于 y 的偏导为什么是 `4y` 而不是 `2y`？这个系数 2 怎么来的？
