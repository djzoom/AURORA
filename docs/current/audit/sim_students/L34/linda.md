# 琳达 — L34 学习日志

## 我理解了什么

采样率必须大于 2 倍的最高频率（Nyquist 定理），否则高频会伪装成低频混叠。用公式 `alias = |freq - sr · round(freq/sr)|` 能计算出高频折叠成的假频率。6kHz 用 8kHz 采样就会变成 2kHz，在采样点上完全看不出差别。

## 遇到的困难和问题

### 【理解】"丢失连续细节导致频率无法区分"是什么意思？

Notebook 说"采样只记录离散时刻的振幅，丢失了两次采样之间的连续细节。超过 Nyquist 的频率 f 与其镜像频率在每个采样点上产生完全相同的数值"。

但这是**怎样完全相同的**呢？为什么 6kHz 和 2kHz 在采样点上数值一样？需要看具体的数学过程——sin(2π·6000·n/8000) 和 sin(2π·2000·n/8000) 到底怎样逐点相等？

# 与 −2kHz 比差异应该很小
assert diff_minus < 1e-10
