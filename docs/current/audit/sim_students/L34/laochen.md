# 老陈 — L34 学习日志

## 我理解了什么

学会了"高频伪装低频"这个现象：用采样率 8000 Hz 采样 6000 Hz 的正弦波，会因为采样点不够密集，采样出来的值跟 2000 Hz 的波一样，系统分不清。这就是**混叠（aliasing）**。折叠公式是 `alias = |freq - sample_rate × round(freq/sample_rate)|`，可以预测任何频率在欠采样时会冒充什么低频。这就是为什么 CD 要用 44100 Hz——能覆盖人耳的 20 kHz。

## 遇到的困难和问题

### 【理解】"0.75 圈和 -0.25 圈产生同样的采样点"——缺少数学推导
**卡在哪**：Cell 3（本课剧情）的直觉部分，"6 kHz 波在每个间隔内转了 0.75 圈——比 0.5 圈多，但数字化系统'看不出' 0.75 圈和 -0.25 圈的区别"。

**具体困难**：为什么这两个不同的"圈数"会产生相同的采样点值？sin(2π×0.75) 真的等于 sin(2π×(-0.25)) 吗？用了什么三角函数规律？

**希望的补充**：需要一行三角恒等式来打破这个疑惑——比如"因为 sin(θ) = sin(θ - 2π)，所以 sin(0.75×2π) = sin((0.75-1)×2π) = sin(-0.25×2π)"。高中我学过周期性，但现在想不起来，课程可以明确写一遍。

### 【推演】相位翻转公式"sin(2π(fs−f)n/fs) = −sin(2πfn/fs)"没有推导
**卡在哪**：Cell 11 验证折叠的代码注释，"正弦向下折叠会附带相位翻转"。

**具体困难**：这个恒等式怎么证明？需要用到和差化积公式吗？为什么是负号？如果我要手动验证 6 kHz 和 2 kHz 的反相关系，应该怎么展开三角函数？

**希望的补充**：给出两行推导——用 sin(A-B)=sinA·cosB-cosA·sinB（或类似恒等式）展开，说明 sin(2π(fs−f)n/fs) 里的 2πfs·n/fs=2πn 项被 sin 抹成周期，留下 −sin(2πfn/fs) 这一项。

### 【理解】"折叠图案每隔 sr Hz 重复一次"——缺少周期性证明
**卡在哪**：Cell 16（参数实验部分），"规律：折叠图案每隔 sr Hz 重复一次；Nyquist（8000 Hz）是对称轴"。

**具体困难**：为什么恰好是 sr 的周期？能否用公式证明 alias(f + sr) = alias(f)？这个规律跟频率的周期性有什么深层关系吗？

**希望的补充**：简洁的证明——"alias(f+sr) = |(f+sr) - sr·round((f+sr)/sr)| = |f + sr - sr·(round(f/sr)+1)| = |f - sr·round(f/sr)| = alias(f)，所以周期是 sr"。

### 【理解】Nyquist 频率为什么是 sr/2
**卡在哪**：Cell 7（概念部分），"采样率 sr 能如实表示的最高频率是 sr/2"。

**具体困难**：为什么恰好是二分之一？能不能是 sr/3 或 sr/1.8？这个 1/2 是怎么推导出来的？

**希望的补充**：给出最少的数学直觉——比如"如果采样间隔是 1/sr 秒，频率为 sr/2 的波在相邻两个采样点之间转半圈（0.5 周期），这是分辨极限；频率更高就无法区分了"。
