# 老陈 — L54 学习日志

## 我理解了什么

这节课的核心是：神经网络太复杂了，不能手写每个参数的偏导数公式。所以工程师造了一个 `Value` 类，让每个运算（加法、乘法等）都自动记录自己的"上游"节点和计算方式。反向传播时，就从损失值开始，沿着这条记录反向推导，梯度自动流回到每个参数——这就是自动微分的核心思想。

我看懂了为什么要用闭包 `_backward` 和为什么梯度要用 `+=` 累加（因为同一个变量可能在多个地方被用到，梯度要加起来）。`backward()` 函数要按某种特殊顺序（拓扑逆序）调用，我找到了思路。
