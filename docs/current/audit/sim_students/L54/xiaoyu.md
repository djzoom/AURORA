# 小雨 — L54 学习日志

## 我理解了什么

1. **计算图的思想**：每次做 `c = a + b` 这样的操作，就隐式建一个节点。反向传播就是从输出节点沿这张图逆向走一遍，把梯度层层传回去。

2. **梯度累积的必要性**：同一个变量 b 如果被用了两次（如 `a*b + b**2`），它的梯度要来自两条路径的相加，不能覆盖。

3. **链式法则的自动化**：每个算子只需要知道自己的"一阶局部偏导"（比如乘法的 `dc/da = b`），反向传播自动把这些链接起来成完整的梯度。
