# 琳达 — L30 学习日志

## 我理解了什么

1. **Softmax 的作用**：把模型输出的任意分数（logits）转成和为 1 的概率分布，最高分数对应最高概率。数值稳定的做法是减去 max，防止 exp 溢出。

2. **交叉熵（Cross Entropy）**：只看真实类别那一格的概率，损失就是 `-log(p[true])`。预测越自信且正确，损失越小。

3. **梯度公式 `dL/dz_i = p_i - y_i`**：这是反向传播的第一步，解释了"预测概率与真实标签的差"就是要调整 logit 的方向和大小。
