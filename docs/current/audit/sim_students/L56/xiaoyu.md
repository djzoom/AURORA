# 小雨 — L56 学习日志

## 我理解了什么

- **反向传播的大逻辑**：梯度像水一样从输出节点 `L` 往回流向输入，每经过一个算子就乘以那个算子的局部导数，这就是链式法则展开的过程。
- **顺序很关键**：必须从后往前（逆拓扑序）调用每个节点的 `_backward()`，这样每个节点收到的梯度才是"齐的"（来自所有下游节点的贡献都加起来了）。
- **梯度累积用 `+=` 不用 `=`**：同一个参数被多次用到（如 `a*a`），梯度要相加而不是覆盖。
