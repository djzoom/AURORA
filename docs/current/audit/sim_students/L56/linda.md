# 琳达 — L56 学习日志

## 我理解了什么

反向传播其实就是**从输出节点开始，沿着计算图往回走，每次用链式法则把上游梯度乘以本地导数，再传给输入节点**。梯度必须 `+=` 累积而不是覆盖，因为一个参数可能被多个地方用到（"菱形图"）。最关键的细节：必须按特定顺序（逆拓扑序）调用 `_backward()`，否则有些节点的梯度还没算完就被使用，结果全是 0。
