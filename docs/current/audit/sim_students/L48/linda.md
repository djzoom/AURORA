# 琳达 — L48 学习日志

## 我理解了什么（简要，3 行以内）

分帧就是用一个滑动的小窗口一次一次地切信号，每次做 FFT；Mel 滤波器组是在频率轴上用三角形加权，特点是低频开放、高频压缩。STFT 热力图用**行**显示频率、用**列**显示时间，整串在一张图里就一目了然了。

## 遇到的困难和问题

### 【理解】Hann 窗函数公式的由来

> 原文："窗函数（Hann，周期型 / DFT-even）`w[n] = 0.5 * (1 - cos(2π n / N))`"

窗函数长什么样我能从代码里看懂，但**为什么是 0.5 乘以 (1 - cos(...))** ？这个公式怎么推导出来的？课程只给了公式本身，没有解释为什么选这个形式。我高中数学知道 cos 的周期性，但这个 0.5 系数和这种组合方式是从哪儿来的，有推导过程吗？

### 【理解】Mel 频率转换公式的数字含义

> 原文："`mel = 2595 * log10(1 + f / 700)`"

这两个数字 **2595 和 700** 是怎么来的？凭什么是这两个数？这个公式体现了人耳的哪个物理特性？光知道"低频密、高频疏"还不够——我想知道这个特定的数学形式是怎么对应人耳感知的。课程没有任何铺垫。

### 【计算】幅度谱和功率谱为什么用不同的 log 倍数

> 原文："S 是幅度谱（amplitude）→ `20·log10`；若是功率谱（power）则用 `10·log10`"

这两个数字（20 vs 10）是怎么来的？有数学关系吗？是因为功率 = 幅度²，所以 log 里的指数倍数也要变？能给个推导吗？

### 【理解】动态范围设置 `vmin = S_db.max() - 60`

> 原文：代码里用 `vmin=S_db.max() - 60` 控制热力图显示范围

为什么是减 60？60 dB 这个数字的来源是什么？是惯例吗？还是说有某种标准？换成 50 或 70 会怎样？

### 【推演】帧数公式是怎么推导的

> 原文："帧数 = `1 + (N_sig + 2 * pad - n_fft) // hop_length`（当 `n_fft` 为偶数时 `= 1 + N // hop_length`）"

这个公式看着很复杂。特别是为什么要 `+ 2 * pad` 这部分——那是因为 `center=True` 在两端填充了嘛，我明白。但从信号总长度 N，到帧数这个公式，能一步步推导吗？`1 + (...)//hop` 这个 `+1` 是什么意思？最后一帧什么时候停止？
