## 我理解了什么

分帧就是把长信号切成短小重叠的片段，每个片段和一个下降到0的窗函数相乘，这样重叠拼接时不会有断层。Mel滤波器组是用一堆三角形在频率轴上做加权平均，使得Mel轴上均匀分布，结果是同样维度下低频细节保留得更多。线性谱和Mel谱对比，说的是一个频率感知问题——人耳对高频的分辨率没有低频那么强。

# 老陈 — L48 学习日志

## 我理解了什么

分帧把长信号切成短的重叠片段，用Hann窗函数让边缘平滑。Mel滤波器组用公式 mel = 2595·log10(1 + f/700) 把线性频率轴压缩成Mel轴，目的是让Mel轴均匀分布，这样同样维度下可以保留更多低频信息。

## 遇到的困难和问题

### 【理解】频谱泄漏是什么，窗函数怎么抑制的

讲义说"窗函数让帧边缘平滑降到0，抑制频谱泄漏"。我知道加Hann窗是标准做法，但泄漏具体是什么现象？为什么帧边缘从突兀的1降到平滑的0，频谱里就少了"泄漏"？这个因果关系没有演示，我心里没有直觉。

### 【理解】Mel刻度公式的来源和为什么是对数形式

公式 mel = 2595·log10(1 + f/700) 突然给出，没有解释为什么是这个形式。为什么用对数（log10）？为什么分母是700，为什么前面系数是2595？这些数字是怎么定出来的？虽然讲义说"Mel刻度对应人耳感知"，但没有说明人耳怎么感知的、这个公式怎么测出来的。

### 【推演】center=True 时帧数公式的化简

讲义说朴素帧数是 1 + (N - n_fft) // hop_length，但 center=True 时变成 1 + (N + 2·(n_fft//2) - n_fft) // hop_length，"当n_fft为偶数时恰好等于 1 + N // hop_length"。但我算不出这两个式子怎么相等。展开第二个式子：
- N + 2·(n_fft//2) - n_fft = N + n_fft - n_fft = N（对，n_fft是偶数）
- 所以 1 + (N - n_fft + n_fft) // hop = 1 + N // hop

好，这样化简过了。但讲义没有明确说"2·(n_fft//2) = n_fft"这一步，我一开始被"+2·(n_fft//2)"这个记号卡住了，不知道它等于多少。

### 【计算】dB转换的系数为什么幅度用20、功率用10

代码里 S（幅度谱）用 20·log10，M（功率谱）用 10·log10。讲义说"幅度 → 20·log10，功率 → 10·log10"，但没有推导。我知道 log(a²) = 2·log(a)，所以如果功率=幅度²，那么 log(功率) = log(幅度²) = 2·log(幅度)，所以 10·log10(功率) = 10·log10(幅度²) = 20·log10(幅度)，两者数值一致。**但讲义应该明说这个关系，不然容易记错。**

### 【理解】坐标轴变换是怎么改变了视觉均匀性

Cell 6 里，线性频率轴上的滤波器本身没变，但 x 轴改成了 Mel 频率轴，就"看起来"均匀了。这是坐标变换的视觉效果，我能理解（像极坐标和直角坐标的关系），但**讲义应该明说"这只是改变了绘图的坐标轴标签，滤波器的实际形状（在Hz轴上）没有变"**。否则初学者容易混淆"Mel轴上均匀"是数学性质还是视觉错觉。

### 【理解】两张热力图对比的意义

Cell 7 里对比线性谱和 Mel 谱：线性谱里升频轨迹在高频区占好多行，Mel 谱里升频轨迹上升均匀。讲义说这说明"低频细节在Mel谱里保留得更丰富"，但这句话怎么从图看出来的？**应该明说：同样的频率跨度，Mel 轴分配更多行数给低频（Mel轴上密），所以同样一条升频直线，在 Mel 谱里显得缓慢均匀；而线性轴上高频行数多，升频直线在高频快速上升。** 这样就清楚了。
