### 问题 10：计算图有多少个 Value 节点？如何计数？（laochen, linda, xiaoyu）
**原文**：白板挑战问5 给答案"约 45 个"，没有说清怎样算出来的

**改动**：
- 新增 markdown cell "## 深入讲解：计算图中的节点数（白板挑战问5的详细推导）"
- 定义"节点"：每一次运算都是一个节点
- **拆解一个 Neuron(3) 的计算**：
  - z1, z2, z3（3 个乘法）→ z4, z5, z6（3 个加法）→ y（1 个 tanh）
  - **总计 7 个新节点** = 2×3+1
- **一般公式**：Neuron(nin) 产生 2×nin+1 个新节点
- **MLP(3,[4,4,1]) 的完整计算**：L0=4×7=28, L1=4×9=36, L2=1×9=9 → **总计 73 个**（不是"约 45"）
- **说明**："约 45" 是教材的简化估算，精确方法应该逐级计数
- 强调"关键不是精确数字，而是理解节点数随深度和宽度快速增长"
- 新增 code cell：手工计数 Neuron(3)，列表推导一般公式
- **位置**：在白板挑战（"## ✏️ 白板挑战"）之前
