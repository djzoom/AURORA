# 小雨 — L57 学习日志

## 我理解了什么

1. **网络搭积木的逻辑**：Neuron（单个加权求和+激活）→ Layer（多个Neuron并行）→ MLP（多层首尾相连），像搭乐高一样一层层往上。

2. **参数个数的计算**：每个Neuron有 nin 个权重 + 1 个偏置，Layer 有 nout 个 Neuron，MLP 各层参数加起来就是总参数（表格和手算都能验证）。

3. **forward pass 的流程**：输入数据从第一层传到最后一层，每层输出作为下一层输入，最后得到结果。
