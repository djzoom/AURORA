# 老陈 — L57 学习日志

## 我理解了什么（简要，3 行以内）

- Neuron 就是一个线性变换（加权求和+偏置）加一个激活函数（tanh）；Layer 是多个 Neuron 并排，都处理同一份输入 x；MLP 是多个 Layer 首尾接起来。
- 参数总数有规律：每层的 Neuron 有 nin 个权重 + 1 个偏置，Layer 有 nout 个这样的 Neuron，所以一层 = nout×(nin+1) 个参数。
- 反向传播在每一层自动进行，因为计算图已经建好了；`parameters()` 方法把全部参数拍平，方便训练时逐个更新。
