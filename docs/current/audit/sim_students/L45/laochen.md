# 老陈 — L45 学习日志

## 我理解了什么

声谱图就是把STFT的复数结果"画出来"。步骤是：从复数提出幅度 → 平方变成功率 → 取对数变成dB → 用热力图显示。图的横轴是时间（秒），纵轴是频率（Hz），颜色越亮表示能量越强。这就像医生的X光——看不见看得见，现在用dB把弱音和强音都显示清楚了。

**我记住了三个关键操作**：
1. 转置 `.T`：让频率变成行，时间变成列（这样imshow才能画成"频率纵轴、时间横轴"）  
2. `origin="lower"`：让低频在下面（不是默认的上面）  
3. dB公式：`10·log₁₀(|S|² + ε)` 或等价的 `20·log₁₀(|S| + ε)`

## 遇到的困难和问题

### 1. 【理解】dB公式里为什么恰好是 10 倍或 20 倍？ 
课程说 `20·log₁₀(A + 1e-8)` 与 `10·log₁₀(A² + ε)` "近似等价"（Cell 5 和 Cell 12），但：
- 为什么不能是 5 倍或 30 倍？这个系数有什么物理原理吗？  
- "近似等价"和"完全等价"的区别在哪？什么时候会有明显差异？  
- **希望补充**：10倍和20倍的来源（是不是功率和振幅的定义导致的？）

### 2. 【推演】时间轴公式 `t = m·hop/sr` 为什么这样推导？
- STFT分帧时，第m帧从第 `m·hop` 个样本开始（这我能理解）  
- 但 **为什么要除以采样率sr**？sr是"每秒采样数"，`m·hop` 是"样本数"，怎么除出秒数的？  
- **希望补充**：写清 `(样本数) / (样本/秒) = 秒` 这个单位约化的过程

### 3. 【理解】"频率分辨率"和"时间分辨率"具体是什么，怎么衡量？
课程说"win_len增大→频率分辨率提高，时间分辨率下降"，表格也列了，但：
- 什么叫"频率分辨率"？能分辨的是什么？  
- 440 Hz 和 445 Hz 的两个信号在频谱上能"分开"吗？如果能，是因为win_len多大时能分开？  
- **希望补充**：用具体数字举例，比如在win=256和win=2048下，各能区分多少Hz的差距

### 4. 【实践】转置和 `origin="lower"` 到底改变了什么？
- STFT输出 `(n_frames, n_bins)`，即"行=时间帧，列=频率bin"  
- 转置后 `(n_bins, n_frames)` → "行=频率bin，列=时间帧"  
- 然后imshow把矩阵的"行"画成纵轴？（这个映射关系没看透）  
- `origin="lower"` 是让第0行（DC/0Hz）在图的下方，但为什么默认是上方？  
- **希望补充**：imshow的坐标系规则明白纸上画一遍（谁是第0行？谁是y轴方向？）

### 5. 【计算】ε的选择范围
- Cell 3 用了不同的ε值：`1e-10`（功率形式）vs `1e-8`（振幅形式）  
- ε太小会怎样？太大会怎样？  
- 对最后画出来的图有什么影响？（噪声底会不会可见？）  
- **希望补充**：给ε加个取值建议（比如"通常 1e-8~1e-10"）
