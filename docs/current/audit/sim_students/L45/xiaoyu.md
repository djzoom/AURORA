# 小雨 — L45 学习日志

## 我理解了什么（简要，3 行以内）

声谱图就是把 STFT 的复数矩阵转成能量图，流程是：幅度 → 功率（平方）→ 对数（dB）→ 热力图。横轴是时间（秒），纵轴是频率（Hz），颜色表示能量强度。用 `imshow` 画时要转置矩阵、设 `origin='lower'` 让低频在下面、用 `extent` 参数标注真实的时间和频率数值。

## 遇到的困难和问题

### 【理解】STFT 为什么输出复数，幅度平方后才是功率？

读到"STFT 输出的是复数矩阵，每个元素携带幅度和相位"，我知道复数的绝对值是幅度，但**为什么 STFT 不直接输出能量（幅度），而要先存幅度和相位两个信息**？丢掉相位、只保留幅度，那前面为什么要算相位呢？

还有"功率谱 = |S|²"，这个"平方"是从哪儿来的？物理学里功率是什么定义？为什么波的功率就是振幅平方？我在中学物理课没学过这个（可能学过但忘了），希望课程能补充一下声学里功率的定义。

### 【计算】20log₁₀(A) 和 10log₁₀(A²) 何时相等，epsilon 的微小差异具体是什么？

课程说这两个公式"近似等价，但 eps 楼层略有差异"。我试了试，当 A=1 时都等于 0，当 A=0.001 时都等于 -60 dB。那**什么时候它们会不一样**？是不是只在 A 非常接近 0 时才有差异？

还有一句"ε=1e-10 对应 -100 dB 噪声底"，我看不出这个对应关系。如果 ε=1e-10，那 10·log₁₀(1e-10) = 10·(-10) = -100 dB，这样看是对的。但为什么说"噪声底"？这是说当信号完全消失、只剩背景噪声时，dB 值会被压到 -100 dB 这么低吗？希望能给个例子（比如实际声音数据）解释这个含义。

### 【推演】n_frames = 1 + (N - win_len) // hop 这个公式怎么推导？

白板挑战问 1 问 n_frames，答案用公式 `1 + (N - win_len) // hop`。我能套进去算出 61，但**这个公式为什么是这样**？

我想象 STFT 分帧：第 0 帧从样本 0 开始，第 1 帧从样本 hop_length 开始……最后一帧能从哪儿开始？如果最后一帧要覆盖 win_len 个样本，那最后一帧的起点最晚是 N - win_len。从 0 到 N - win_len，间隔是 hop_length，帧数是多少？`(N - win_len) / hop_length + 1` 对吗？那为什么写成 `1 + (N - win_len) // hop`，这个 1 和 // 符号代表什么？（我只会 Python 基础，// 是整除吧，那如果有余数会怎么样？）

### 【理解】imshow 的 .T 转置和 origin='lower' 分别做什么，为什么都要用？

我知道 STFT 矩阵是 (n_frames, n_bins)，行是时间，列是频率。转置后变 (n_bins, n_frames)，行变成频率。然后 imshow 把行当纵轴、列当横轴，所以频率在纵轴，时间在横轴——这个我能理解。

但 origin='lower' 的作用**我还是有点混淆**。转置后第 0 行是频率 0（最低频率），imshow 默认把第 0 行放在图的最上面（origin='upper'）。如果不改 origin，最低频率就会出现在图的顶部，这看起来很奇怪（频率应该越往上越高）。所以用 origin='lower' 让第 0 行跑到底部，最低频率就在下面了。这样对吧？希望能给个两列对比图（一列用 origin='upper'，一列用 origin='lower'），直观地看到效果。

### 【实践】plot_spectrogram 中 extent 的四个数字怎么从矩阵形状算出来？

要设 `extent=[0, duration, 0, sr/2]`，其中：
- `duration = len(x) / sr` — 这个清楚，是信号总长
- `sr/2` — 这是奈奎斯特频率（最高频率），清楚

但这和 STFT 矩阵的 shape (n_frames, n_bins) 怎么对应？n_frames 和 n_bins 分别用来算什么？我理解 **extent 中的四个数应该是坐标轴的范围**，但如果我不知道 n_frames 和 duration 的关系、或者 n_bins 和 sr 的关系，怎么验证 extent 是否正确设置了？希望课程能讲讲：给定 n_frames、n_bins、sr、hop_length，怎么反推出 extent 的四个数，以及**这个设置是否能让图中的每个像素都对应正确的时间和频率位置**。
