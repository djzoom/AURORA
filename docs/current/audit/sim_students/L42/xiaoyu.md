# 小雨 — L42 学习日志

## 我理解了什么

1. **FFT 是频率分析工具**：把时域的信号数据转换成频域，显示里面有哪些频率成分——纯音显示单一尖峰，和弦显示多个峰，白噪声显示平坦的宽带。

2. **蝶形网络优化计算量**：DFT 原本是 N×N 矩阵乘以 N 维向量，要算 N² 次操作。FFT 通过递归分折（偶数索引和奇数索引分开算），分成三层蝶形，把操作数从 N² 降到 N log N。

3. **频谱的镜像对称**：实数信号的 FFT 结果是对称的，前半段（k=0 到 N/2）和后半段是镜像，所以只需要看前半段就行。

## 遇到的困难和问题

### 【理解】复指数是什么，它的实部为什么是余弦、虚部是正弦？
> 位置："每一行是一个频率 k 的复指数，实部和虚部分别是余弦/正弦波"

DFT 矩阵定义用的是 `W^(k*n) = exp(-2πi·k·n/N)`，这里出现了"复指数"。课程说它的实部是余弦、虚部是正弦，但没有解释**为什么**。我没学过复数和虚数单位 i 的运算规则，不知道 exp(某个复数) 的结果是什么样的。需要补充：复数基础、欧拉公式、或者至少说明 exp(ix) = cos(x) + i·sin(x) 这个关系。

### 【理解】什么叫"共轭"？
> 位置："共轭对称 X[N-k] = conj(X[k])"

`conj()` 函数没有定义。我猜共轭可能是虚部取反，但不确定。需要明确说明：复数共轭是什么（实部不变，虚部取反吗？），以及为什么 DFT 会有这个性质。

### 【理解】什么是"酉矩阵"，为什么这个性质重要？
> 位置："F 是 N×N 酉矩阵（`F @ conj(F).T = N * I`）"

这是线性代数的概念，但课程没有解释。我不知道酉矩阵的几何意义、为什么 DFT 矩阵是酉的、这个性质带来什么好处。验证代码显示了公式，但没有说这个性质**有什么用处**。

### 【推演】为什么要做位逆序（bit-reversal），它的目的是什么？
> 位置：练习提示第 1 步，`bit_rev = [int(f'{i:03b}'[::-1], 2) for i in range(N)]`

这行代码我可以一步步读懂（二进制格式化、字符串反转、重新转十进制），但**为什么**FFT 需要这个操作？课程只说"按位逆序重排输入"，没有解释这个操作的数学或算法意义。

### 【推演】蝶形公式如何推导？
> 位置：第 2 节，`X[k] = E[k] + W^k * O[k]` 和 `X[k+N/2] = E[k] - W^k * O[k]`

课程说 Cooley-Tukey FFT "把 N 点 DFT 拆成两个 N/2 的 DFT"，然后直接给了这两行公式。我看不到推导过程——为什么加法对应上索引，减法对应下索引？这两个公式从哪个更基本的 DFT 定义推出来的？

### 【计算】旋转因子 W^k 的几何意义是什么？
> 位置：处处出现的 `W^k = exp(-2πi·k/N)`，以及蝴蝶图中的标注 `W^(N)^{k}`

为什么它被叫"旋转因子"？在复平面上它代表什么？它的模是 1 吗？它与"旋转"这个几何操作有什么关系？这对理解蝶形网络的图形很重要。

### 【计算】怎样计算频率 bin 位置的公式 `bin = round(f * N / sr)`？

### 【理解】为什么 aurora FFT 要求输入长度是 2 的幂次？

代码中 `N = 256 = 2^8`，课程说不是 2 的幂次要用 zero-padding 补到下一个 2 的幂次。但没有解释**为什么** Cooley-Tukey 算法必须是 2 的幂次？为什么不能是其他情况？如何选择补零的长度？

### 【理解】练习中三层循环的索引逻辑
