## 遇到的困难和问题

### 问题 1【推演】`np.sin(2 * np.pi * freq * t)` 里的 2π 是哪来的？

位置：第 1 节代码格，注释"aurora.audio.sine 的核心就是这一行"。

我高中学的 sin 是输入角度（比如 sin 30° = 0.5），这里输入的却是 `2 * np.pi * 440 * t` 这种巨大的数。为什么乘 2π？sin 括号里现在到底是"度"还是别的什么单位？"弧度"两个字全课一次都没出现。我知道 L04 会讲三角函数，但这是全课第一段代码，一行解释都没有，我只能"先信了"。

**希望补充**：在这格加两三行注释或一小段 markdown：np.sin 吃的是弧度、转一整圈是 2π、`freq * t` 表示"到时刻 t 已经转了几圈"，所以乘 2π 就变成弧度。哪怕说"细节 L04 再讲，先记住这个读法"也行。

### 问题 2【理解】`t = np.arange(round(duration * sr)) / sr` 完全没解释

位置：同一格，`sr, duration, freq = 16000, 0.02, 440  # 采样率, 时长(秒), A4 音`。

"采样率"是什么本课没讲（下一课 L02 才讲）。我自己打印了 `t[:5]` 才看明白：先生成 0..319 的整数，再除以 16000 变成一串时间点，相邻两点差 1/16000 秒。为什么点的个数 = 时长×采样率、为什么要除回 sr，格子里一个字都没说。另外"A4 音"是什么我也不知道（我猜是某个钢琴键？）。

**希望补充**：给这一行拆成两步注释（多少个点、每个点对应第几秒），并注明"采样率的原理 L02 详讲，这里只需知道 t 是 0.02 秒内均匀取的 320 个时刻"；"A4 音"加半句解释。

### 问题 5【理解】FFT 输出里的 `j` 和那个 `2` 没有任何解释

位置：FFT 验证格的输出 `[0.+0.j 2.+0.j 0.+0.j 2.+0.j]`；热身 Q2"out[0] 是直流分量（DC，频率 = 0 Hz）……均值是多少？"

Q2 本身我做对了：均值 = (1+0-1+0)/4 = 0，out[0].real 也确实是 0，对上了。但两个后续疑问没着落：(a) 输出里的 `j` 是什么？复数要 L05 才讲，这里连"这是复数、暂时只看 .real"都没说；(b) 题目暗示 out[0] 和"均值"有关，可这次均值恰好是 0——如果均值不是 0，out[0] 到底等于均值本身还是均值乘个什么？我没法用这一个例子确认自己真懂了。另外 out[1] 和 out[3] 那两个 `2` 是什么意思也完全没人提。

**希望补充**：一句话说明"输出是复数（L05 详讲），现在只看实部；out[0] = 所有采样点之和 = 均值 × N"，最好再给一个均值不为 0 的小例子（比如 `[2,1,0,1]`）让我们验证。
