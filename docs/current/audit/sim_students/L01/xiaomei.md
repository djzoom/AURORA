### 问题 1【推演】正弦波那一行公式完全没有解释，我只能"信"它

位置：第 1 节代码格，`x = np.sin(2 * np.pi * freq * t)`，旁边注释说
"aurora.audio.sine 的核心就是这一行"。

高中我学过 y = sin(x)，x 是一个角度。但这里 x 变成了 `2 × π × 频率 × 时间`，
为什么乘 2π？为什么频率乘时间就变成了角度？还有上一行
`t = np.arange(round(duration * sr)) / sr` 为什么要除以 sr？
课程一个字都没解释，就说"核心就是这一行"。我把图跑出来了，数了波峰确实
约等于 440 × 0.02 ≈ 8.8 个，所以它"是对的"，但我推不出来它为什么对。

希望补充：给我一张图——比如一个点绕圆圈转，一秒转 440 圈，每转一圈走过
2π 弧度，sin 就是这个点的高度。有这张图我就能自己把公式拼出来。哪怕加一句
"这行公式 L02 会拆开讲，现在先看图感受"也行，现在的写法像是默认我已经懂了。

### 问题 3【解答】FFT 输出 [0, 2, 0, 2]，除了 out[0] 全都没有下文；"基频"这个词没定义

位置：热身推理 Q1"这个信号代表的频率是 ______（几倍的基频？）"和 FFT 验证格。

三个卡点：
1. **"基频"是什么？** 全课没出现过这个词的定义。我对着 [1, 0, -1, 0] 画了
   一下，能看出它是"一整个来回"（1 → 0 → -1 → 0 → 回到 1），但"几倍的基频"
   我答不上来，因为不知道分母是什么。
2. Q2 我算出均值 = 0，运行后 out[0].real 确实是 0，对上了。但**为什么**
   "频率为 0 的分量"会等于信号的均值？课程只说"out[0] 是直流分量（DC，
   频率 = 0 Hz）"，均值和 DC 之间的那根线没有连上。
3. 最困惑的：完整输出是 `[0.+0.j, 2.+0.j, 0.+0.j, 2.+0.j]`。信号明明只
   "振"了一种节奏，为什么 out[1] 和 out[3] **两个**位置都是 2？还有那个
   `+0.j`——复数要 L05 才学。热身题只对了 out[0] 的答案，剩下的数字就
   晾在那里，像考完试只发一道题的答案。

希望补充：（a）用一句话+小图定义基频（"这段长度里正好装下一个完整波
= 基频的 1 倍"之类）；（b）加一句"为什么 DC = 均值/总和"的直观说明；
（c）对 out[1]、out[3] 至少给一句"这是对称性，L38 会解释"，别让我以为
是自己算错了。
