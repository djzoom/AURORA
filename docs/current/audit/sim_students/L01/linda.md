### 问题 3【推演】热身 Q1 我推不出来："基频"、"归一化 sr=1" 都没讲过

**位置**：热身推理 Q1："信号 `[1, 0, -1, 0]` 只有 4 个采样点，代表的是什么频率？……这个信号完成了几个完整的振荡周期？"括号里还问"几倍的基频"。

**卡在哪**：三个词我都接不住。(a) "采样率 sr = 1（归一化）"——归一化是什么意思？前面正弦波例子里 sr 是 16000，怎么突然变成 1 了？(b) "基频"这个词全课第一次出现，没有定义。(c) 我盯着 `[1, 0, -1, 0]` 四个数字想"几个完整周期"，想不出来——四个孤立的点，怎么看出"振荡"？1 到 0 到 -1 到 0，然后呢？要脑补它回到 1 才算一圈吗？没有图我建立不起这个画面。最后我只能空着 Q1。

**希望课程补充**：配一张小图——把这 4 个点画出来，再叠一条穿过它们的余弦曲线，标出"这就是一个完整周期"；并且用一句话定义基频（比如"N 个点里恰好转完一圈的那个频率，= sr/N"）。有图我就能推了，纯文字提示对高中水平的我不够。

### 问题 4【计算】热身 Q2 的提法有误导：out[0] 其实是"总和"，不是"均值"

**位置**：热身推理 Q2："`[1, 0, -1, 0]` 的均值是多少？out[0].real 应该接近什么值？"

**卡在哪**：这个问法强烈暗示 out[0] ≈ 均值。我算出均值 = 0，跑出来 out[0] = 0，貌似对了。但我多手做了个实验：把信号换成 `[1, 1, 1, 1]`，均值是 1，可 out[0] 却是 **4**！所以 out[0] 根本不等于均值，而是所有采样点的**总和**（= 4 × 均值）。这道题只是因为均值恰好是 0，才让"均值"和"总和"撞在同一个答案上。如果我把"DC 分量 = 均值"当结论记下来，以后一定会栽跟头。另外"直流分量（DC）"这个词本身也没解释——声音里哪来的"电流"？

**希望课程补充**：要么把 Q2 改成"所有采样点加起来是多少"，要么明确写一句"out[0] = 所有采样点之和 = N × 均值，本例均值为 0 所以两者恰好都为 0"。再加半句解释 DC 为什么叫直流/为什么代表 0 Hz。
