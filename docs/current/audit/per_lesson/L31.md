## 概述

L31 is a well-structured visualization notebook that covers LLN convergence, Gaussian PDF/CDF, z-score, softmax, cross-entropy, and Shannon entropy. All executable code runs correctly (assertions pass, imports resolve), and the hand-written CDF and entropy cells are pedagogically solid. However, the Aurora connection section makes two phantom references to functions that do not exist in `src/aurora/`, which directly contradicts the course's defining principle of connecting concepts to real codebase. Additionally, the notebook offers zero student exercises — every cell is a completed demonstration — which is a notable gap for a lesson at this position in the sequence.

## 亮点

- The Shannon entropy cell (Section 7) is the strongest in the notebook: self-contained, mathematically correct (H(p,q) = H(p) + KL(p‖q) identity verified numerically), with clear bar-chart annotations and a passing assertion.
- The CDF cell (Section 8) correctly uses `np.trapezoid` (NumPy 2.0+ API), includes the 68–95–99.7 rule with tight tolerances (< 0.002), and all three assertions pass on the actual values.
- Temperature scaling demo (Section after 5) is a genuinely useful extension that connects softmax to knowledge distillation and beam search — rare pedagogical depth for a visualization lesson.
- `sviz.py` is a clean, well-organised helper with stable internal implementations; it correctly uses `np.cumsum` for the LLN running mean and numerically stable softmax (`z - z.max()`).
