**[Exercise Design / Pedagogical Quality]** — MFCC 答案格对 DCT-II（步骤 7）和 Mel 滤波器（步骤 5）没有代码实现，只有 print 文字描述。

第 7 步的题目要求写出 DCT-II 公式，但对答案格只打印一句说明，没有公式代码。对比 FFT 和 Attention 都有完整可运行的参考实现，MFCC 答案格质量明显偏低，学生无法通过运行对答案格来验证自己对 DCT-II 公式的推导是否正确。

Fix: 在对答案格中加入 DCT-II 的 NumPy 实现（约 5 行）：
```python
def dct2(x):
    M = len(x)
    ns = np.arange(M)
    return np.array([np.sum(x * np.cos(np.pi * k * (2*ns + 1) / (2*M)))
                     for k in range(M)])
```
并用 `scipy.fft.dct(x, type=2, norm=None)` 做 assert 验证。
