### 🟡 Minor

1. **[Runability / 措辞]** — "本课收束"（cell-17）最后一句以"更新。"结尾，明显是编辑时遗留的草稿标记，不是正文。Fix: 删除"更新。"。

2. **[Exercise Design / 存根设计]** — `butterfly` stub（cell-12）全部 TODO 行赋值为 `...`（Ellipsis），函数直接 `return ...`。测试格（cell-13）检测 `result is ...` 能处理完全未实现的情况，但若学生只替换部分 `...`（如实现了 `twiddles` 但留下 `top = ...`），函数仍然 `return ...`，测试仍然显示"请先实现"，无法区分"完全未做"和"部分完成"。Fix: 用 `raise NotImplementedError("TODO: ...")` 替换每处 `...` 赋值，让 Python 在调用点立即报错并给出清晰栈跟踪；将测试格的 `except TypeError` 改为 `except NotImplementedError`（当前的 TypeError 分支永远不会被触发，因为纯粹的 `...` 赋值不产生 TypeError）。
