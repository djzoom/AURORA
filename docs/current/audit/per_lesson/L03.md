### 🟠 Major

**[Exercise Design]** — Cell c11 instructs students "在下面 Markdown cell 里写预测，然后运行代码验证" but the very next cell (c12) is already the verification code block. There is no blank Markdown cell for students to record their predictions. ### 🟡 Minor

**[Code Quality]** — `vowel_approx(t, f0, f1, f2, sr=16000)` in cell c10 declares `sr` as a keyword argument with a default value, but `sr` is never referenced inside the function body. This is dead/misleading code — a student reading the signature would expect `sr` to control something (sample rate of synthesis?).

Fix: Remove `sr=16000` from the signature, since the function operates on a pre-built time array `t`.

---

**[Pedagogical Precision]** — The `show_spec` docstring comment in c2 says `noverlap=128` gives "时间分辨率更平滑". Overlap does not improve time *resolution* (which is fixed by hop length = NFFT − noverlap = 128 samples ≈ 8 ms); it produces a denser, visually smoother display. Conflating "smoother display" with "resolution" is a common student misconception.

Fix: Change comment to: "相邻帧重叠 128 点 → 每步跳 128 点（hop）= 8 ms，显示更连续（分辨率由 hop 决定，不是 overlap 本身）" or simply "可视化更平滑".

### 💡 建议

1. **Log-scale disclosure**: `plt.specgram` displays power in dB by default (logarithmic color mapping). The lesson never mentions this, so when students later implement their own linear-scale spectrogram in L43-L45 the colors will look different. A one-line note ("颜色是对数刻度 dB，L43 自己实现时会细讲") prevents future confusion.

2. **Vowel formant accuracy note**: The `vowel_approx` function comment says "非精确语音合成，仅用于谱图演示", which is appropriate. But the F2 for `vowel_i` is set to 3000 Hz — real /i/ has F2 ≈ 2200-2800 Hz. The visual demonstration still works, but if the lesson claims the values are "simplified formant parameters", using published approximate values (Peterson & Barney 1952) would strengthen the Aurora "from first principles" ethos.

The notebook itself correctly reads "L03 · 谱图（spectrogram）直觉". The course's lesson index/registry (if any) should be updated to reflect the rename.
