## 概述

本课对 scaled dot-product attention 的数学动机（缩放因子、Q/K/V 分离）讲解清晰，"bank/river/rates" 消歧例子是课程亮点。然而，最后的"完整前向传播"验证单元绕过了学生实现的函数，使唯一的综合验证形同虚设；section 3 遗留了一个 "L2" 的错误课号引用；PE 和 FFN 两节都被标记为"## 6."，节编号重复。数学公式本身正确，但练习设计和可验证性有明显缺口。

## 亮点

- "bank/river/rates" 的具体消歧例子：精准演示 Q/K/V 分离的意义，是全课最佳直觉解释之一。
- 缩放因子的方差推导（Var(Q·K) = d_k → 除以 √d_k 归一化为 1）数学严密，符合原始论文。
- cell-6 的 d_k 尖锐度对比实验（d_k=4/64/512 未缩放 vs 缩放）直观有力，是动机讲解的好补充。
- cell-11 的三级检查结构（形状 → 权重和为 1 → causal mask 形状）骨架设计合理。
- PE 实现代码简洁正确，向量化 broadcasting 写法规范。
