# L51 — MFCC 工程实战 — 在真实 WAV 音频上提取特征，librosa 对答案

**总体评级**: B- | **问题数**: 0 critical / 3 major / 6 minor

## 概述

本节是 Aurora audio 流水线的集成演示课，覆盖了 read_wav → STFT → Mel → MFCC 四层可视化，并通过可选 librosa 对比验证数值正确性。代码结构清晰，说明文字水准良好，核心实现路径（src/aurora/audio/）与实际文件结构吻合。然而课名标榜"工程实战"，全节却没有一道 TODO 练习，仅为纯演示；stale 的 `day3_` 输出文件名暴露了从早期课程结构直接复制粘贴的痕迹。

## 亮点

- **四层可视化设计**：波形 → STFT → log-Mel → MFCC 层层递进，每层配有表格说明"看什么"，教学逻辑清晰。
- **VCV 合成信号**：`make_vcv_signal()` 用谐波叠加合成元音、白噪声模拟爆破辅音，让无真实 WAV 的环境仍能运行完整流水线，设计合理。
- **参数实验表格**：Cell 9 的 Markdown 表格准确描述了 n_fft/hop_length 对时频分辨率的权衡，配合 Cell 10 三幅对比图，是一处优质的概念强化。
- **librosa 对比原则正确**：Aurora 与 librosa 默认参数对齐（center=True、Hann 窗、HTK mel、ref=1.0），比较前提成立，2 dB 阈值在实现差异范围内基本合理。
