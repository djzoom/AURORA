### 🟠 Major

**[Accuracy / Shape Description]** — cell `cell-3` 声明"U 是 `(m, m)`，Vt 是 `(n, n)`"，但 cell `cell-8` 使用 `full_matrices=False`（thin SVD），对 A=(2,3) 实际输出 `Vt.shape=(2,3)` 而非 `(3,3)`。学生运行 cell-8 后会观察到与 cell-3 文档相矛盾的 shape，造成混淆。全课程中两种 SVD 变体（cell-8 用 thin，cell-10/15f78c03/stub 用 full）混用且从未解释。  
**Fix**：cell-3 改为双行："full SVD：U `(m,m)`, S `(k,)`, Vt `(n,n)`；thin SVD（full_matrices=False）：U `(m,k)`, S `(k,)`, Vt `(k,n)`，k=min(m,n)"，并在 cell-8 的 print 旁注释说明正在使用 thin SVD。

**[Exercise Design / Stub Convention]** — `return None` 而非 `raise NotImplementedError`，不仅违反课程 stub 惯例，还直接导致上述 TypeError 崩溃，使 stub 约定与其他课次不一致。
