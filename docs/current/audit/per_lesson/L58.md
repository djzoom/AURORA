## 问题列表

### 🔴 Critical

**[Accuracy + Exercise Design]** — `Neuron.__call__` 对所有层（包括输出层）无条件施加 ReLU，使模型输出恒 ≥ 0。以标签 ±1 进行 hinge loss 训练时：正类损失可降至 0，负类损失为 `max(0, 1+score) ≥ 1`，永远非零。由于分数恒 ≥ 0，分类器永远预测正类，准确率固定在 50%。主验收断言 `assert acc > 0.85` 即便学生完美实现 `train()`，也必然失败。

Fix: 在 `Neuron.__init__` 增加 `nonlin=True` 参数，`__call__` 中改为 `return act.relu() if self.nonlin else act`；在 `MLP.__init__` 中将最后一层的所有 `Neuron` 以 `nonlin=False` 实例化（karpathy 原版 micrograd 的做法）。相应地更新 `MLP.__init__` 的最后一层构造逻辑。
