## 跨课共性建议（优先看）

1. **导航短标题不一致**（如 L01「Aurora 是什么」vs 课扉长标题）——全库统一「Lxx · 短标签」。
2. **实验入口 / 动手观察 / 代码实验** 三连导语在 L05–L12 等同质——可合并为一条。
3. **中英夹注**在首课与路线图过密；成熟术语后续课可只留中文。
4. **Why invent / Common Pitfall / Future Payoff** 中英标题建议统一中文。
5. **🎨 可视化课**易重讲内容课全文——主路径应「读图 + 本课新点」，复习进折叠（L08 最典型）。
6. **高密度课**（L13、L29、L38、L63、L70）需要**分层/折叠**，不是删深度。
7. **短文课**需区分「练习主导且表够用」（如 L74、L98）与「主题大而文紧」（如 L83）——对策不同。
8. **交叉引用课号**偶有漂移（如 L04 写 L07 讲采样定理，实际 aliasing 在 L34）——应用脚本扫一遍。
