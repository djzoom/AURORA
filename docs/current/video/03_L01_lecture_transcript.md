### 〔第七节·动手〕check_imports:当你的环境医生

(切到第 9 节,语气转成手把手。)

好,道理讲够了,来动手写这门课**第一个你自己的函数**。

(讲医生比喻,慢一点,这是本课的核心类比。)

想象你是个医生。病人进来说:"大夫,我哪儿不太舒服。" 你会不会随手抓瓶药说"试试这个"?不会。好医生会**先做一套检查**,把问题精确到具体的器官。

写代码碰到环境报错,一模一样。"import 失败"是**症状**,不是**诊断**。你不能对着一个 `ImportError` 干瞪眼。

我们要写的 `check_imports`,就是那套检查工具:给它一串包名,它告诉你——哪个装了,哪个没装,精确到每一个包。

(切到 TODO 格,指提示。)

怎么写?提示就在格子里。核心是一个工具:`importlib.util.find_spec(name)`。对每个包名查一下,它返回 `None`,就是没装,记 `False`;返回不是 `None`,就是装了,记 `True`。就三行。

```python
def check_imports(names):
    result = {}
    for name in names:
        result[name] = importlib.util.find_spec(name) is not None
    return result
```

(运行下面的检查格,变绿。)

跑一下它的检查格——绿了。恭喜,你在这门课写的**第一个函数**,通过了。

(温和地补一句兜底。)

要是你这儿是红的,九成是两件事:循环里忘了 `return`,或者把 `None` 的判断写反了。别慌,对着提示再看一眼,改完重跑。这门课里红色不可怕,红色只是在跟你说话。

### 〔第八节·习惯〕先猜,再验证——这门课最重要的动作

(切到 FFT 热身格,亮出 `[1, 0, -1, 0]`。语气神秘一点。)

在写下一个函数之前,我要教你这门课**最重要的一个习惯**,比任何代码都重要:**先猜,再验证。**

(指着信号。)

看这个信号:`[1, 0, -1, 0]`,四个采样点。我先不运行,先问你两个问题,你在心里答。

问题一:这四个点,代表的是什么频率?提示——它在这段里,完成了几个完整的振荡周期?

问题二:FFT 输出的 `out[0]`,是直流分量(DC),也就是零频率。零频率说白了就是**这段信号的平均值**。那 `[1, 0, -1, 0]` 的平均值,是多少?

(停,认真地等。)

想好了吗?一加零减一加零,除以四……对,是零。所以你**猜** `out[0]` 应该接近零。

(切到 FFT 验证格,运行,打出 ✅。)

现在验证。回车——`out[0].real` 约等于零。**跟你猜的一样吧?**

(转向镜头,把这一下的意义讲透。)

这一声打勾,有两层意思。表面上,它说你的环境完全正常,FFT 能用。但更深一层——你刚刚**先用脑子预测,再用代码验证**。这就是研究员和"调库工"最大的区别:调库工运行完看结果;研究员运行前心里就有数,运行只是**确认**。

这门课接下来,每一个能预测的地方,我都想让你先猜。猜错了更好——那说明你的直觉正好在这儿有个漏洞,补上它。

### 〔第九节·动手〕environment_report:给环境拍张快照

(切到第 10 节。)

第二个函数,`environment_report`。刚才的 `check_imports` 是查单个包,这个是给你**整个环境拍一张快照**,四个信息:Python 在哪、什么版本、aurora 能不能导入、项目根目录在哪。

(讲找根目录的思路,这是本函数唯一的"想一想"。)

怎么找根目录?思路是这样:从你**当前所在的文件夹**开始,一级一级往上走,看谁的文件夹里有 `pyproject.toml` 这个文件——谁有,谁就是项目根。

```python
for folder in [Path.cwd(), *Path.cwd().parents]:
    if (folder / 'pyproject.toml').exists():
        root = folder
        break
```

四个键填上,返回。(运行检查格,变绿。)绿了——你的第二个函数也过了。

(顺势讲一个关键概念,别错过。)

顺带说个重要的东西:你装这门课时跑的 `make install`,底下是 `pip install -e .`,这个 `-e` 叫**可编辑安装**(editable install)。意思是 `import aurora` 直接连到源码目录 `src/aurora/`——你改了源码,不用重装,重启一下内核就生效。这就是为什么这门课能"一边学一边改源码"。

(兜底。)
