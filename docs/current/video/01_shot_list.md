## §1 拍摄前 · 技术设置清单(一次性,拍前全部打勾)

### 录制规格
- [ ] 录屏用 **OBS**;画布 **3840×2160(4K)** 或至少 **1920×1080**;**60 fps**(打字/滚动才顺滑)。
- [ ] 码率 CBR **20–40 Mbps**;编码优先硬件(NVENC / Apple VT);格式 `.mov`/`.mp4`。
- [ ] 出镜机位单独录:**24 或 30 fps**,快门 1/50 或 1/60,尽量 4K 留裁切余量。
- [ ] **音频单独录一轨**(领夹麦/枪麦),不靠机内;电平峰值 **−12 到 −6 dB**,房间做基本吸音。
- [ ] 全程 **DND 免打扰**:关系统通知、微信/邮件、日历弹窗。

### 录屏画面"干净化"(新手视频最忌乱)
- [ ] 终端字号调大到 **18–22 pt**;高对比主题(纯黑底白字 / Solarized Dark);窗口约 **90–100 列**宽。
- [ ] 终端**清空 scrollback**,提示符简洁(别暴露一长串个人路径 / 用户名可改)。
- [ ] Jupyter 用**干净浏览器 profile**:隐藏书签栏、关插件图标、浏览器缩放 **125–150%**、全屏。
- [ ] 隐藏桌面图标、Dock/任务栏自动隐藏、隐藏时钟里的个人信息。
- [ ] **放大光标 + 高亮**;命令段开 **击键可视化**(Mac: KeyCastr / Win: Carnac),让观众看清每个键。

### 双系统就位(关键命令要 Mac+Win 各来一遍)
- [ ] Mac 机 + Windows 机(或虚拟机)都装好录屏,画面尺寸/字号统一,方便做画中画。
- [ ] 至少这两处必须双系统各拍:**打开终端的入口** · **虚拟环境激活命令**。

### 环境状态准备(为了"真实地跑给你看" + 可重拍)
- [ ] 预先把 `.venv` **删掉**,好让 `python3 -m venv .venv` 是**真的当场新建**(段落 4 要真效果)。
- [ ] 但另存一份**已装好的环境**做备份,NG 重拍时不用等 `pip install`。
- [ ] 准备一台"**故意装错**"的机器/快照,用来拍段落 8 的三个报错(未切内核、缺内核、NotImplementedError)。
- [ ] `pip install` 那格可**真跑一次录进度**,后期加速;网络慢就先预热缓存。

### 台词/命令备稿(防打错字)
- [ ] 把 §6 的**命令清单**打印/贴在副屏:短命令**当场手打**(保留键盘声),长 URL / 长命令**粘贴**(别在镜头前打错)。
- [ ] 讲师旁白逐字稿上提词器;live-coding 段先排练 1–2 遍走位。

# 手打（保留键盘声，短）
python3 --version
cd AURORA
ls
python3 -m venv .venv
source .venv/bin/activate                 # Mac/Linux
.venv\Scripts\Activate.ps1                # Windows PowerShell
jupyter lab

# 粘贴（长/易错，别在镜头前手打）
git clone https://github.com/djzoom/AURORA.git
pip install -e ".[dev,notebooks]"
python -m ipykernel install --user --name aurora --display-name "Python (AURORA)"
pip install -e ".[dev,notebooks]" -i https://pypi.tuna.tsinghua.edu.cn/simple   # 备用镜像
