## 制作备忘

- **前 3 秒定生死**:两行 API 刷字 + "这是你写的吗?"必须在 0:03 前砸出,否则划走。
- **正弦波那一跳**是全片情绪最高点,给足镜头推近 + 音效"叮"的一下,别一闪而过。
- **盖楼动画**要能看清是"一层叠一层、不能跳",呼应"强顺序性"卖点;FFT 那块单独点亮。
- **误差 < 1e-10** 一定要露一帧,这是"真能跑通"的信任锚,区别于口嗨。
- **不焦虑**:全片不出现"不学就被淘汰/学不会就完了";段落 4 明确"不踩别人"。
- **平台裁剪**:小红书版把 CTA 字幕换成"私信『FFT』领课程地图";B站可保留主页链接。
- 数字口径以 `brand_brief.md` 为准:99 课 / 11 模块 / 6 个月 / 4 作品 / 前 53 课只用 numpy。
